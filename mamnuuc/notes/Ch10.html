  2. 2.
    ​

    I claim that an→3/2. Proof: Let ε>0 be given. Then there exists N∈ℤ+ such that N>5/(2⁢ε). Now, for all n≥N,

    |an-32|=54⁢n-2≤54⁢n-2⁢n=52⁢n≤52⁢N<ε.
  3. 3.
    ​

    Assume that there exists a sequence (xn) in D\{a} such that xn→a. We seek to prove that a is a cluster point of D. So, let ε>0 be given. Since xn→a, there exists N∈ℤ+ such that for all n≥N, |xn-a|<ε and so, in particular, |xN-a|<ε. But xN∈D and xN≠a, so 0<|xN-a|<ε. Such a term xN exists whatever ε>0 is chosen, so a is a cluster point of D.

    Conversely, assume a is a cluster point of D. Then, for each ε>0, there exists x∈D such that 0<|x-a|<ε, and hence x∈D\{a} such that |x-a|<ε. This is true, in particular, if ε=1/n, where n∈ℤ+. That is, for each n∈ℤ+, there exists xn∈D\{a} such that |xn-a|<1/n. The sequence (xn) lies in D\{a} and converges to a by the Squeeze Rule.

  4. 4.
    ​

    Let ε>0 be given. Then take δ=4⁢2⁢ε>0. Then, for all x∈[0,∞) such that 0<|x-2|<δ,

    |x-2x-2-12⁢2| = |(x-2)⁢(x+2)(x-2)⁢(x+2)-12⁢2|=|1x+2-12⁢2|
    = |x-22⁢2⁢(x+2)|=12⁢2⁢|x-2(x+2)2|
    < 14⁢2⁢|x-2|<14⁢2⁢δ=ε.

    This means that the function f:[0,∞)→ℝ, f⁢(x)=x is differentiable at 2, with derivative 12⁢2.

  5. 5.
    ​

    Assume, towards a contradiction, that limx→0⁡|x|x exists, call it L. Then, for each ε>0, there exists δ>0 such that, for all x∈(-δ,0)∪(0,δ),

    ||x|x-L|<ε.

    This holds, in particular, if ε=1: there exists δ>0 such that, for all x∈(-δ,0)∪(0,δ), ||x|/x-L|<1, and hence

    L-1<|x|x<L+1.

    This holds, in particular, for x=δ/2>0, so

    L-1<1<L+1,

    from which it follows that L>0 (since L+1>1). But it also holds for x=-δ/2<0, so

    L-1<-1<L+1,

    whence L<0 (since L-1<-1). This is a contradiction, so the limit does not exist.

    This means that the function f⁢(x)=|x| is not differentiable at 0.

  6. 6.
    ​

    We must show that

    limx→1⁡f⁢(x)-f⁢(1)x-1

    exists. I claim that it does, and equals 4. Proof: Given any ε>0, let

    δ=min⁡{1,ε/11}.

    Then for all x∈ℝ with 0<|x-1|<δ,

    |f⁢(x)-f⁢(1)x-1-4| = |x4-1x-1-4|
    = |(x+1)⁢(x2+1)-4|
    = |x3+x2+x-3|=|(x-1)⁢(x2+2⁢x+3)|
    ≤ |x-1|⁢(x2+2⁢|x|+3)  by the triangle inequality
    ≤ |x-1|⁢(4+4+3)  since |x-1|<1, so |x|<2
    < 11⁢δ≤ε.
