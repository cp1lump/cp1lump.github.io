9.1 Problem Set 1: Limits of functions, differentiability

Submit on Gradescope by 17:00, Friday 11 February 2022

  1. 1.
    ​

    Let D⊆ℝ. Write down precise mathematical formulations of the following statements, using quantifiers (∀, ∃):

    1. (a)
      ​

      D is bounded above.

    2. (b)
      ​

      D has a maximum element.

    3. (c)
      ​

      D does not have a maximum element.

  2. 2.
    ​

    Prove from first principles (i.e. give a direct ε–N proof) that the following sequence converges:

    an=3⁢n+12⁢n-1.
  3. 3.
    ​

    Prove Proposition 1.28: a∈ℝ is a cluster point of a set D⊆ℝ if and only if there exists a sequence (xn) in D\{a} which converges to a.

  4. 4.
    ​

    Give a direct ε–δ proof that

    limx→2⁡x-2x-2=12⁢2.

    Interpret this result in terms of differentiability.

  5. 5.
    ​

    Give a direct ε–δ proof that limx→0⁡|x|x does not exist. Interpret this result in terms of differentiability.

  6. 6.
    ​

    Give a direct ε–δ proof that the function f:ℝ→ℝ, f⁢(x)=x4 is differentiable at 1.

9.2 Problem Set 2: Differentiability on an interval

Submit on Gradescope by 17:00, Friday 4 March 2022

  1. 1.
    ​

    Prove that, for all x∈ℝ, |sin⁡x|≤|x|.

  2. 2.
    ​

    Prove that the function f:ℝ→ℝ, f⁢(x)=4⁢x7-14⁢x4+30⁢x-17, is injective.

  3. 3.
    ​

    Let f:ℝ→ℝ such that f⁢(x)=2⁢x4+x4⁢sin⁡(1/x) if x≠0, and f⁢(0)=0.

    1. (a)
      ​

      Prove that f attains a minimum at 0. (Hint: no need for calculus!)

    2. (b)
      ​

      Prove that f is differentiable (everywhere), and that f′⁢(0)=0. (Hint: you can reduce the work involved using the Localization Lemma, Lemma 2.24.)

    3. (c)
      ​

      Prove that, for all ε>0, f′ takes both positive and negative values on (-ε,ε).

  4. 4.
    1. (a)
      ​

      Construct the first Taylor approximant to f:(-1,1)→ℝ, f⁢(x)=sin-1⁡x about a=12.

    2. (b)
      ​

      Use this approximant, and Taylor’s Theorem, to find upper and lower bounds on sin-1⁡(3/4).

  5. 5.
    ​

    In each of the following cases, either write down a function with the specified properties, or explain why no such function exists.

    1. (a)
      ​

      An unbounded function f:[0,1]→ℝ.

    2. (b)
      ​

      An unbounded differentiable function f:[0,1]→ℝ.

    3. (c)
      ​

      A bounded differentiable function f:ℝ→ℝ whose derivative f′:ℝ→ℝ is unbounded.

    4. (d)
      ​

      A differentiable function f:(0,1)→ℝ which is unbounded above but whose derivative is bounded.

9.3 Problem Set 3: Integration

Submit on Gradescope by 17:00, Friday 18 March 2022

  1. 1.
    ​

    Let A={x+1n:x∈(0,1),n∈ℤ+}. Determine the supremum and infimum of A, if they exist. Rigorously justify your answers.

  2. 2.
    ​

    Let f:[-1,2]→ℝ, f⁢(x)=|x|, and 𝒟={-1,0,1/2,2}.

    1. (a)
      ​

      Draw the graph of f.

    2. (b)
      ​

      Compute the lower and upper Riemann sums l𝒟⁢(f), u𝒟⁢(f).

  3. 3.
    1. (a)
      ​

      Prove that ∑j=1nj3=14⁢n2⁢(n+1)2.

    2. (b)
      ​

      Evaluate ∫01x3⁢𝑑x directly from the definition, without using the Fundamental Theorem of the Calculus. (Hint: use Lemma 4.18 or Theorem 4.21.)

  4. 4.
    ​

    Let f:(-1,1)→ℝ, f⁢(x)=(1+x3)-3. Compute g′:(0,1)→ℝ where

    1. (a)
      ​

      g⁢(x)=∫0xf.

    2. (b)
      ​

      g⁢(x)=∫-xxf.

    3. (c)
      ​

      g⁢(x)=∫0f⁢(x)f.

  5. 5.
    ​

    Let f:ℝ→ℝ be Riemann integrable on every interval [a,b]⊂ℝ and define g:ℝ→ℝ by g⁢(x)=∫0xf. Prove that g is continuous. (Warning: you may not assume that f is continuous.)

9.4 Problem Set 4: Uniform convergence

Submit on Gradescope by 17:00, Friday 29 April 2022

  1. 1.
    ​

    For each n∈ℤ+ let fn:ℝ→ℝ be the function

    fn⁢(x)=n⁢xn⁢|x|+1.
    1. (a)
      ​

      Prove that (fn) converges pointwise to some function f:ℝ→ℝ.

    2. (b)
      ​

      Prove that (fn) does not converge uniformly.

  2. 2.
    ​

    Construct a sequence of unbounded functions gn:[0,1]→ℝ that converges pointwise to a bounded function g:[0,1]→ℝ.

  3. 3.
    ​

    For each n∈ℤ+ let fn:[0,12]→ℝ, fn⁢(x)=1/(1+xn).

    1. (a)
      ​

      Prove that (fn) converges uniformly.

    2. (b)
      ​

      Compute the limit limn→∞⁡∫01/2fn, rigorously justifying your answer.

  4. 4.
    ​

    For each n∈ℤ+ let fn:ℝ→ℝ be the function

    fn⁢(x)=∑k=1n1k2⁢cos⁡(k⁢x).

    Prove that (fn) is uniformly Cauchy, and hence converges uniformly.

  5. 5.
    1. (a)
      ​

      Let f,g:D→ℝ be bounded functions. Prove that ∥f⁢g∥≤∥f∥⁢∥g∥.

    2. (b)
      ​

      Assume fn:D→ℝ converges uniformly to f and gn:D→ℝ converges uniformly to g. Prove that (fn⁢gn) converges uniformly to f⁢g.
