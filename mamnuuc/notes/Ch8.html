8.2 Workshop solutions for week 2

A video of me solving these problems is available, see [VIDEO].

  1. 1.
    1. (a)
      ​

      ∃K∈ℝ,∀x∈D,x≥K.

    2. (b)
      ​

      ∀K∈ℝ,∃x∈D,f⁢(x)>K.

    3. (c)
      ​

      ∀y∈ℝ,∃x∈D,f⁢(x)=y.

    4. (d)
      ​

      ∃y∈ℝ,∀x∈D,f⁢(x)≠y.

  2. 2.
    ​

    I claim that an→1. Proof: let ε>0 be given. Then there exists N∈ℤ+ such that N>3/ε. Now, for all n≥N,

    |an-1| = |2-(-1)nn2+2|≤3n2+1<3n2≤3N2<ε.

    Hence an→1.

  3. 3.
    ​

    For all x∈(0,∞),

    |2⁢x2+3⁢x+4x2+1-2| = |3⁢x+2x2+1|
    ≤ 3⁢xx2+1+2x2+1
    < 3⁢xx2+2x2
    = 3x+2x2.

    Let ε>0 be given. Let K=max⁡{1,5/ε}. Then, for all x>K,

    |2⁢x2+3⁢x+4x2+1-2| < 3x+2x2
    < 3K+2K2
    ≤ 3K+2K  (since K≥1)
    ≤ ε.
  4. 4.
    ​

    First note that the maximal domain of the function f⁢(x)=(x+2)/(x3+8) is D=ℝ\{-2}, and -2 is a cluster point of D. Let ε>0 be given. Then let δ=min⁡{1,ε}. Then for all x∈D such that 0<|x+2|<δ,

    |f⁢(x)-112| = |1x2-2⁢x+4-112|
    = |x2-2⁢x-812⁢(x2-2+4)|
    = |x-4|⁢|x+2|12⁢((x-1)2+3)
    ≤ |x-4|36⁢|x+2|
    < 736⁢|x+2|  (since |x+2|<1, so x-4∈(-7,-5))
    ≤ |x+2|
    < ε  (since |x+2|<δ≤ε).

8.4 Workshop solutions for week 3

A video of me solving these problems is available, see [VIDEO]

  1. 1.
    1. (a)
      ​

      xn=50+1/n→50, but f⁢(xn)=7→7≠f⁢(50)=26. Hence f⁢(xn)↛f⁢(50), so f is discontinuous at 50.

    2. (b)
      ​

      Let xn be any sequence that converges to 49.9. We must prove that f⁢(xn)→f⁢(49.9)=26. So, let ε>0 be given. Since xn→49.9, there exists N∈ℤ+ such that, for all n≥N, |xn-49.9|<0.1, and hence xn<49.9+0.1=50. Hence, for all n≥N, f⁢(xn)=26. So, for all n≥N, |f⁢(xn)-26|=0<ε. Hence f⁢(xn)→f⁢(49.9).

  2. 2.
    ​

    I claim that f′⁢(1)=5. Proof: Given any ε>0, let δ=ε/2. Then for all x∈ℝ with 0<|x-1|<δ,

    |f⁢(x)-f⁢(1)x-1-5| = |2⁢x2+x-3x-1-5|
    = |(2⁢x+3)-5|
    = 2⁢|x-1|
    < 2⁢δ
    = ε.

    Hence

    limx→1⁡f⁢(x)-f⁢(1)x-1=5,

    so f is differentiable at 1 (with f′⁢(1)=5).

  3. 3.
    1. (a)
      ​

      (g∘f)′⁢(1)=g′⁢(-1)⁢f′⁢(1)=14.

    2. (b)
      ​

      h′⁢(1)=f′⁢(f⁢(1)2)×2⁢f⁢(1)⁢f′⁢(1)=-8.

  4. 4.
    1. (a)
      ​

      [0,1) is not open since it contains 0, but for all ε>0, (0-ε,0+ε) is not a subset of [0,1) (since it contains -ε/2, for example).

    2. (b)
      ​

      ℝ\[0,1)=(-∞,0)∪[1,∞) is not open since it contains 1, but for all ε>0, (1-ε,1+ε) is not a subset of ℝ\[0,1).

    3. (c)
      ​

      ℝ\[0,1]=(-∞,0)∪(1,∞) is open. If x>1 we may choose ε=x-1>0. If x<0, we may choose ε=-x>0. In either case, (x-ε,x+ε)⊂ℝ\[0,1].

    4. (d)
      ​

      ℝ\{2n:n∈ℤ} is not open since it contains 0, but for all ε>0 there exists N∈ℤ+ such that 2N>1/ε, whence 0<2-N<ε. So 2-N∈(0-ε,0+ε) but 2-N∉ℝ\{2n:n∈ℤ}.

8.6 Workshop solutions for week 4

  1. 1.
    1. (a)
      ​

      ∀x∈D, f⁢(x)≤f⁢(c).
      D could be any set. In particular, it need not be a subset of ℝ.

    2. (b)
      ​

      We can deduce that f′⁢(b)≥0.
      Proof: Let xn be any sequence in [a,b) converging to b. Since

      limx→b⁡f⁢(x)-f⁢(b)x-b=f′⁢(b),

      the sequence

      zn=f⁢(xn)-f⁢(b)xn-b→f′⁢(b)

      (Theorem 1.35). But f attains a maximum at b, so for all n∈ℤ+, f⁢(xn)≤f⁢(b). Further, xn∈[a,b) so xn<b. Hence, for all n, f⁢(xn)-f⁢(b)≤0 and xn-b<0, so zn≥0. Hence, f′⁢(b)=lim⁡zn≥0 (Proposition 1.7). □

  2. 2.
    1. (a)
    2. (b)
      ​

      I claim that f is differentiable at 0, and f′⁢(0)=1.

      Proof: By Theorem 1.35, it suffices to show that, for all sequences (xn) in (0,1] such that xn→0,

      f⁢(xn)-f⁢(0)xn-0=f⁢(xn)xn→1.

      To prove this, note that, for all x∈(0,1), there exists some (unique) n∈ℤ+ such that x∈(1/(n+1),1/n]. Then

      f⁢(x)=1n≥x,

      and

      x > 1n+1
      ⇒ 1x < n+1
      ⇒ n > 1x-1=1-xx
      ⇒ f⁢(x)=1n < x1-x.

      Hence, for all x∈(0,1),

      x≤f⁢(x)<x1-x.

      [Aside: can you see this on your graph? Add the curves y=x and y=x/(1-x).] Hence

      1≤f⁢(xn)xn<11-xn

      and so f⁢(xn)/xn→1 by the Squeeze Rule and the Algebra of Limits.

    3. (c)
      • •
        ​

        Bounded? Yes, above by 1, below by 0

      • •
        ​

        Differentiable? No. for example, it’s discontinuous at 1/2, so can’t be differentiable at 1/2.

      • •
        ​

        Continuous? No, as already observed.

      • •
        ​

        Surjective? No. Since it’s bounded and has codomain ℝ, it can’t be surjective. For example, it never attains the value -2.

      • •
        ​

        Injective? No. For example f⁢(3/4)=f⁢(1)=1.

      • •
        ​

        Monotonic? Yes, it’s increasing. Note that this means that for all x,y∈[0,1], if x<y then f⁢(x)≤f⁢(y). (Note the non-strict inequality.)
