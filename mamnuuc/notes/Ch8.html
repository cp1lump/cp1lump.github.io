Chapter 8 Workshop problems and solutions

8.1 Workshop questions for week 2

  1. 1.
    ​

    Let f:D→ℝ (read as “let f map D to ℝ"), where D⊆ℝ. Write down precise mathematical formulations of the following statements, using quantifiers (∀, ∃):

    1. (a)
      ​

      D is bounded below.

    2. (b)
      ​

      f is unbounded above.

    3. (c)
      ​

      f is surjective.

    4. (d)
      ​

      f is not surjective.

  2. 2.
    ​

    Prove from first principles (i.e. give a direct ε–N proof) that the following sequence converges:

    an=n2+(-1)nn2+2
  3. 3.
    ​

    Prove from first principles (i.e. give a direct ε–K proof) that

    limx→∞⁡2⁢x2+3⁢x+4x2+1=2.
  4. 4.
    ​

    Give a direct ε–δ proof that limx→-2⁡x+2x3+8=112.

8.2 Workshop solutions for week 2

A video of me solving these problems is available, see [VIDEO].

  1. 1.
    1. (a)
      ​

      ∃K∈ℝ,∀x∈D,x≥K.

    2. (b)
      ​

      ∀K∈ℝ,∃x∈D,f⁢(x)>K.

    3. (c)
      ​

      ∀y∈ℝ,∃x∈D,f⁢(x)=y.

    4. (d)
      ​

      ∃y∈ℝ,∀x∈D,f⁢(x)≠y.

  2. 2.
    ​

    I claim that an→1. Proof: let ε>0 be given. Then there exists N∈ℤ+ such that N>3/ε. Now, for all n≥N,

    |an-1| = |2-(-1)nn2+2|≤3n2+1<3n2≤3N2<ε.

    Hence an→1.

  3. 3.
    ​

    For all x∈(0,∞),

    |2⁢x2+3⁢x+4x2+1-2| = |3⁢x+2x2+1|
    ≤ 3⁢xx2+1+2x2+1
    < 3⁢xx2+2x2
    = 3x+2x2.

    Let ε>0 be given. Let K=max⁡{1,5/ε}. Then, for all x>K,

    |2⁢x2+3⁢x+4x2+1-2| < 3x+2x2
    < 3K+2K2
    ≤ 3K+2K  (since K≥1)
    ≤ ε.
  4. 4.
    ​

    First note that the maximal domain of the function f⁢(x)=(x+2)/(x3+8) is D=ℝ\{-2}, and -2 is a cluster point of D. Let ε>0 be given. Then let δ=min⁡{1,ε}. Then for all x∈D such that 0<|x+2|<δ,

    |f⁢(x)-112| = |1x2-2⁢x+4-112|
    = |x2-2⁢x-812⁢(x2-2+4)|
    = |x-4|⁢|x+2|12⁢((x-1)2+3)
    ≤ |x-4|36⁢|x+2|
    < 736⁢|x+2|  (since |x+2|<1, so x-4∈(-7,-5))
    ≤ |x+2|
    < ε  (since |x+2|<δ≤ε).
