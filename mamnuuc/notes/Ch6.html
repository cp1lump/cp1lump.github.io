6.1 Swapping limits

There are many occasions in calculus where we must compute a double limit. For example, say we wish to compute the derivative of the function f=exp. We know that f⁢(x) is the limit of a convergent series, that is,

f⁢(x)=limk→∞⁡∑n=0kxnn!.

Its derivative at a∈ℝ, therefore, is

f′⁢(a)=limx→a⁡f⁢(x)-f⁢(a)x-a=limx→a⁡limk→∞⁡1x-a⁢∑n=0kxn-ann!,

assuming this exists. Wouldn’t it be nice if we could swap the order of these limits? Since

limx→a⁡xn-anx-a=n⁢an-1

we would then deduce that

f′⁢(a)=limk→∞⁡∑n=0k1n!⁢n⁢an-1=limk→∞⁡∑m=0k-1amm!=f⁢(a),

establishing a fundamental property of the exponential function (it’s differentiable, and equals its own derivative).

Sadly, this argument is fatally flawed because, in general, limits do not commute.

Example 6.1

Consider the function

f:ℝ2\{(0,0)}→ℝ,f⁢(x,y)=x2-y2x2+y2.

Its domain is the whole two-dimensional plane ℝ2 except for the single point (0,0). For any fixed x∈ℝ\{0}, we can compute

limy→0⁡f⁢(x,y)=limy→0⁡x2-y2x2+y2=x2x2=1.

Hence

limx→0⁡(limy→0⁡f⁢(x,y))=limx→0⁡1=1.

What if we swap the order of the two limits? For any fixed y∈ℝ\{0},

limx→0⁡f⁢(x,y)=limx→0⁡x2-y2x2+y2=-y2y2=-1.

Hence

limy→0⁡(limx→0⁡f⁢(x,y))=limy→0-1=-1.

Clearly, the order in which we evaluate the limits matters! □

Here’s another example to illustrate this, where one of the “limits" is disguised in the form of a Riemann integral.

Example 6.2

For each n∈ℤ+, let fn:[0,1]→ℝ be the function

fn⁢(x)={2⁢n2⁢x,0≤x≤12⁢n,2⁢n-2⁢n2⁢x,12⁢n≤x≤1n,0,1n≤x≤1

whose graph is depicted below:

Choose and fix any x∈(0,1]. Then, for all n>1/x, x>1/n so fn⁢(x)=0. Hence, for each fixed x∈(0,1], the real sequence fn⁢(x) is eventually 0. Hence fn⁢(x)→0. Furthermore fn⁢(0)=0 for all n, so this sequence also converges to 0. So, if we define g:[0,1]→ℝ to be the constant function g⁢(x)=0, then for all x∈[0,1], limn→∞⁡fn⁢(x)=g⁢(x).

Consider now the real sequence

an=∫01fn.

Does this sequence converge, and if so, what is its limit? We might be tempted to argue that

limn→∞⁡an=limn→∞⁡∫01fn⁢(x)⁢𝑑x=∫01limn→∞⁡fn⁢(x)⁢d⁢x=∫01g⁢(x)⁢𝑑x=0,

swapping the order of the limit as n→∞ and the “limit" defining the integral. But this is complete nonsense, since as we see immediately from the graph of fn,

an=12×1n×n=12,

and the limit of this (constant) sequence is 12! □

6.2 Pointwise convergence versus uniform convergence

Given a sequence of functions, fn:D→ℝ, where D⊆ℝ is some fixed set, what does it mean to say that (fn) converges to f:D→ℝ? The obvious answer is:

Definition 6.4

A sequence of functions fn:D→ℝ converges pointwise to a function f:D→ℝ if, for each fixed x∈D, the real sequence (fn⁢(x)) converges (in the sense of Definition 1.1) to the real number f⁢(x).

This is the sense in which the sequence of functions fn:[0,1]→ℝ in Example 6.2 converges to the function g⁢(x)=0. Here’s another instructive example:

Example 6.5

Consider the sequence fn:[0,1]→ℝ, fn⁢(x)=xn. If 0≤x<1 then

fn⁢(x)=xn→0.

If x=1, however,

fn⁢(1)=1n=1→1.

Hence (fn) converges pointwise to the discontinuous function

f⁢(x)={0,0≤x<1,1,x=1. (6.1)

This illustrates a key weakness of pointwise convergence: it doesn’t preserve continuity. Nonetheless, pointwise limits do have some nice properties.

Proposition 6.7

If (fn) converges pointwise, its limit is unique.

Proof.

Assume, towards a contradiction, that (fn) converges to both f and g, where f≠g. Then, by assumption, there exists c∈D such that f⁢(c)≠g⁢(c). But fn⁢(c)→f⁢(c) and fn⁢(c)→g⁢(c), which contradicts the uniqueness of the limit of a convergent real sequence. ∎

Similarly, we can show that pointwise limits have an Algebra of Limits property.

Proposition 6.8

Let (fn) converge to f pointwise and (gn) to g pointwise on D. Then

  1. (i)
    ​

    (fn+gn) converges to f+g pointwise on D and

  2. (ii)
    ​

    (fn⁢gn) converges to f⁢g pointwise on D.

Proof.

A much stronger, and more useful, notion of convergence is that of uniform convergence. To define this, it’s convenient to make a preliminary definition.

Definition 6.9

The sup norm of a bounded function f:D→ℝ is

∥f∥:=sup{|f(x)|:x∈D}.

In the case where f,g:[a,b]→ℝ are continuous, we can give ∥f-g∥ a geometric interpretation: it’s the maximum separation between (x,f⁢(x)) and (x,g⁢(x)) as x ranges over [a,b]. We may think of this as a “distance" between the functions f and g.

So ∥f-g∥ defines a sort of “distance" between bounded functions, in the same way that |a-b| defines a distance between real numbers a and b. This line of thought leads to the main definition of this chapter:

Definition 6.11

A sequence of bounded functions fn:D→ℝ converges uniformly to a function f:D→ℝ if the real sequence ∥fn-f∥→0.

Example 6.5 revisited Claim: The sequence fn:[0,1]→ℝ, fn⁢(x)=xn does not converge uniformly to f, the function defined in (6.1).

Proof.

For each fixed n,

|fn⁢(x)-f⁢(x)|=|xn-f⁢(x)|={xn,0≤x<1,0,x=1.

Hence

{|fn(x)-f(x)|:x∈[0,1]}={xn: 0≤x<1}∪{0}=[0,1)

and so

∥fn-f∥=sup{|fn(x)-f(x)|:x∈[0,1]}=sup[0,1)=1

for all n. This sequence clearly doesn’t converge to 0. ∎

So pointwise convergence certainly does not imply uniform convergence. However, the converse holds:

Theorem 6.12

If a sequence fn:D→ℝ converges uniformly to f:D→ℝ, it converges pointwise to f.

Proof.

Assume the sequence of functions fn:D→ℝ converges uniformly to f:D→ℝ. Then, for each fixed x∈D,

0≤|fn(x)-f(x)|≤sup{|fn(y)-f(y)|:y∈D}=∥fn-f∥→0,

so |fn⁢(x)-f⁢(x)|→0 by the Squeeze Rule. Hence fn⁢(x)→f⁢(x), Since this holds for each x∈D, fn converges pointwise to f. ∎

Corollary 6.13

If (fn) converges uniformly, its limit is unique.

Proof.

Assume fn converges uniformly to both f and g. Then fn converges pointwise to f and fn converges pointwise to g (Theorem 6.12). Hence f=g (Proposition 6.7). ∎

Returning once more to Example 6.5, I claim that, given any b∈(0,1), the sequence of functions fn:[0,b]→ℝ, fn⁢(x)=xn does converge uniformly to the constant function f:[0,b]→ℝ, f⁢(x)=0.

Proof.

Now

∥fn-f∥=sup{|xn-0|: 0≤x≤b}=sup[0,bn]=bn.

But 0<b<1, so bn→0. ∎

So the sequence of functions fn⁢(x)=xn converges uniformly to 0 on [0,b] for all b∈[0,1).

We have seen that pointwise convergence doesn’t always preserve continuity. That is, a sequence of continuous functions can converge pointwise to a discontinuous function (Example 6.5). A key advantage of uniform convergence is that it does preserve continuity.

Theorem 6.14

Let fn:D→ℝ be a sequence of continuous functions converging uniformly to f:D→ℝ. Then f is continuous.

Before diving into the proof of this, let’s outline the strategy. We wish to show that the limit function f is continuous at a for each a∈D. Using the ε—δ criterion for continuity (Theorem 1.42), this amounts (roughly speaking) to showing that |f⁢(x)-f⁢(a)| can be made as small as we like by taking |x-a| sufficiently small. The key observation is that, for any fixed n∈ℤ+,

|f⁢(x)-f⁢(a)| = |f⁢(x)-fn⁢(x)+fn⁢(x)-fn⁢(a)+fn⁢(a)-f⁢(a)|
≤ |fn⁢(x)-f⁢(x)|+|fn⁢(x)-fn⁢(a)|+|fn⁢(a)-f⁢(a)|
≤ ∥fn-f∥+|fn⁢(x)-fn⁢(a)|+∥fn-f∥.

Now ∥fn-f∥ can be made as small as we like by choosing n suitably large (since fn converges to f uniformly, meaning ∥fn-f∥→0). Having chosen n suitably large, fn is a fixed continuous function, so |fn⁢(x)-fn⁢(a)| can be made as small as we like by taking |x-a| sufficiently small. OK, let’s write it up:

Proof of Theorem 6.14:  We must prove that f is continuous at a for all a∈D. By Theorem 1.42, it suffices to show that, for each ε>0, there exists δ>0 such that, for all x∈D with |x-a|<δ, |f⁢(x)-f⁢(a)|<ε.

So let a fixed ε>0 be given. Since fn converges to f uniformly, ∥fn-f∥→0. Hence, there exists N∈ℤ+ such that, for all n≥N, ∥fn-f∥<ε/3. In particular,

∥fN-f∥=sup{|fN(x)-f(x)|:x∈D}<ε3,

and hence

|fN⁢(x)-f⁢(x)|<ε3   for all x∈D. (6.2)

Furthermore, fN is, by assumption, continuous, so, by Theorem 1.42, there exists δ>0 such that, for all x∈D with |x-a|<δ,

|fN⁢(x)-fN⁢(a)|<ε3.

Hence, for all x∈D with |x-a|<δ,

|f⁢(x)-f⁢(a)| = |f⁢(x)-fN⁢(x)+fN⁢(x)-fN⁢(a)+fN⁢(a)-f⁢(a)|
≤ |fN⁢(x)-f⁢(x)|+|fN⁢(x)-fN⁢(a)|+|fN⁢(a)-f⁢(a)|
< ε3+|fN⁢(x)-fN⁢(a)|+ε3  by (6.2)
< ε.

□

This gives an indirect way of showing that fn⁢(x)=xn does not converge uniformly on [0,1]. Assume, towards a contradiction, that fn converges uniformly to some function f. Then fn converges pointwise to f (Theorem 6.12) and hence f must be the discontinuous function defined in equation (6.1) (by Proposition 6.7). But this contradicts Theorem 6.14 (because each fn is continuous on [0,1]).

6.3 Uniform convergence and calculus

We began this chapter by considering (see Example 6.2) a sequence of continuous, and hence Riemann integrable, functions fn:[0,1]→ℝ which converge pointwise to the constant function f=0, but whose integrals

∫01fn↛∫01f=0.

So pointwise convergence is insufficient for us to justify swapping the order of a limit (as n→∞) and an integral. However:

Theorem 6.16

Let fn:[a,b]→ℝ be a sequence of continuous functions converging uniformly to f:[a,b]→ℝ. Then

∫abfn→∫abf.
Proof.

First note that f is necessarily continuous (Theorem 6.14) and hence Riemann integrable (Theorem 4.26), so the proposed limit ∫abf certainly exists. Now, for all n,

0≤|∫abfn-∫abf| = |∫ab(fn-f)|  (Theorem 4.31)
≤ ∫ab|fn-f|  (Theorem 4.29)
≤ (b-a)⁢∥fn-f∥  (Theorem 4.28).

Hence, ∫abfn→∫abf by the Squeeze Rule. ∎

We deduce from this that the sequence fn:[0,1]→ℝ, of “triangle" functions defined in Example 6.2 does not converge uniformly. Note that its pointwise limit (the constant function 0) is continuous, so we couldn’t deduce this from Theorem 6.14.

Example 6.17

Claim: limn→∞⁡∫0πsin⁡(x+x2n)⁢𝑑x=2.

Proof.

Let fn:[0,π]→ℝ, fn⁢(x)=sin⁡(x+x2/n). Hopefully it’s clear that fn converges pointwise to the function sin since, for each fixed x∈[0,π], x+x2/n→x and so fn⁢(x)→sin⁡x (assuming, that is, that sin is continuous – which we actually haven’t proved yet. It’s coming in the next chapter). If we can show that fn converges to sin uniformly, the claim immediately follows, since ∫0πsin=2.

So, it remains to show that fn→sin uniformly. Fix n and x, and apply the Mean Value Theorem to the function sin on the interval [x,x+x2/n]: there exists c∈[x,x+x2/n] such that

fn⁢(x)-sin⁡x(x+x2/n)-x=cos⁡c.

Hence

|fn⁢(x)-sin⁡x| = |cos⁡c|⁢|x2/n|≤x2n.

This inequality holds for all x, and hence

∥fn-sin∥ = sup{|fn(x)-sinx|: 0≤x≤π}≤π2n.

So fn converges to sin:[0,π]→ℝ uniformly (by the Squeeze Rule). Hence, by Theorem 6.16,

∫0πfn → ∫0πsin⁡x⁢d⁢x
= [-cos⁡x]0π=2

by the Fundamental Theorem of the Calculus (version 2). ∎

Note we have no hope of computing the integrals ∫0πfn exactly (I invite you to try to write down an antiderivative for fn), so this limit can’t be computed directly.

Our next goal is to prove an analogue of Theorem 6.16 for derivatives. Under what circumstances can we be sure that a sequence fn of differentiable functions converges to a differentiable function? The functions fn in the following theorem are assumed to be continuously differentiable, meaning that each function fn:[a,b]→ℝ is differentiable and that its derivative fn′:[a,b]→ℝ is continuous. (Recall that every differentiable function is continuous, but its derivative isn’t necessarily continuous: see Example 3.14 for a counterexample.)

Theorem 6.19

Let fn:[a,b]→ℝ be a sequence of continuously differentiable functions which converge pointwise to f:[a,b]→ℝ, and whose derivatives fn′:[a,b]→ℝ converge uniformly to g:[a,b]→ℝ. Then f is continuously differentiable and f′=g.

Proof.

For each n∈ℤ+ let Fn:[a,b]→ℝ be defined by

Fn⁢(x)=∫axfn′.

Note that this exists since fn′ is, by assumption, continuous. By version 2 of the Fundamental Theorem of the Calculus,

Fn⁢(x)=fn⁢(x)-fn⁢(a).

By Theorem 6.16, for each fixed x,

fn⁢(x)-fn⁢(a)=Fn⁢(x)→∫axg,

since fn′ converges uniformly to g on [a,x]. But fn converges pointwise to f, so fn⁢(x)→f⁢(x) and fn⁢(a)→f⁢(a). Hence, fn⁢(x)-fn⁢(a)→f⁢(x)-f⁢(a) (by the Algebra of Limits). But limits are unique, so

f⁢(x)-f⁢(a)=∫axg.

Hence, by version 1 of the Fundamental Theorem of the Calculus, f is differentiable and f′⁢(x)=g⁢(x). ∎

At the moment, it’s hard to see why Theorem 6.19 is useful, because it’s hard to imagine a situation where we can check that fn′ converges uniformly to a continuous function g without already knowing that the limit f of fn is differentiable. For example, if we apply this Theorem to the sequence of functions in Exercise 6.18 we can deduce (at some effort) that the constant function f⁢(x)=1 is differentiable and has derivative 0 – hardly a startling revelation!

We will see in the next chapter that Theorem 6.19 can be extremely powerful when applied to the sequence of partial sums of a convergent power series. But before we can explore that application, we need to develop a way to prove that a sequence fn of functions converges uniformly to something f even when we can’t really say explicitly what f is (except tautologically: f⁢(x)=limn→∞⁡fn⁢(x)).

6.4 Completeness of the set of bounded functions

Recall that a real sequence (an) is Cauchy if, for each ε>0, there exists N∈ℤ+ such that, for all n,m≥N, |an-am|<ε (Definition 1.15), and that this condition turns out to be equivalent to being convergent, that is, (an) converges if and only if it is Cauchy (Theorem 1.19). This Theorem allows us to prove that a real sequence converges to something A∈ℝ even when we have no idea what A actually is. The aim of this section is to repeat this trick for uniform convergence of sequences of functions.

For a sequence of functions fn:D→ℝ, where D is some fixed subset of ℝ, we can define what it means to be uniformly Cauchy, in much the same way that we defined uniform convergence (Definition 6.11):

Definition 6.20

A sequence of bounded functions fn:D→ℝ is uniformly Cauchy if, for each ε>0, there exists N∈ℤ+ such that, for all n,m≥N, ∥fn-fm∥<ε.

As usual, ∥⋅∥ denotes the sup norm here. Note that this is only well defined for bounded functions, which is why Definition 6.20 specifies that each fn:D→ℝ is bounded.

Our goal in this section is to prove that a sequence of bounded functions is uniformly convergent if and only if it is uniformly Cauchy. The “only if" direction of this statement is much easier to prove than the “if" direction, so let’s start with that.

Theorem 6.22

Let fn:D→ℝ be a sequence of bounded functions converging uniformly to a bounded function f:D→ℝ. Then (fn) is uniformly Cauchy.

The proof of this follows the proof of Lemma 1.16 (the analogous statement for real sequences) almost word for word. To make this work, we need a “triangle inequality" for the sup norm.

Lemma 6.23

For all bounded functions f,g:D→ℝ,

∥f+g∥≤∥f∥+∥g∥.
Proof.

For all x∈D, |f⁢(x)+g⁢(x)|≤|f⁢(x)|+|g⁢(x)|≤∥f∥+∥g∥, since ∥f∥ is an upper bound on {|f⁢(x)|:x∈D} and ∥g∥ is an upper bound on {|g⁢(x)|:x∈D}. Hence, ∥f∥+∥g∥ is an upper bound on {|f⁢(x)+g⁢(x)|:x∈D}. But ∥f+g∥ is, by definition, the least upper bound on this set. Hence ∥f+g∥≤∥f∥+∥g∥. ∎

Proof of Theorem 6.22:  Let ε>0 be given. Since (fn) converges to f uniformly, there exists N∈ℤ+ such that, for all n≥N, ∥fn-f∥<ε/2. Hence, for all n,m≥N,

∥fn-fm∥=∥fn-f-(fm-f)∥≤∥fn-f∥-∥fm-f∥<ε2+ε2

by Lemma 6.23. □

OK, now for the hard(er) part. We first note that, if (fn) is uniformly Cauchy, it is certainly pointwise Cauchy, hence pointwise convergent. That is:

Lemma 6.24

Let fn:D→ℝ be a uniformly Cauchy sequence of bounded functions. Then (fn) converges pointwise to some function f:D→ℝ.

Proof.

Choose and fix x∈D. I claim that the real sequence (fn⁢(x)) is Cauchy. To see this, let ε>0 by given. Since (fn) is uniformly Cauchy, there exists N∈ℤ+ such that, for all n,m≥N, ∥fn-fm∥<ε. Hence, for all n,m≥N,

|fn⁢(x)-fm⁢(x)|≤∥fn-fm∥<ε.

Since the real sequence (fn⁢(x)) is Cauchy, it converges, by Theorem 1.19, to some limit f⁢(x)∈ℝ. Allowing x to vary in D, we obtain a function f:D→ℝ to which (fn) converges pointwise. ∎

Note that we don’t (yet) know that the pointwise limit f:D→ℝ of our uniformly Cauchy sequence of bounded functions is itself a bounded function! We need to prove this.

Lemma 6.25

Let a uniformly Cauchy sequence of bounded functions fn:D→ℝ converge pointwise to f:D→ℝ. Then f is bounded.

Proof.

We must show that there exists M>0 such that, for all x∈D, |f⁢(x)|≤M.

Since (fn) is uniformly Cauchy, there exists N∈ℤ+ such that, for all n,m≥N, ∥fn-fm∥<1. Furthermore, fN:D→ℝ is a bounded function, so there exists K>0 such that, for all x∈D, |fN⁢(x)|≤K. I claim that |f| is bounded above by K+1.

To see this, choose and fix x∈D. Then, for all n∈ℤ+,

|f⁢(x)| = |f⁢(x)-fn⁢(x)+fn⁢(x)-fN⁢(x)+fN⁢(x)|
≤ |f⁢(x)-fn⁢(x)|+|fn⁢(x)-fN⁢(x)|+|fN⁢(x)|
≤ |f⁢(x)-fn⁢(x)|+∥fn-fN∥+K

Hence, for all n≥N,

|f⁢(x)|≤|f⁢(x)-fn⁢(x)|+1+K. (6.3)

But, by assumption, fn⁢(x)→f⁢(x), so the sequence on the right hand side of (6.3) converges to 1+K. Since this sequence is bounded below by |f⁢(x)|, it follows that 1+K≥|f⁢(x)| (Proposition 1.7). This is true whatever x∈D we choose, so |f| is bounded above by M=K+1. ∎

So we now know that every uniformly Cauchy sequence of bounded functions converges pointwise to a bounded function. The last job is to show that this convergence is actually uniform (recall uniform convergence implies pointwise convergence, but not vice versa, so this is not automatic).

Theorem 6.26

A sequence fn:D→ℝ of bounded functions converges uniformly if and only if it is uniformly Cauchy.

Proof.

We have already proved the “only if" part (Theorem 6.22).

Assume that (fn) is uniformly Cauchy. Then (fn) converges pointwise to some bounded function f:D→ℝ by Lemmas 6.24 and 6.25. We must show that ∥fn-f∥→0.

Let ε>0 be given. Since (fn) is uniformly Cauchy, there exists N∈ℤ+ such that, for all n,m≥N, ∥fn-fm∥<ε/4. I claim that, for all n≥N,

∥fn-f∥<ε.

To see this, choose and fix x∈D. Then, for all n,m≥N,

|fn⁢(x)-f⁢(x)| = |fn⁢(x)-fm⁢(x)+fm⁢(x)-f⁢(x)| (6.4)
≤ |fn⁢(x)-fm⁢(x)|+|fm⁢(x)-f⁢(x)|
≤ ∥fn-fm∥+|fm⁢(x)-f⁢(x)|
< ε4+|fm⁢(x)-f⁢(x)|.

Note this inequality holds for all m≥N. Now (fn) converges pointwise to f, so fn⁢(x)→f⁢(x). Hence, there exists N1∈ℤ+ (depending on x) such that, for all m≥N1, |fm⁢(x)-f⁢(x)|<ε/4. Applying (6.4) in the case m=max⁡{N,N1}, we see that, for all n≥N,

|fn⁢(x)-f⁢(x)|<ε4+ε4=ε2.

This inequality holds for all x∈D, and N is independent of x, so, for all n≥N,

∥fn-f∥=sup{|fn(x)-f(x)|:x∈D}≤ε2<ε.

Since ε>0 was arbitrary, ∥fn-f∥→0. ∎

Let us denote the set of bounded functions on D

B⁢(D):={f:D→ℝ:f is bounded}

and the set of continuous functions on D

C⁢(D):={f:D→ℝ:f is continuous}

For general sets D, neither of these is a subset of the other. For example f⁢(x)=x2 is in C⁢(ℝ) but not B⁢(ℝ), and g⁢(x)=1 for x≥0, g⁢(x)=0 for x<0 is in B⁢(ℝ) but not C⁢(ℝ). In the case where D=[a,b], a closed bounded interval, however, every continuous function is bounded (by the Extreme Value Theorem), so C⁢([a,b])⊂B⁢([a,b]).

We have just shown that the set B⁢(D) enjoys (with respect to the sup norm) a fundamental property of the set ℝ (with respect to absolute value): every Cauchy sequence in the set converges to a limit in the set. Mathematicians call this property completeness. So what we have just shown is that B⁢(D) is complete with respect to the sup norm. In the case where D=[a,b], it’s not hard to deduce that C⁢(D) is also complete with respect to the sup norm.

Theorem 6.27

If a sequence of continuous functions fn:[a,b]→ℝ is uniformly Cauchy, then it converges uniformly to some continuous function f:[a,b]→ℝ.

Proof.

Each fn is a bounded function (by the Extreme Value Theorem), so (fn) converges uniformly to some bounded function f:[a,b]→ℝ by Theorem 6.26. But f is continuous by Theorem 6.14. ∎

The neat thing about Theorems 6.26 and 6.27 is that they allow us to prove that a sequence of functions converges uniformly without knowing what its limit is. As we will see, this is extremely useful when we come to consider functions defined by power series.

Summary

  • •
    ​

    A sequence of functions fn:D→ℝ converges pointwise to f:D→ℝ if, for each x∈D, fn⁢(x)→f⁢(x).

  • •
    ​

    The sup norm of a function f:D→ℝ is ∥f∥=sup{|f(x)|:x∈D}.

  • •
    ​

    A sequence of functions fn:D→ℝ converges uniformly to f:D→ℝ if the real sequence ∥fn-f∥ converges to 0.

  • •
    ​

    If fn converges uniformly to f, it converges pointwise to f. The converse is false.

  • •
    ​

    If a sequence of continuous functions fn:D→ℝ converges uniformly to f then f is continuous.

  • •
    ​

    If a sequence of continuous functions fn:[a,b]→ℝ converges uniformly to f then ∫abfn→∫abf.

  • •
    ​

    If a sequence of continuously differentiable functions fn:[a,b]→ℝ converges pointwise to f, and its sequence of derivatives fn′:[a,b]→ℝ converges uniformly to g, then f is differentiable and f′=g.

  • •
    ​

    A sequence fn:D→ℝ of bounded functions is uniformly Cauchy if, for each ε>0, there exists N∈ℤ+ such that, for all n,m≥N, ∥fn-fm∥<ε.

  • •
    ​

    A sequence of bounded functions is uniformly convergent if and only if it is uniformly Cauchy.
