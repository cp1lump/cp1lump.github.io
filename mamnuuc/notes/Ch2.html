Differentiable functions

Chapter 2 Differentiable functions

2.1 The main definition

Now that we have a rigorous definition of the limit of a function, it is straightforward to define derivatives.

Definition 2.1

Let f:D→ℝ, where D is some subset of ℝ, and a∈D be a cluster point of D. Then f is differentiable at a if the limit

limx→a⁡f⁢(x)-f⁢(a)x-a

exists. In this case, we denote the limit f′⁢(a) and call it the derivative of f at a. We say that f is differentiable if it is differentiable at a for all a∈D.

Remarks

  • •
    ​

    In general, in order to define limx→a⁡g⁢(x), we only need a to be a cluster point of the domain of g: it isn’t necessary in general for a to be in the domain of g, so g⁢(a) may or may not exist. For example

    limx→1⁡x2-1x-1

    exists, although the function is undefined at x=1. Note, however, that the limit defining f′⁢(a) contains the number f⁢(a), so to be differentiable at a, the point a must be both an element and a cluster point of the domain of f. For example, a function f:(0,∞)→ℝ cannot be differentiable at 0 (since 0∉(0,∞)), and a function f:ℤ→ℝ cannot be differentiable anywhere (since ℤ has no cluster points).

  • •
    ​

    The limit in Definition 2.1 is defined as in Definition 1.29. That is, f:D→ℝ is differentiable at a if there exists a real number, denoted f′⁢(a), such that, for each ε>0, there exists δ>0 such that, for all x∈D with 0<|x-a|<δ,

    |f⁢(x)-f⁢(a)x-a-f′⁢(a)|<ε.
  • •
    ​

    Geometrically, the quantity

    s⁢(x)=f⁢(x)-f⁢(a)x-a

    is the slope of the straight line passing through the points (a,f⁢(a)) and (x,f⁢(x)). Such a straight line is called a chord on the graph y=f⁢(x). As x approaches a, the points defining the chord get “arbitrarily close” to one another, and the chords approach a straight line through (a,f⁢(a)) with slope f′⁢(a), the so-called tangent line. This is why the derivative f′⁢(a) is often interpreted as the slope of the graph y=f⁢(x) at the point (a,f⁢(a)). Note, however, that this is just an interpretation. The definition of f′⁢(a) is in terms of limits which are, in turn, defined precisely in Definition 1.29.

  • •
    ​

    If f:D→ℝ is differentiable, then its derivative is also a function f′:D→ℝ, mapping D to ℝ. There is a very popular alternative notation for the derivative in this case, namely

    f′⁢(x)=d⁢fd⁢x.

    This notation is convenient in some circumstances, but it tends to blur the distinction between a function (in this case f′) and the value of the function at a particular point (in this case f′⁢(x)), so we will tend to avoid it.

Let’s verify that some simple, familiar functions have the derivatives we expect. In each case, we will give a direct ε—δ proof that the limit

limx→a⁡f⁢(x)-f⁢(a)x-a

exists and has the expected value.

Example 2.2

Let f:ℝ→ℝ such that f⁢(x)=c, a constant. Then f is differentiable (everywhere) and f′⁢(a)=0 for all a∈ℝ.

Proof.

We must show that, for each ε>0 there is some δ>0 such that

|f⁢(x)-f⁢(a)x-a-0|<ε

for all x satisfying 0<|x-a|<δ. So, let ε>0 be given. Let δ=1>0. Then for all x∈ℝ with 0<|x-a|<δ,

|f⁢(x)-f⁢(a)x-a-0| = |c-cx-a|=0<ε.

∎

So, for a constant function, the same δ will work for every ε>0. (We chose δ=1 but any other δ>0 works equally well.) Here’s another example that’s equally obliging:

Example 2.3

Let f:ℝ→ℝ such that f⁢(x)=x. Then f is differentiable (everywhere) and f′⁢(a)=1 for all a∈ℝ.

Proof.

We must show that, for each ε>0 there is some δ>0 such that

|f⁢(x)-f⁢(a)x-a-1|<ε

for all x satisfying 0<|x-a|<δ. So, let ε>0 be given. Let δ=1>0. Then for all x∈ℝ with 0<|x-a|<δ,

|f⁢(x)-f⁢(a)x-a-1|=|x-ax-a-1|=0<ε.

∎

The next examples are not quite so straightforward.

Example 2.4

Let f:ℝ→ℝ such that f⁢(x)=x2. Then f is differentiable and f′⁢(a)=2⁢a for all a∈ℝ\{0}.

Proof.

Choose and fix a∈ℝ. For any given ε>0, let δ=ε. Then for all x∈ℝ with 0<|x-a|<δ,

|f⁢(x)-f⁢(a)x-a-2⁢a| = |x2-a2x-a-2⁢a|=|x+a-2⁢a|=|x-a|<δ=ε.

∎

Example 2.5

Let f:ℝ\{0}→ℝ such that f⁢(x)=1/x. Then f is differentiable and f′⁢(a)=-1/a2 for all a∈ℝ\{0}.

If you find this proof difficult to follow, it may help to watch this [VIDEO].

Choose and fix a≠0. For any given ε>0, let

δ=min⁡{|a|2,|a|3⁢ε2}>0.

Then, for all x∈ℝ\{0} with 0<|x-a|<δ,

|f⁢(x)-f⁢(a)x-a+1a2| = |(1/x)-(1/a)x-a+1a2|
= |a-xa⁢x⁢(x-a)+1a2|
= |-1a⁢x+1a2|

= 1|a|2⁢|x|⁢|x-a|

∎

OK, all is as we expected. What about functions which are not differentiable? How do we prove they aren’t? A function f:D→ℝ is not differentiable at a precisely if the limit

limx→a⁡f⁢(x)-f⁢(a)x-a

does not exist. We can prove this by thinking carefully about the negation of the statement defining a limit (Definition 1.29), but it’s usually easier to exploit Theorem 1.35. Recall that this says that a function, g say, has limit L at a if and only if, for all sequences (xn) in D\{a} converging to a, g⁢(xn) converges to L. So to prove that limx→a⁡g⁢(x) doesn’t exist, it’s enough to find just one sequence (xn) in D\{a} converging to a whose image sequence (g⁢(xn)) does not converge.

Example 2.7

Let f:ℝ→ℝ such that f⁢(x)=|x|. Then f is not differentiable at 0.

Proof.

Assume, towards a contradiction, that f is differentiable at 0 with derivative f′⁢(0). Then

limx→0⁡f⁢(x)-f⁢(0)x-0=f′⁢(0).

Let xn=(-1)n/n. Note that this is a sequence in ℝ\{0} converging to 0. Hence, by Theorem 1.35,

f⁢(xn)-f⁢(0)xn-0→f′⁢(0).

But

f⁢(xn)-f⁢(0)xn-0=|(-1)n/n|(-1)n/n=1(-1)n=(-1)n

which does not converge, a contradiction. Hence f is not differentiable at 0. ∎

Remark Alternatively, we could have defined yn=1/n and zn=-1/n and noted that both are sequences in ℝ\{0} converging to 0, so if f is differentiable at 0, then both

f⁢(yn)-f⁢(0)yn-0 and f⁢(zn)-f⁢(0)zn-0

must converge to f′⁢(0). But the first converges to 1, while the second converges to -1, which contradicts the uniqueness of limits (Theorem 1.33).

There is a link between differentiablity and continuity, as we now show.

Proposition 2.8

Let f:D→ℝ be differentiable at a∈D. Then f is continuous at a.

Proof.

By Theorem 1.42, it suffices to show that limx→a⁡f⁢(x)=f⁢(a). Define

s:D\{a}→ℝ,s⁢(x)=f⁢(x)-f⁢(a)x-a.

Then, by assumption, s has a limit (denoted f′⁢(a)) at a. But for all x∈D\{a},

f⁢(x)=f⁢(a)+(x-a)⁢s⁢(x),

and hence, by the Algebra of Limits (Theorem 1.34),

limx→a⁡f⁢(x)=f⁢(a)+0×f′⁢(a)=f⁢(a).

∎

Remark. The converse of Proposition 2.8 is false: if f is continuous at a, a cluster point of its domain, it does not follow that f is differentiable there. We’ve already seen a counteraxample: f⁢(x)=|x| is continuous at 0 but is not differentiable at 0 (see Example 2.7). In this case, the function fails to be differentiable only at a single isolated point. It is straightforward to construct functions which are differentiable only at a single isolated point.

Example 2.9

Let f:ℝ→ℝ be defined such that

f⁢(x)={xif x∈ℚ,x-x2if x∉ℚ.

I claim that f is differentiable at 0, with f′⁢(0)=1, but is not differentiable anywhere else.

Proof.

Let a∈ℝ and assume that f is differentiable at a. Then, by Proposition 2.8, f is continuous at a. For each n∈ℤ+ there exist rn∈ℚ and in∈ℝ\ℚ such that a<rn<a+1/n and a<in<a+1/n. Clearly rn→a and in→a by the Squeeze Rule. Hence, by the definition of continuity, f⁢(rn)→f⁢(a) and f⁢(in)→f⁢(a). But rn is rational, so f⁢(rn)=rn→a, and in is irrational, so f⁢(in)=in-in2→a-a2. Both these limits equal f⁢(a), so a=a-a2, whence a=0. So if f is differentiable at a, then a=0. Equivalently, f is not differentiable at any a≠0.

It remains to show that f is differentiable at 0 with f′⁢(0)=1, that is,

limx→0⁡f⁢(x)-f⁢(0)x-0=1.

In this case, it’s easier to use Theorem 1.35 than to give a direct ε—δ argument. So, let (xn) be any sequence in ℝ\{0} converging to 0, and consider

sn=f⁢(xn)-f⁢(0)xn-0=f⁢(xn)xn.

We must show that sn→1. If xn is rational, then |sn-1|=|xn/xn-1|=0, whereas if xn is irrational then

|sn-1|=|xn-xn2xn-1|=|xn|.

Hence, for all n, 0≤|sn-1|≤|xn|, so |sn-1|→0 by the Squeeze Rule. Hence sn→1. It follows that f′⁢(0)=1, as claimed. ∎

So this (admittedly rather bizarre) function is differentiable at 0, and has a positive derivative there. Naive intuition would suggest, therefore, that the function should be increasing, at least for values of x sufficiently close to 0. In fact, this is false!

Example 2.9 continued Claim: the function f is not increasing on any neighbourhood of 0. That is, there does not exist ε>0 such that f is increasing on (-ε,ε).

Proof.

Let ε>0 be given. Then there exists an irrational number x such that 0<x<ε. By definition, f⁢(x)=x-x2<x. Similarly, there exists a rational number y such that x-x2<y<x. But then y<x and f⁢(y)=y>x-x2=f⁢(x). Hence, f is not increasing on the interval (-ε,ε). This is true no matter which positive number ε we choose, which establishes the claim. ∎

Your reaction to this may well be “so what, that’s a really crazy function – after all, it’s only differentiable at the single point 0." We will see later an example of a function f:ℝ→ℝ which is differentiable everywhere and has f′⁢(0)=1, but still is not increasing on any neighbourhood of 0.

2.2 The rules of differentiation

In principle, if we are given an explicit function, like

f:ℝ→ℝ,f⁢(x)=(x3+1)2,

we can show it is differentiable and compute its derivative by directly applying Definition 2.1. This quickly becomes complicated and tedious, however. So we next develop some “derivative theorems” which will allow us to reduce differentiation of many functions to an algorithmic process. You are (I would hope) already familiar with these and quite fluent in applying them. The point of this section is to show that, now we have a mathematically precise definition of the derivative, we can rigorously prove them, thus putting differential calculus on a solid foundation.

Proposition 2.10 (Linearity)

Let f:D→ℝ, g:D→ℝ be differentiable at a∈D with derivatives f′⁢(a) and g′⁢(a), respectively, and c be a real constant. Then

  1. (i)
    ​

    c⁢f:D→ℝ is differentiable at a with derivative c⁢f′⁢(a).

  2. (ii)
    ​

    f+g:D→ℝ is differentiable at a with derivative f′⁢(a)+g′⁢(a).

Proof.

These both follow immediately from the Algebra of Limits (Theorem 1.34):

limx→a⁡c⁢f⁢(x)-c⁢f⁢(a)x-a = limx→a⁡c⁢(f⁢(x)-f⁢(a)x-a)=c⁢f′⁢(a)
limx→a⁡f⁢(x)+g⁢(x)-(f⁢(a)+g⁢(a))x-a = limx→a⁡(f⁢(x)-f⁢(a)x-a+g⁢(x)-g⁢(a)x-a)=f′⁢(a)+g′⁢(a)

∎

Remark Actually the first part of this proof assumes that limx→a⁡c=c. Have we actually proved this? Not directly. (You should compose a direct ε–δ proof in your head now. For a given ε>0, how small must we take δ?) We have proved it indirectly however: constant functions are differentiable (Example 2.2), hence continuous (Proposition 2.8), so equal their limit at each point (Theorem 1.42).

Proposition 2.10 followed easily from the Algebra of Limits. Our next differentiation rule, and its proof, are considerably more subtle. Recall that the composition of two functions f:A→B and g:B→C, denoted g∘f:A→C is the function which first “does" f, then feeds the result into g, that is

(g∘f)⁢(x)=g⁢(f⁢(x)).

This only makes sense if the range of f is a subset of the domain of g. We’ve already proved that the composition of two continuous functions is continuous (Theorem 1.45). There’s a very famous rule which tells us how to compute the derivative of a composition of two functions, if we know the derivatives of its component pieces.

Theorem 2.11 (Chain Rule)

Let f:D→E be differentiable at a∈D and g:E→ℝ be differentiable at f⁢(a)∈E. Then g∘f:D→ℝ is differentiable at a, and

(g∘f)′⁢(a)=g′⁢(f⁢(a))⁢f′⁢(a).

To prove this, we will need to slightly alter the way we think of the derivative. The idea is that, if a function g:ℝ\{a}→ℝ has limit L at a, we can extend the function to the domain ℝ by defining g⁢(a)=L, and this extended function is continuous at a by Theorem 1.42. Conversely, if the function g:ℝ\{a}→ℝ has a continuous extension to ℝ, it has a limit at a and that limit is g⁢(a). So we can reinterpret “g has a limit at a” as saying that “g has an extension which is continuous at a.” Applying this idea to the difference quotient of a function f,

g⁢(x):=f⁢(x)-f⁢(a)x-a,

we obtain Carathéodory’s criterion for differentiablity:

Proposition 2.12 (Carathéodory’s Criterion)

Let f:D→ℝ and a∈D be a cluster point of D. Then f is differentiable at a∈D if and only if there exists a function ϕ:D→ℝ that is continuous at a and satisfies

f⁢(x)-f⁢(a)=ϕ⁢(x)⁢(x-a)        (♣)

for all x∈D. In this case, f′⁢(a)=ϕ⁢(a).

Proof.

(⇒): Assume f is differentiable at a with derivative f′⁢(a). Define the function

ϕ:D→ℝ,ϕ⁢(x)={f⁢(x)-f⁢(a)x-aif x≠af′⁢(a)if x=a.

Clearly ϕ satisifes (♣) for all x∈D\{a}, and (♣) holds automatically at x=a since both sides equal 0. Furthermore,

limx→a⁡ϕ⁢(x)=limx→a⁡f⁢(x)-f⁢(a)x-a=f′⁢(a)=ϕ⁢(a),

so ϕ is continuous at a by Theorem 1.42.

(⇐): Assume that a function continuous at a satisfying (♣) exists. Then, dividing (♣) by (x-a),

limx→a⁡f⁢(x)-f⁢(a)x-a=limx→a⁡ϕ⁢(x)=ϕ⁢(a)

by Theorem 1.42, since ϕ is continuous at a. Hence, f is differentiable at a, with derivative ϕ⁢(a). ∎

Proof of the Chain Rule:  By Carathéodory’s Criterion, there exist functions ϕ:D→ℝ and ψ:E→ℝ such that ϕ is continuous at a, ψ is continuous at f⁢(a) and, for all x∈D, and y∈E,

ϕ⁢(x)⁢(x-a)=f⁢(x)-f⁢(a),ψ⁢(y)⁢(y-f⁢(a))=g⁢(y)-g⁢(f⁢(a)).

Define the function

Φ:D→ℝ,Φ⁢(x)=ψ⁢(f⁢(x))⁢ϕ⁢(x).

By Theorem 1.45 (and the Algebra of Limits), Φ is continuous at a. Furthermore, for all x∈D,

Φ⁢(x)⁢(x-a) = ψ⁢(f⁢(x))⁢ϕ⁢(x)⁢(x-a)
= ψ⁢(f⁢(x))⁢(f⁢(x)-f⁢(a))
= g⁢(f⁢(x))-g⁢(f⁢(a)).

Hence, by Carathéodory’s Criterion, g∘f is differentiable at a, and its derivative is

Φ⁢(a)=ψ⁢(f⁢(a))⁢ϕ⁢(a)=g′⁢(f⁢(a))⁢f′⁢(a).

□

Example 2.13

Let f:ℝ→ℝ such that f⁢(x)=(x2+1)2. Then f is differentiable and f′⁢(x)=4⁢(x2+1)⁢x.

Proof.

f=h∘g where h,g:ℝ→ℝ

g⁢(x) = x2+1
h⁢(x) = x2.

Now, by Examples 2.4 and 2.2 and the Linearity Property (Proposition 2.10) g,h are differentiable, with derivatives

g′⁢(x) = 2⁢x+0
h′⁢(x) = 2⁢x.

Hence, by the Chain Rule, f is differentiable, and

f′⁢(x)=h′⁢(g⁢(x))⁢g′⁢(x)=2⁢g⁢(x)⁢2⁢x=4⁢(x2+1)⁢x.

∎

Remark You are probably more familiar with the Chain Rule expressed something like the following:

d⁢yd⁢x=d⁢yd⁢ud⁢ud⁢x.        (*)

Certainly, this has a look of plausibility about it (don’t the “d⁢u’s" just cancel?) and is arguably more memorable than

(g∘f)′(a)=g′(f(a))f′(a).        (†)

Nonetheless, it is vastly inferior. Ask yourself what, exactly, does (*) mean? The derivative of a function y (where? At x?) equals the derivative (where? At u?) of the same (?) function – but with respect to a different variable, whatever that means — times the derivative of some other function u (but isn’t u a variable? Can we use the same symbol to denote both a function and a variable?) evaluated somewhere (x? Maybe?). Note that “the function y thought of as a function of u instead of x," where “u is a function of x," is actually a different function! And to interpret (*) we’re forced to talk about how we think about the function y, instead of what it actually is. This is the hallmark of imprecise, badly formulated mathematics.

By contrast (†) is a completely unambiguous, precise statement standing on its own. We don’t need to explain separately “how we’re thinking of" the terms it contains in order to make sense of it. Be warned: if ever I ask for a statement of the Chain Rule, it is Theorem 2.11 I want, not wooly, ambiguous stuff in the style of (*). □

You may have expected us to prove the Product Rule before the Chain Rule: it is, after all, considerably simpler. The reason we proved the Chain Rule first is that we can deduce the Product Rule from it.

Proposition 2.15 (Product Rule)

Let f:D→ℝ, g:D→ℝ be differentiable at a∈D with derivatives f′⁢(a) and g′⁢(a), respectively. Then f⁢g:D→ℝ is differentiable at a with derivative f′⁢(a)⁢g⁢(a)+f⁢(a)⁢g′⁢(a).

Proof.

Let s:ℝ→ℝ such that s⁢(x)=x2 and recall that this is differentiable with derivative s′⁢(x)=2⁢x by Example 2.4. Hence, by the Chain Rule and the Linearity Property, s∘f, s∘g and s∘(f+g) are all differentiable at a, with derivatives

(s∘f)′⁢(a) = 2⁢f⁢(a)⁢f′⁢(a)
(s∘g)′⁢(a) = 2⁢g⁢(a)⁢g′⁢(a)
(s∘(f+g))′⁢(a) = 2⁢(f⁢(a)+g⁢(a))⁢(f′⁢(a)+g′⁢(a)).

Hence, by Proposition 2.10,

f⁢g=12⁢{(f+g)2-f2-g2}=12⁢{s∘(f+g)-s∘f-s∘g}

is differentiable at a with derivative

(f⁢g)′⁢(a) = 12⁢{2⁢(f⁢(a)+g⁢(a))⁢(f′⁢(a)+g′⁢(a))-2⁢f⁢(a)⁢f′⁢(a)-2⁢g⁢(a)⁢g′⁢(a)}
= f′⁢(a)⁢g⁢(a)+f⁢(a)⁢g′⁢(a).

∎

Example 2.18

Let m be a positive integer and f:ℝ\{0}→ℝ be the function f⁢(x)=1/xm. Then f is differentiable, and f′⁢(x)=-m/xm+1.

Proof.

f=h∘g where g:ℝ\{0}→ℝ, g⁢(x)=1/x and h:ℝ→ℝ, h⁢(x)=xm. By Examples 2.5 and 2.17, both g and h are differentiable with derivatives

g′⁢(x) = -1x2
h′⁢(x) = m⁢xm-1.

Hence, by the Chain Rule, f is diffentiable, and

f′⁢(x)=h′⁢(g⁢(x))⁢g′⁢(x)=m⁢g⁢(x)m-1⁢(-1x2)=-m⁢(1x)m-1⁢(1x)2=-mxm+1

by Example 2.5 and Proposition 2.17. ∎

Let f:D→ℝ and g:D→ℝ\{0}, be differentiable at a∈D. Then f/g is differentiable at a and

(fg)′⁢(a)=f′⁢(a)⁢g⁢(a)-f⁢(a)⁢g′⁢(a)g⁢(a)2.
Proof.

So we now know that the function f⁢(x)=xk is differentiable for all integers k (everywhere it is well-defined, i.e. everywhere except 0 if k<0), and that its derivative is f′⁢(x)=k⁢xk-1. We can also define f:(0,∞)→ℝ, f⁢(x)=xr where r is any rational number: if r=p/q, we define xr=(x1/q)p. It turns out that this function is also differentiable, with the expected derivative:

Proposition 2.20

Let r be a rational number and f:(0,∞)→ℝ be the function f⁢(x)=xr. Then f is differentiable and

f′⁢(x)=r⁢xr-1.
Proof.

Postponed. We’ll actually prove something more general once we’ve introduced (rigorously) the natural logarithm. ∎

It may occur to you that one could prove Proposition 2.20 by arguing as follows. Let r=p/q where p,q∈ℤ+, and define g⁢(x)=f⁢(x)q=xp. Then this is certainly differentiable, with derivative p⁢xp-1. Hence, by the Chain Rule,

p⁢xp-1 = q⁢f⁢(x)q-1⁢f′⁢(x)
⇒  f′⁢(x) = p⁢xp-1q⁢(xp/q)q-1=pq⁢xp-1-p⁢(q-1)q
= pq⁢xpq-1=r⁢xr-1.

The only thing wrong with this “proof” is that, in applying the Chain Rule (to F=h∘f where h⁢(x)=xq) you are presupposing that f is differentiable, that is, you’re assuming what you want to prove! So all this argument really shows is that if f is differentiable, then its derivative must be f′⁢(x)=r⁢xr-1.

Note that the claimed formula is certainly true for r=1/2 (see Exercise 2.6).

2.3 Open sets and the Localization Lemma

We will often have to deal with functions f:ℝ→ℝ that are defined piecewise, by combining different formulae for f⁢(x) valid on different subsets of the domain. A simple example is

f:ℝ→ℝ,f⁢(x)=|x|={x,x∈[0,∞),-x,x∈(-∞,0).

It would be handy if we could deduce information about the differentiablity of such functions directly from their constituent pieces. Luckily, we can, because limits, and hence derivatives, are inherently local objects. That is, if two functions f,g coincide for all x “close to" a point a, they have the same derivative at a. To formulate this statement precisely (and hence be able to prove it) we need to introduce the concept of open subsets of ℝ.

Definition 2.21

U⊆ℝ is open if, for each a∈U there exists δ>0 such that (a-δ,a+δ)⊆U.

Example 2.22
  1. (i)
    ​

    ℝ itself is open: given any a∈ℝ, (a-50,a+50)⊆ℝ, for example.

  2. (ii)
    ​

    (0,1) is open: given any a∈(0,1) we can take δ=min⁡{|a|,|1-a|}>0. Then (a-δ,a+δ)⊆(0,1).

  3. (iii)
    ​

    [0,1) is not open: 0∈[0,1) but, for all δ>0, (-δ,δ)⊈[0,1).

  4. (iv)
    ​

    ∅ is open (careful!).

We can now give a precise formulation of the idea that derivatives are local.

Lemma 2.24 (Localization Lemma)

Let f:D→ℝ coincide with a differentiable function g:U→ℝ on some open set U⊆D. Then f is differentiable, with derivative g′, on U.

Proof.

Let a∈U. We must show that

limx→a⁡f⁢(x)-f⁢(a)x-a=g′⁢(a).

Let ε>0 be given. Since U is open, there exists δ1>0 such that (a-δ1,a+δ1)⊆U⊆D. Since g is differentiable at a, there exists δ2>0 such that for all x∈U with 0<|x-a|<δ2,

|g⁢(x)-g⁢(a)x-a-g′⁢(a)|<ε.

Let δ=min⁡{δ1,δ2}>0. Then for all x∈D with 0<|x-a|<δ, x∈U so f⁢(x)=g⁢(x) (and f⁢(a)=g⁢(a)), and 0<|x-a|<δ2, so

|f⁢(x)-f⁢(a)x-a-g′⁢(a)|=|g⁢(x)-g⁢(a)x-a-g′⁢(a)|<ε.

∎

Example 2.25

The function f:ℝ→ℝ, f⁢(x)=|x| coincides with the differentiable function g⁢(x)=x on the open set (0,∞). Hence, by the Localization Lemma, f is differentiable on (0,∞) with derivative f′⁢(x)=1. Similarly, f coincides with the differentiable function h⁢(x)=-x on the open set (-∞,0). Hence, by the Localization Lemma, f is differentiable on (-∞,0) with derivative f′⁢(x)=-1.

Warning! The condition that U is an open set is crucial for the Localization Lemma. For example, f⁢(x)=|x| coincides with the differentiable function g⁢(x)=x on U=[0,∞), but f is not differentiable on U (it fails to be differentiable at 0).

Summary

  • •
    ​

    A function f:D→ℝ is differentiable at a∈D if

    limx→a⁡f⁢(x)-f⁢(a)x-a

    exists. In this case, we denote the limit f′⁢(a) and call it the derivative of f at a. We say that f is differentiable if it is differentiable at a for all a∈D.

  • •
    ​

    The limit in f′⁢(a) is defined as in Definition 1.29.

  • •
    ​

    If a function is differentiable at a, it is continuous at a. The converse is false.

  • •
    ​

    Using this definition of f′⁢(a), we can prove that derivatives obey the usual rules of differential calculus, to wit:

    Linearity:(α⁢f+β⁢g)′⁢(a)=α⁢f′⁢(a)+β⁢g′⁢(a),Product Rule:(f⁢g)′⁢(a)=f′⁢(a)⁢g⁢(a)+f⁢(a)⁢g′⁢(a),Quotient Rule:(f/g)′⁢(a)=f′⁢(a)⁢g⁢(a)-f⁢(a)⁢g′⁢(a)g⁢(a)2,Chain Rule:(g∘f)′⁢(a)=g′⁢(f⁢(a))⁢f′⁢(a).
  • •
    ​

    U⊆ℝ is open if for each a∈U there exists δ>0 such that (a-δ,a+δ)⊆U.

  • •
    ​

    Derivatives are local: if f:D→ℝ coincides with a differentiable function g:U→ℝ on some open set U⊆D, then f is differentiable, with derivative g′, on U.
