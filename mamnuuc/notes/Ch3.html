Functions differentiable on an interval

Chapter 3 Functions differentiable on an interval

Two applications of differential calculus are particularly pervasive in the natural sciences. In one, we use the derivative of a function to deduce its maximum (or minimum) value and the input which produces that value. In the second, we are given information, not about the function itself, but about its derivative (which is often interpreted as a “rate of change” with respect to time). We then seek to reconstruct the function itself from this information. These two applications rely fundamentally on the Interior Extremum Theorem, and the Mean Value Theorem, respectively, and the main purpose of this section is to state and prove these. Both are statements concerning functions which are differentiable (at least) on an open interval (a,b). The functions in question may be differentiable elsewhere too (perhaps on the whole of ℝ, in fact), so it’s important when you read “f is differentiable on (a,b)” not to add the unwarranted assumption “and only on (a,b)”.

3.1 The Interior Extremum Theorem

Recall that f:D→ℝ attains a maximum at a∈D if f⁢(x)≤f⁢(a) for all x∈D. Similarly, f attains a minimum at a∈D if f⁢(x)≥f⁢(a) for all x∈D. Note that these definitions have absolutely nothing to do with calculus! In fact, they make perfectly good sense whatever set D, the domain of f, is: it need not be a subset of ℝ!

We say that f attains an extremum at a∈D if it attains either a maximum or minimum at a. (The word “extremum” means “maximum or minimum”, in much the same way that “monotonic” means “increasing or decreasing”. Its plural is “extrema”.)

Theorem 3.1 (The Interior Extremum Theorem)

Let f:(a,b)→ℝ be differentiable and f attain an extremum at c∈(a,b). Then f′⁢(c)=0.

Proof.

Consider first the case that f attains a maximum at c∈(a,b). By assumption, f′⁢(c) exists. Hence, by Theorem 1.35, given any sequence (xn) in (a,b)\{c} which converges to c,

s⁢(xn):=f⁢(xn)-f⁢(c)xn-c→f′⁢(c). (3.1)

This holds, in particular, for the sequence xn=c+(b-c)/(n+1)∈(c,b). Note that xn>c for all c, so xn-c>0, and f⁢(c) is the maximum value of f, so f⁢(xn)-f⁢(c)≤0. Hence s⁢(xn)≤0 for all n so, by Proposition 1.7, its limit is non-positive, i.e. f′⁢(c)≤0.

But (3.1) also holds in the case where xn=c-(c-a)/(n+1)∈(a,c). Now xn-c<0 for all n, and f⁢(xn)-f⁢(c)≤0 (since, as before, f attains its maximum at c), so s⁢(xn)≥0 for all n. Hence, by Proposition 1.7, its limit is non-negative, i.e. f′⁢(c)≥0.

Hence, f′⁢(c)=0, as was to be shown.

Consider now the case that f attains a minimum at c∈(a,b). Let g=-f. Then g is differentiable on (a,b) and attains a maximum at c, so, as we just proved, g′⁢(c)=0. Hence f′⁢(c)=-g′⁢(c)=0. ∎

Definition 3.2

Let f:D→ℝ be differentiable. Then c∈D is a critical point of f if f′⁢(c)=0.

Given a differentiable function f:[a,b]→ℝ, we know it attains both a maximum and a minimum by the Extreme Value Theorem. Either or both of these extrema might occur at an endpoint of the interval [a,b]. But if they don’t, they occur at some interior point c∈(a,b), and this point must be a criticial point of f by the Interior Extremum Theorem. So to find the maximum and minimum values attained by f, we just need to evaluate it at the endpoints a,b and any interior critical points, and extract the extreme values.

Example 3.3

Find the maximum and minimum values attained by the function

f:[0,3]→ℝ,f⁢(x)=x2-2⁢x+7.

Solution: First note that, since f is continuous, it certainly attains both a maximum and a minimum by the Extreme Value Theorem (Theorem 1.48). Since f is differentiable on (0,3), it follows from the Interior Extremum Theorem that each extremum occurs either at an endpoint, that is, 0 or 3, or at an interior critical point of f. Now

f′⁢(x)=2⁢x-2=0

if and only if x=1, so 1 is the only critical point of f. Since f⁢(1)=6, f⁢(0)=7, and f⁢(3)=10, we deduce that f attains a maximum value of 10 at the right endpoint 3, and a minimum value of 6 at the interior critical point 1.

Remark In this case, using calculus was really overkill, since we could have deduced the same information by simply completing the square:

f⁢(x)=(x-1)2+6.

We can also use information about the derivative to show, in some circumstances, that the extrema of a differentiable function on [a,b] do not occur at its endpoints.

Proposition 3.4

Let f:[a,b]→ℝ be differentiable.

  1. (i)
    ​

    If f′⁢(a)>0, then f⁢(a) is not the maximum value attained by f.

  2. (ii)
    ​

    If f′⁢(a)<0, then f⁢(a) is not the minimum value attained by f.

  3. (iii)
    ​

    If f′⁢(b)>0, then f⁢(b) is not the minimum value attained by f.

  4. (iv)
    ​

    If f′⁢(b)<0, then f⁢(b) is not the maximum value attained by f.

Proof.

The proofs of each part are very similar, so we will just prove (i).

Assume f′⁢(a)>0. Assume further, towards a contradiction, that f does attain its maximum at a. Since f is differentiable at a,

sn=f⁢(xn)-f⁢(a)xn-a→f′⁢(a)>0,

for any sequence (xn) in (a,b] converging to a (by Theorem 1.35). But f⁢(xn)≤f⁢(a) by assumption, and xn>a, so sn≤0 for all n. Hence f′⁢(a)≤0 by Proposition 1.7, a contradiction. ∎

3.2 The Mean Value Theorem

Our next theorem has hypotheses (assumptions on f) which look very restrictive – so restrictive that one might wonder whether the theorem is of any practical use at all. In fact, as we shall see, it has very powerful and useful consequences.

Theorem 3.6 (Rolle’s Theorem)

Let f be continuous on [a,b] and differentiable on (a,b). Assume f⁢(a)=f⁢(b). Then there exists c∈(a,b) such that f′⁢(c)=0.

Proof.

By the Extreme Value Theorem, f attains both a maximum and a minimum value on [a,b]. If both of these occur at the endpoints of [a,b] then, since f⁢(a)=f⁢(b), the maximum value coincides with the minimum value, whence it follows that f:[a,b]→ℝ is constant. But then f′⁢(x)=0 for all x∈(a,b), so c=(b+a)/2, for example, has f′⁢(c)=0.

Hence, we may assume that either the maximum or the minimum value does not occur at an endpoint. But then f attains an extremum at some interior point c∈(a,b), so f′⁢(c)=0 by the Interior Extremum Theorem. ∎

Theorem 3.7 (The Mean Value Theorem)

Let f be continuous on [a,b] and differentiable on (a,b). Then there exists c∈(a,b) such that

f′⁢(c)=f⁢(b)-f⁢(a)b-a.
Proof.

Let α=(f⁢(b)-f⁢(a))/(b-a) and define g:[a,b]→ℝ such that

g⁢(x)=f⁢(x)-α⁢(x-a).

Then g is continuous, and is differentiable on (a,b). Furthermore g⁢(a)=f⁢(a) and g⁢(b)=f⁢(a). So g satisfies the hypotheses of Rolle’s Theorem, and we deduce that there exists c∈(a,b) such that g′⁢(c)=0. But

g′⁢(c)=f′⁢(c)-α

which completes the proof. ∎

Remarks

  • •
    ​

    The Mean Value Theorem has a nice geometric interpretation. Consider the graph y=f⁢(x) of a differentiable function f:ℝ→ℝ. For each pair of distinct numbers a<b, we can construct the chord (straight line) passing through the points (a,f⁢(a)) and (b,f⁢(b)) on the graph. Its slope is (f⁢(b)-f⁢(a))/(b-a). The Mean Value Theorem asserts that, at some point (c,f⁢(c)) on the graph between these two points, the tangent line to the graph is parallel to the chord.

  • •
    ​

    The Mean Value Theorem guarantees the existence of a point c with the required f′⁢(c). It says nothing about uniqueness: there could be more than one such point.

  • •
    ​

    Clearly Rolle’s Theorem is a special case of the Mean Value Theorem (geometrically, the case where the chord is horizontal). It’s slightly surprising at first sight, therefore, that Rolle’s Theorem actually implies the Mean Value Theorem.

You perhaps don’t realize it, but you have probably been using the Mean Value Theorem rather a lot. We have seen (Example 2.2) that if a function is constant then its derivative is zero everywhere. When one solves a differential equation, one often uses the converse fact: if a function has zero derivative everywhere, then it must be constant. But how do we know this? It’s a consequence of the Mean Value Theorem!

Example 3.8

Suppose we know that f:ℝ→ℝ is differentiable and satisfies f′⁢(x)=x2 for all x, and f⁢(0)=2. Then surely

f⁢(x)=x33+2,

right? But how do we know this? Certainly, F⁢(x):=x33+2 is an example of a function whose derivative at each x is x2, and whose value at 0 is 2, so it’s possible that f=F. But, perhaps there’s more than one function satisfying those properties?

In fact, we can prove that f=F, as follows. Let g:ℝ→ℝ be

g⁢(x)=f⁢(x)-F⁢(x)=f⁢(x)-x33-2.

Then, for all x,

g′⁢(x)=f′⁢(x)-x2=0.

Now assume, towards a contradiction, that g is not constant. Then there exist a,b∈ℝ with a<b such that g⁢(a)≠g⁢(b). But then, by the Mean Value Theorem applied to g on [a,b], there exists c∈(a,b) such that

g′⁢(c)=g⁢(b)-g⁢(a)b-a≠0

a contradiction since g′⁢(x)=0 everywhere. Hence, g is constant, so for all x∈ℝ g⁢(x)=g⁢(0)=0, that is, f=F. □

When we first learn calculus, we are taught to think of f′⁢(x) as the “rate of change of f with respect to the variable x”. So if, at a particular point a∈D, f′⁢(a)>0, we expect that f should be strictly increasing, at least sufficiently close to a. You’ve already seen (Example 2.9) that this expectation can be false: the relationship between f′ and the increasing/decreasing behaviour of f can be subtle. Before proceeding further, let’s make absolutely precise what we mean by increasing and decreasing functions.

Definition 3.9

f:D→ℝ is increasing if for all x,y∈D with x<y, f⁢(x)≤f⁢(y). It is strictly increasing if for all x,y∈D with x<y, f⁢(x)<f⁢(y). Similarly, f is decreasing if for all x,y∈D with x<y, f⁢(x)≥f⁢(y), and strictly decreasing if for all x,y∈D with x<y, f⁢(x)>f⁢(y).

Example 3.10

f:ℝ→ℝ, f⁢(x)=x3. This is strictly increasing: if x<y then

f⁢(x)-f⁢(y) = (x-y)⁢(x2+x⁢y+y2)
= (x-y)⁢((x+y/2)2+3⁢y2/4)
< 0

since ((x+y/2)2+3⁢y2/4)>0 unless x=y=0.

Note that Definition 3.9 makes no mention of the derivative of f and may, in principle, apply when f is not differentiable, or even continuous. For example, the step function defined in Example 1.38, is an increasing function. Our next result uses the Mean Value Theorem to establish some useful links between f′ and the increasing/decreasing behaviour of f.

We finish this section by studying a function f:ℝ→ℝ with some very counterintuitive properties.

Example 3.14

Consider the function f:ℝ→ℝ defined so that

f⁢(x)={x+2⁢x2⁢sin⁡(1/x)if x≠00if x=0

Claim: f is differentiable everywhere, has f′⁢(0)=1>0, but is not increasing on any neighbourhood of 0.

Proof.

f coincides with the function g:U→ℝ, g⁢(x)=x+2⁢x2⁢sin⁡(1/x) on the open set U=ℝ\{0}. g is differentiable by the Product and Chain Rules, so f is differentiable on U by the Localization Lemma, and for all a≠0,

f′⁢(a)=g′⁢(a)=1+4⁢a⁢sin⁡(1/a)-2⁢cos⁡(1/a). (3.2)

Consider now the case a=0. Let (xn) be any sequence in ℝ\{0} converging to 0. Then

sn:=f⁢(xn)-f⁢(0)xn-0=1+2⁢xn⁢sin⁡(1/xn).

Now -|xn|≤xn⁢sin⁡(1/xn)≤|xn|, so xn⁢sin⁡(1/xn)→0 by the Squeeze Rule, and hence sn→1 (by the Algebra of Limits). Hence, f is differentiable at 0 and f′⁢(0)=1.

For the final part, assume, to the contrary, that there exists ε>0 such that f:(-ε,ε)→ℝ is increasing. Then, by Proposition 3.11, f′⁢(x)≥0 for all x∈(-ε,ε). But there exists m∈ℤ+ such that m>1/(2⁢π⁢ε). Then x=1/(2⁢π⁢m)∈(0,ε) and, by equation (3.2),

f′⁢(x)=1+4⁢x×0-2×1=-1<0,

a contradiction. ∎

Note that we proved f can’t be increasing on any interval containing 0 by showing that its derivative takes negative values on every such interval. So we are again using Proposition 3.11.

3.3 Darboux’s Theorem

If a function f:D→ℝ is differentiable, it is continuous (Proposition 2.8), and its derivative defines a function f′:D→ℝ. Note that it does not follow that f′:D→ℝ is continuous! In fact, Example 3.14 provides a counterexample: we saw that this function is differentiable everywhere and has f′⁢(0)=1. We also observed that, for any positive integer m, f′⁢(xm)=-1, where xm=1/(2⁢π⁢m). Clearly xm→0, but f′⁢(xm)↛f′⁢(0)=1, so f′ is discontinuous at 0.

So, the derivative of a differentiable function f:[a,b]→ℝ, may fail to be a continuous function. Nonetheless, f′:[a,b]→ℝ is guaranteed to satisfy one of the key properties of continuous functions on [a,b]: it has the intermediate value property. That is, given any number k between f′⁢(a) and f′⁢(b), there exists c∈[a,b] such that f′⁢(c)=k. This holds even though, in general, f′ may fail to be a continuous function!

Theorem 3.15 (Darboux’s Theorem)

Let f be differentiable on [a,b] and k be any number between f′⁢(a) and f′⁢(b). Then there exists c∈[a,b] such that f′⁢(c)=k.

Proof.

If k=f′⁢(a) we may take c=a. If k=f′⁢(b) we may take c=b. It remains to consider the cases (i) f′⁢(a)<k<f′⁢(b) and (ii) f′⁢(b)<k<f′⁢(a). In case (i), let g⁢(x)=k⁢x-f⁢(x). This is differentiable, hence continuous, on [a,b] so, by the Extreme Value Theorem, it attains a maximum somewhere on [a,b]. Now g′⁢(a)=k-f′⁢(a)>0 so the maximum can’t be at a (Proposition 3.4). Similarly, g′⁢(b)=k-f′⁢(b)<0 so the maximum can’t be at b (Proposition 3.4 again). Hence, the maximum is attained at some interior point c∈(a,b). But then g′⁢(c)=0 by the Interior Extremum Theorem, and g′⁢(c)=k-f′⁢(c), which completes case (i). In case (ii), let f~⁢(x)=-f⁢(x). Then f~ is differentiable on [a,b] and f~′⁢(a)<-k<f~′⁢(b) so, by part (i), there exists c∈(a,b) such that f~′⁢(c)=-k. But then f′⁢(c)=k. ∎

So the function

g:ℝ→ℝ,g⁢(x)={1+4⁢x⁢sin⁡(1/x)-2⁢cos⁡(1/x)if x≠01if x=0

has the intermediate value property on every interval (by Darboux’s Theorem, since it’s the derivative of the differentiable function in Example 3.14) despite being discontinuous at 0.

We can give a geometric interpretation to the Extended Mean Value Theorem similar to the interpretation of the ordinary Mean Value Theorem, but to do so we need to consider not the graph y=f⁢(x) but rather the parametrized curve with components (g,f). That is, consider the map

γ:[a,b]→ℝ2,γ⁢(t)=(g⁢(t),f⁢(t)).

The tangent line to γ at the point γ⁢(t)=(g⁢(t),f⁢(t)) points parallel to the vector γ′⁢(t)=(g′⁢(t),f′⁢(t)). Hence, its slope is f′⁢(t)/g′⁢(t). So the Extended Mean Value Theorem says that there is a point on the curve γ, somewhere between its start point γ⁢(a) and its end point γ⁢(b), at which its tangent line is parallel to the straight line connecting γ⁢(a) to γ⁢(b).

If we apply Theorem 3.16 in the case where g⁢(x)=x, we immediately deduce the usual Mean Value Theorem22 2 Or, even better, apply it in the case g⁢(x)=x-a. Then the proof of Theorem 3.16 reduces to the proof of Theorem 3.7.. We will see that other choices of g can be useful. For example, we can use Theorem 3.16 to rigorously justify a popular trick for computing limits called “L’Hospital’s Rule”.

Theorem 3.17 (L’Hospital’s Rule)

Let I⊆ℝ be an open interval, f,g:I→ℝ be differentiable functions, a∈I, and f⁢(a)=g⁢(a)=0. Assume that, for all x∈I, g′⁢(x)≠0 and all x∈I\{a}, g⁢(x)≠0. Then

limx→a⁡f⁢(x)g⁢(x)=limx→a⁡f′⁢(x)g′⁢(x)

provided the limit on the right exists.

Proof.

Let (xn) be any sequence in I\{a} converging to a, and for each n, consider f,g restricted to the closed interval with endpoints a and xn. These functions satisfy the hypotheses of Theorem 3.16, so there exists cn, between a and xn, such that

f′⁢(cn)g′⁢(cn)=f⁢(xn)-f⁢(a)g⁢(xn)-g⁢(a)=f⁢(xn)g⁢(xn).

By the Squeeze Rule, cn→a, and so, assuming the limit

limx→a⁡f′⁢(x)g′⁢(x)

exists, call it L∈ℝ, say, f′⁢(cn)/g′⁢(cn)→L (by the definition of limit). Hence f⁢(xn)/g⁢(xn)→L, as was to be shown. ∎

3.5 Higher derivatives and Taylor’s Theorem

If a function f:D→ℝ is differentiable (everywhere on D) then its derivative defines another function f′:D→ℝ, and we can ask whether this function, in turn, is differentiable (recall that, in general, it might not even be continuous). If f′ is differentiable, then its derivative is denoted f′′:D→ℝ and called the second derivative of f. Similarly, if f′′:D→ℝ is differentiable, its derivative f′′′:D→ℝ is the third derivative of f. Proceeding inductively, we can define the nth derivative of f, denoted f(n) to be the derivative (if it exists), of f(n-1), where f(0) is, by definition, just the function f itself (so f(1)=f′, f(2)=f′′ etc.). If f(n):D→ℝ exists, we say that f is n times differentiable, and if f is n times differentiable for all n∈ℤ+, we say that f is smooth. It is convenient to develop some terminology and notation for describing the differentiability of functions:

Definition 3.19

A function f:D→ℝ is continuously differentiable if it is differentiable and f′:D→ℝ is continuous. The function is n times continuously differentiable if all its derivatives up to the nt⁢h exist everywhere on D, and are continuous. We denote the set of such functions by Cn⁢(D) and say that “f is Cn" if f∈Cn⁢(D). The function f is smooth if it is Cn for all n∈ℤ+. The set of smooth functions on D is denoted C∞⁢(D). For completeness, we denote the set of continuous functions D→ℝ by C0⁢(D).

An interesting interpretation of the Mean Value Theorem can be given as follows. Let I⊆ℝ be an open interval, f:I→ℝ be differentiable, and choose and fix some a∈I. Then for any other x∈I, we may apply the Mean Value Theorem to f:[a,x]→ℝ (if x>a) or f:[x,a]→ℝ (if x<a) to deduce that there exists c between a and x such that

f⁢(x)=f⁢(a)+f′⁢(c)⁢(x-a).

Note that the same equation holds trivially in the case where x=a (with c=x=a). So, for x close to a (meaning that |x-a| is small) we can approximate f by the constant f⁢(a), and the error in this approximation is of size |f′⁢(c)|⁢|x-a|, where c is somewhere between a and x (and, of course, depends on x).

The moral is that differentiability (once) of f on an open interval allows one to approximate f by a degree 0 polynomial function (i.e. a constant), with an error controlled by f′. It turns out that if f is n times differentiable one can do better: one can approximate f by a degree (n-1) polynomial function, with an error controlled by f(n). To prove this, we will again use the Extended Mean Value Theorem (Theorem 3.16).

Theorem 3.21 (Taylor’s Theorem)

Let I⊆ℝ be an interval, f:I→ℝ be (n+1) times differentiable, and a,x∈I. Then there exists c between a and x such that

f⁢(x) = f⁢(a)+f′⁢(a)⁢(x-a)+f′′⁢(a)2!⁢(x-a)2+⋯+f(n)⁢(a)n!⁢(x-a)n
+f(n+1)⁢(c)(n+1)!⁢(x-a)n+1.
Proof.

First note that the claimed equation holds trivially (with c=a) in the case where x=a. Choose and fix a,x∈I, x≠a, and consider the function F:I→ℝ,

F⁢(t)=f⁢(x)-f⁢(t)-f′⁢(t)⁢(x-t)-f′′⁢(t)2!⁢(x-t)2-⋯-f(n)⁢(t)n!⁢(x-t)n.

Then the claim to be proved is that there exists c between a and x such that

F⁢(a)=f(n+1)⁢(c)(n+1)!⁢(x-a)n+1.

Now, by the Product Rule, F is differentiable, and, as is easily verified,

F′⁢(t)=-f(n+1)⁢(t)n!⁢(x-t)n.

Let G⁢(t)=(x-t)n+1. Then F,G satisfy the hypotheses of Theorem 3.16 on the interval with endpoints a,x (note that G′⁢(t)=0 only if t=x), so there exists c between a and x such that

F′⁢(c)G′⁢(c) = F⁢(x)-F⁢(a)G⁢(x)-G⁢(a)=F⁢(a)G⁢(a)
i.e.   -f(n+1)⁢(c)⁢(x-c)nn!⁢[-(n+1)⁢(x-c)n] = F⁢(a)(x-a)n+1,

whence the claimed expression for F⁢(a) immediately follows. ∎

The polynomial

pn⁢(x)=f⁢(a)+f′⁢(a)⁢(x-a)+f′′⁢(a)2!⁢(x-a)2+⋯+f(n)⁢(a)n!⁢(x-a)n

is often called the nth Taylor approximant of the function f about a, and

f(n+1)⁢(c)(n+1)!⁢(x-a)n+1

is called the remainder.

Example 3.22

Let f:[0,∞)→ℝ be the function f⁢(x)=x. Construct the third Taylor approximant for f about 4, and hence find an approximation to f⁢(5)=5. Use Taylor’s Theorem to find upper and lower bounds on 5.

Solution The first four derivatives of f are

f′⁢(x) = x-1/22,
f′′⁢(x) = -x-3/24,
f′′′⁢(x) = 3⁢x-5/28,
f(4)⁢(x) = -15⁢x-7/216,

so f⁢(4)=2, f′⁢(4)=14, f′′⁢(4)=-2-5, and f′′′⁢(4)=3/28. It follows that the third Taylor approximant for f about 4 is

p3⁢(x) = f⁢(4)+f′⁢(4)⁢(x-4)+f′′⁢(4)2!⁢(x-4)2+f′′′⁢(4)3!⁢(x-4)3
= 2+14⁢(x-4)-164⁢(x-4)2+1512⁢(x-4)3.

Using the approximation f⁢(x)≈p3⁢(x) gives

5=f⁢(5)≈p3⁢(5)=2+14-164+1512=2.236328125.

By Taylor’s Theorem, the true value of f⁢(5) is

f⁢(5)=p3⁢(5)+f(4)⁢(c)4!⁢(5-4)4=p3⁢(5)-527⁢c7/2

for some c∈(4,5). It follows that the true value of 5 is strictly less than p3⁢(5), but strictly greater than p3⁢(5)-5/(27⁢47/2). That is, we have established that

2.236022949<5<2.236328125.

For comparison, my pocket calculator tells me that 5≈2.236067977. □

You already knew how to prove that the Alternating Harmonic Series converges: you can apply the Alternating Series Test. But this test, like all the others (Comparison Test, Ratio Test etc), only tells you that the series converges, it doesn’t tell you anything about its limit. Taylor’s Theorem allowed us to prove convergence and compute the limit. Note that the argument made crucial use of the remainder term: a Taylor expansion without remainder isn’t much good for proving anything.

Summary

  • •
    ​

    The Interior Extremum Theorem: If f:(a,b)→ℝ is differentiable, and has an extremum (maximum or minimum) at c∈(a,b), then f′⁢(c)=0.

  • •
    ​

    The Mean Value Theorem: If f:[a,b]→ℝ is continuous, and differentiable on (a,b), then there exists c∈(a,b) such that

    f′⁢(c)=f⁢(b)-f⁢(a)b-a.
  • •
    ​

    Let f:I→ℝ be differentiable, where I⊆ℝ is an interval. We can use the Mean Value Theorem to prove that

    • ∙
      ​

      If f′⁢(x)=0 for all x∈I then f is constant.

    • ∙
      ​

      If f′⁢(x)>0 for all x∈I then f is strictly increasing.

    • ∙
      ​

      If f′⁢(x)<0 for all x∈I then f is strictly decreasing.

    We often use the first of these when we solve ordinary differential equations.

  • •
    ​

    The derivative f′ of a differentiable function may fail to be continuous. However…

  • •
    ​

    Darboux’s Theorem: Let f:[a,b]→ℝ be differentiable and k be any number between f′⁢(a) and f′⁢(b). Then there exists c∈[a,b] such that f′⁢(c)=k.

  • •
    ​

    The Extended Mean Value Theorem: If f:[a,b]→ℝ and g:[a,b]→ℝ are continuous, and differentiable on (a,b), and g′⁢(x)≠0 for all x∈(a,b), then there exists c∈(a,b) such that

    f′⁢(c)g′⁢(c)=f⁢(b)-f⁢(a)g⁢(b)-g⁢(a).
  • •
    ​

    We can use the Extended Mean Value Theorem to prove Taylor’s Theorem.

  • •
    ​

    Taylor’s Theorem: If f is (n+1) times differentiable on some open interval I and a,x∈I, then there exists c between a and x such that

    f⁢(x)=f⁢(a)+f′⁢(a)1!⁢(x-a)+⋯+f(n)⁢(a)n!⁢(x-a)n+f(n+1)⁢(c)n!⁢(x-a)n+1.
  • •
    ​

    Taylor’s Theorem allows us to approximate (n+1) times differentiable functions by a polynomial of degree n, with an error which is controlled by f(n+1).
