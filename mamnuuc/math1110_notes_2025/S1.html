1 Numbers and inequalities

1.1 Sets of numbers

We start with various different types of numbers. We express these as sets, which you will be learning about in more detail in Core Mathematics - essentially a set is a collection of objects, and we can think of a set as a “box” containing mathematical objects. Here are five very important sets which occur regularly in mathematics:

ℤ+\mathbb{Z}^{+}, the set of positive integers:

This is the set of “counting” numbers, that is the set of all whole numbers larger than zero, which can be written as

ℤ+={1,2,3,4,…}.\mathbb{Z}^{+}=\{1,2,3,4,\ldots\}\ .

Here we have used set notation, the start of the set denoted by {\{ and the end of it by }\} (so everything in between is in the set).11 1 Some mathematicians call this set ℕ\mathbb{N}, but others disagree and say that ℕ\mathbb{N} should include 0. To avoid confusion we use ℤ+\mathbb{Z}^{+}.

ℤ\mathbb{Z}, the set of integers:

This is the set of all whole numbers, including 0 and negative numbers. That is

ℤ={…,−3,−2,−1,0,1,2,3,…}.\mathbb{Z}=\{\ldots,-3,-2,-1,0,1,2,3,\ldots\}\ .
ℚ\mathbb{Q}, the set of rational numbers

. This is the set of all numbers that may be written as a fraction of integers. That is

ℚ\displaystyle\mathbb{Q} =“any number which may be written as ab where a∈ℤ and b∈ℤ+”\displaystyle=\text{``any number which may be written as $\frac{a}{b}$ where $% a\in\mathbb{Z}$ and $b\in\mathbb{Z}^{+}$''}

In the above the ℚ\mathbb{Q} stands for quotients.

ℝ\mathbb{R}, the set of real numbers:

This is the set of all real numbers, including algebraic real numbers, such as 2∈ℝ\sqrt{2}\in\mathbb{R} and transcendental real numbers, for example e,π∈ℝe,\pi\in\mathbb{R}.

ℂ\mathbb{C}, the set of complex numbers:

All complex numbers, including for example −1=i∈ℂ\sqrt{-1}=i\in\mathbb{C}.

An object that is in the a set is called an element of that set. To say that an object is in a set we use ∈\in, so given a set AA “x∈Ax\in A” means “xx is an element of AA”. If we want to say an object isn’t in the set we use ∉\notin. So, for example

7∈ℤ+,54392375004∈ℤ+,but12∉ℤ+,−10∉ℤ+.7\in\mathbb{Z}^{+},\qquad 54392375004\in\mathbb{Z}^{+},\qquad\text{but}\qquad% \frac{1}{2}\notin\mathbb{Z}^{+},\qquad-10\notin\mathbb{Z}^{+}\ .

Note that in the above, every natural number is also an integer, every integer is a rational number, every rational number is a real number and so on. However, there are real numbers that are not rational numbers, and rational numbers that are not integers and so on.

A set AA is said to be a subset of another set BB (or we might say AA is contained in BB) if every element in set AA is also in set BB. We write this, A⊆BA\subseteq B. We have that:

ℤ+⊆ℤ⊆ℚ⊆ℝ⊆ℂ.\mathbb{Z}^{+}\subseteq\mathbb{Z}\subseteq\mathbb{Q}\subseteq\mathbb{R}% \subseteq\mathbb{C}\ .

In this course we will be interested in understanding the properties of ℝ\mathbb{R}. Our aim is to derive all of the properties of the real numbers, starting only from the fundamental sets described above.

In this course, we will be particularly interested in the real numbers, ℝ\mathbb{R}. As well as being a set, ℝ\mathbb{R} has further structures which have been familiar to you since primary school:

ℝ\mathbb{R} is a field:

The real numbers come with standard operations which are familiar to you, namely addition, subtraction, multiplication and division, that is (+−×÷)(+\ -\ \times\ \div), along with the familiar algebraic properties (for example, for any real numbers a,b,ca,b,c we know that a×(b+c)=(a×b)+(a×c)a\times(b+c)=(a\times b)+(a\times c)).

ℝ\mathbb{R} is an ordered field:

As well as the above we have an ordering relation <<. Specifically, given any two real numbers a,b∈ℝa,b\in\mathbb{R} exactly one of a=ba=b, a<ba<b and a>ba>b is true. The ordering << is assumed to have all of the usual properties that you are used to (for example, for any a,b,c∈ℝa,b,c\in\mathbb{R}, if a<ba<b and 0<c0<c then a⁢c<b⁢cac<bc).

Our aim is to use the properties the above properties to build up the theorems that we need. For now, we have a further property that we will assume:

FACT: (The Archimidean property of real numbers) Given any real number x∈ℝx\in\mathbb{R} there is a positive integer n∈ℤ+n\in\mathbb{Z}^{+} with x<nx<n.

We now restrict ourselves to these properties and statements that we may derive from these properties! For example, for now we will avoid using functions that we haven’t proven the properties of (or even rigorously defined) functions such as sin⁡(x)\sin(x), cos⁡(x)\cos(x), exp⁡(x)\exp(x), xpx^{p} for pp not an integer. We need to prove the properties that we know about these before using them!

1.2 Modulus and measurement on the real numbers

We now define an important function which will be extremely important to us in this course is the modulus function (or absolute value function). This is written as |x||x| and is defined by

|x|:={x if ⁢x≥0−x if ⁢x<0.|x|:=\begin{cases}x&\text{ if }x\geq 0\\ -x&\text{ if }x<0\end{cases}\ .

So |4|=4|4|=4, |−2|=−(−2)=2|-2|=-(-2)=2 and so on. Clearly |x|≥0|x|\geq 0 for all xx, by definition.

If we think of real numbers as lying on an infinite straight line, then |x||x| can be interpreted geometrically as the distance from xx to 0. Furthermore, given any pair of real numbers x,yx,y, the absolute value of their difference |x−y||x-y| can be thought of as the distance between them. Note that |x−y|=|y−x||x-y|=|y-x| for all xx and yy. For example the distance between 44 and −2-2 is

|4−(−2)|=|6|=6or|(−2)−4|=|−6|=−(−6)=6.|4-(-2)|=|6|=6\quad\mbox{or}\quad|(-2)-4|=|-6|=-(-6)=6.

We emphasize that this is just a mental model of ℝ{\mathbb{R}} which gives a nice interpretation of |⋅||\cdot|. We are not defining real numbers as points on an infinite line, nor are we defining |x||x| to be the distance from xx to 0.

We will often care about inequalities involving |x||x|.

Lemma 1.1

For any real numbers xx and cc, |x|<c|x|<c if and only if −c<x<c-c<x<c.

In the above “if and only if” means that |x|<c|x|<c is true exactly when −c<x<c-c<x<c (and |x|<c|x|<c is false exactly when −c<x<c-c<x<c is false). As mathematicians are busy people, we often shorten “if and only if” to “iff”. The above seems reasonable – we might draw the following helpful picture:

However, drawing a graph helps with intuition, but does not constitute a proof!

Proof.

Suppose first that |x|<c|x|<c holds. Note that in this case, 0<c0<c (as 0≤|x|<c0\leq|x|<c). We have two cases to consider:

  • •
    ​

    If x≥0x\geq 0 then this implies that 0≤x=|x|<c0\leq x=|x|<c so we have that −c<x<c-c<x<c.

  • •
    ​

    If x<0x<0, we have −x=|x|<c-x=|x|<c. Rearranging we have that −c<x-c<x. Therefore in this case −c<x<0-c<x<0 which implies −c<x<c-c<x<c.

As these are the only two possibilities for xx, if |x|<c|x|<c we always have −c<x<c-c<x<c

Now suppose that −c<x<c-c<x<c: Again, this implies that 0≤c0\leq c (this follows from rearranging −c<c-c<c). If x≥0x\geq 0 then |x|=x<c|x|=x<c. If x<0x<0 then |x|=−x<c|x|=-x<c (where this inequality comes from rearranging −c<x-c<x). Therefore we see that if −c<x<c-c<x<c then |x|<c|x|<c. ∎

Note that in the above we had to have two steps to prove |x|<c|x|<c iff −c<x<c-c<x<c. We had to prove both “if |x|<c|x|<c is true then −c<x<c-c<x<c is true” and “if −c<x<c-c<x<c is true then |x|<c|x|<c”. In general, we need to do both! There are lots of true statements of the form “If A then B” is true but “If B then A” is false. For example, “If I am wearing a hat then there is someone in the room that I am in who is wearing a hat” is always true. However, “If there is someone in the room that I am in who is wearing a hat then I am wearing a hat” is false in general.

Remark 1.2

There is a second statement which is the weak inequality version (or “≤\leq version”) of Lemma 1.1:

For any real numbers xx and cc, |x|≤c|x|\leq c if and only if −c≤x≤c-c\leq x\leq c.

This is also true - the proof is identical to the proof of Lemma 1.1 (with << replaced with ≤\leq). I leave it as an exercise to check this!

Example 1.3

We may see that |x−2|<1|x-2|<1 is all numbers xx that have −1<x−2<1-1<x-2<1 or 1<x<31<x<3. We may think of these as all numbers of distance less than 1 from 2.

For any x∈ℝx\in\mathbb{R} we have that

|−x|=|x|,x≤|x|−x≤|x|,|-x|=|x|,\qquad x\leq|x|\qquad-x\leq|x|\ ,

and for any x,y∈ℝx,y\in\mathbb{R} we have that

|x⁢y|=|x|⁢|y|.|xy|=|x||y|\ .

Finally, we come to an inequality which will be very important throughout the course.

Lemma 1.5 (The triangle inequality)

Suppose that a,b∈ℝa,b\in\mathbb{R}. Then

|a+b|≤|a|+|b|.|a+b|\leq|a|+|b|\ .
Proof.

From Lemma 1.4, we have that a≤|a|a\leq|a| and b≤|b|b\leq|b|. Therefore a+b≤|a|+b≤|a|+|b|a+b\leq|a|+b\leq|a|+|b|. On the other hand, −a≤|a|-a\leq|a|, −b≤|b|-b\leq|b| so −a−b≤|a|+|b|-a-b\leq|a|+|b|, or (rearranging) −|a|−|b|≤a+b-|a|-|b|\leq a+b. Putting these together we have that

−|a|−|b|≤a+b≤|a|+|b|,-|a|-|b|\leq a+b\leq|a|+|b|\ ,

so by Lemma 1.1, |a+b|≤|a|+|b||a+b|\leq|a|+|b| as claimed. ∎

Corollary 1.6 (The reverse triangle inequality)

Suppose that x,y∈ℝx,y\in\mathbb{R}. Then

||x|−|y||≤|x−y|||x|-|y||\leq|x-y|
Proof.

We have that from the triangle inequality (setting a=x−ya=x-y, b=yb=y)

|x|≤|x−y|+|y|⟹|x|−|y|≤|x−y|,|x|\leq|x-y|+|y|\qquad\implies\qquad|x|-|y|\leq|x-y|\ ,

and (setting a=y−xa=y-x, b=xb=x)

|y|≤|y−x|+|x|⟹−|x−y|≤|x|−|y|.|y|\leq|y-x|+|x|\qquad\implies\qquad-|x-y|\leq|x|-|y|\ .

Putting these together we have that

−|c−d|≤|c|−|d|≤|c−d|-|c-d|\leq|c|-|d|\leq|c-d|

which is equivalent to the statement (by Lemma 1.1 and Remark 1.2) ∎
