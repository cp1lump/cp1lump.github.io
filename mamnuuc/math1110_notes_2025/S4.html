4 Subsequences

4.1 Definition and convergence properties

Definition 4.1

A sequence (bk)(b_{k}) is a subsequence of a sequence (an)(a_{n}) if there exists a strictly increasing sequence of positive integers (nk)(n_{k}) such that bk=ankb_{k}=a_{n_{k}} for all k∈ℤ+k\in{\mathbb{Z}}^{+}.

In other words, all the terms of (bk)(b_{k}) must occur in (an)(a_{n}), and they must occur in the same order, but we are allowed to omit some (possibly infinitely many) terms from (an)(a_{n}).

Example 4.2

It’s the subsequence consisting of only the even-numbered terms of (an)(a_{n}). From this example we see that a divergent (that is non-convergent) sequence can have a convergent subsequence.

As we have seen, it is possible for a divergent sequence to have a convergent subsequence. On the other hand, if a sequence is convergent, with limit LL, one expects that all its subsequences should also converge to LL. This turns out to be true.

Theorem 4.3

If an→La_{n}\rightarrow L and (bk)(b_{k}) is a subsequence of (an)(a_{n}), then bk→Lb_{k}\rightarrow L.

Proof.

Let ε>0\varepsilon>0 be given. Since an→La_{n}\rightarrow L, there exists N∈ℤ+N\in{\mathbb{Z}}^{+} such that |an−L|<ε|a_{n}-L|<\varepsilon for all n≥Nn\geq N. Since (bk)(b_{k}) is a subsequence of (an)(a_{n}), bk=ankb_{k}=a_{n_{k}} where nkn_{k} is a strictly increasing sequence of positive integers. Now n1≥1n_{1}\geq 1, and if nk≥kn_{k}\geq k then nk+1≥nk+1≥k+1n_{k+1}\geq n_{k}+1\geq k+1. Hence, by induction, nk≥kn_{k}\geq k for all kk. Hence, for all k≥Nk\geq N, nk≥nN≥Nn_{k}\geq n_{N}\geq N, so |bk−L|=|ank−L|<ε|b_{k}-L|=|a_{n_{k}}-L|<\varepsilon. Hence bk→Lb_{k}\rightarrow L. ∎

We can use this to give a sneaky proof that an=(−1)na_{n}=(-1)^{n} doesn’t converge, without having to resort to an ε\varepsilon–NN argument:

Example 4.4 (Equation (2.8) revisited)

Claim: an=(−1)na_{n}=(-1)^{n} doesn’t converge to any limit.

Proof.

Assume, to the contrary, that an→La_{n}\rightarrow L. Then, by Theorem 4.3, the subsequences a2⁢ka_{2k} and a2⁢k+1a_{2k+1} also converge to LL. But a2⁢k=1→1a_{2k}=1\rightarrow 1 and a2⁢k+1=−1→−1a_{2k+1}=-1\rightarrow-1, so 1=L=−11=L=-1, a contradiction. ∎

Note that we’ve actually used Theorem 4.3 a couple of times already. In Examples 3.21 and 2.3 (revisited) we asserted that if an→La_{n}\rightarrow L then bn=an+1→Lb_{n}=a_{n+1}\rightarrow L also. This follows from Theorem 4.3 since (bn)(b_{n}) is a subsequence of (an)(a_{n}). It’s important to realize that Theorem 4.3 does not say that if a subsequence of (an)(a_{n}) converges to LL then (an)(a_{n}) converges to LL (we’ve already seen a counterexample to this: an=(−1)na_{n}=(-1)^{n}). Just showing that a subsequence of (an)(a_{n}) converges is not enough (usually) to show that (an)(a_{n}) converges. There is an important case, however, where convergence of the subsequence does imply convergence of the whole sequence, namely, subsequences obtained by chopping off the first mm terms of the sequence.

Lemma 4.5 (Tail Lemma)

Let (an)(a_{n}) be a real sequence, mm be a positive integer and bn=an+mb_{n}=a_{n+m}. If bn→Lb_{n}\rightarrow L then an→La_{n}\rightarrow L.

Proof.

Assume bn→Lb_{n}\rightarrow L. Let ε>0\varepsilon>0 be given. Then there exists N1∈ℤ+N_{1}\in{\mathbb{Z}}^{+} such that, for all n≥N1n\geq N_{1}, |bn−L|<ε|b_{n}-L|<\varepsilon. Let N=N1+mN=N_{1}+m. Then, for all n≥Nn\geq N, n−m≥N1n-m\geq N_{1}, so

|an−L|=|bn−m−L|<ε.|a_{n}-L|=|b_{n-m}-L|<\varepsilon.

Hence an→La_{n}\rightarrow L. ∎

This Lemma is very useful in practice, because it allows us to ignore any number of terms at the start of a sequence when attempting to prove convergence.

4.2 The Bolzano-Weierstrass Theorem

We have seen that all subsequences of a convergent sequence are convergent, and that non-convergent sequences can sometimes have convergent subsequences. The goal of this section is to prove a famous and very powerful result called the Bolzano-Weierstrass33 3 Weierstraß for German purists. Theorem, which says that every bounded sequence has at least one convergent subsequence. It won’t be immediately clear why this fact is such a big deal. Later we will use it to prove the Extreme Value Theorem, which is a crucial result when one comes to define integration carefully, for example.

The strategy of proof is quite straightforward: we first prove that every sequence has a monotone subsequence. Having done this, the Bolzano-Weierstrass Theorem follows immediately from the Monotone Convergence Theorem.

Lemma 4.7

Every sequence has a monotone subsequence.

Proof.

Let the sequence in question be (an)(a_{n}). We make the following definition: a term ama_{m} is dominant if an≤ama_{n}\leq a_{m} for all n>mn>m (that is, ama_{m} is at least as big as all subsequent terms). Consider the set of all dominant terms of the sequence (an)(a_{n}). There are exactly two possibilities: either (i) the set of dominant terms is infinite, or (ii) the set of dominant terms is finite (which includes the case where there are no dominant terms).

Case (i): if the set of dominant terms is infinite, then the subsequence consisting of just these terms is, by definition, decreasing. Hence (an)(a_{n}) has a decreasing subsequence.

Case (ii): if the set of dominant terms is finite (or empty), there is a term in the sequence, ama_{m} say, beyond which there are no dominant terms. Let n1=m+1n_{1}=m+1. Since an1a_{n_{1}} is not dominant, there exists n>n1n>n_{1} such that an>an1a_{n}>a_{n_{1}}. Choose such an nn and call it n2n_{2}. Then an2a_{n_{2}} is also not dominant (since n2>n1>mn_{2}>n_{1}>m), so there exists a positive integer greater than n2n_{2}, let’s call it n3n_{3}, such that an3>an2a_{n_{3}}>a_{n_{2}}. In this way we construct a strictly increasing sequence of positive integers n1,n2,n3,…n_{1},n_{2},n_{3},\ldots such that ank+1>anka_{n_{k+1}}>a_{n_{k}} for all kk. But then (ank)(a_{n_{k}}) is an increasing subsequence of (an)(a_{n}). Hence (an)(a_{n}) has an increasing subsequence.

In either case, we see that (an)(a_{n}) has a monotone subsequence, as was to be proved. ∎

Theorem 4.8 (The Bolzano-Weierstrass Theorem)

Let (an)(a_{n}) be a bounded real sequence. Then (an)(a_{n}) has a convergent subsequence.

Proof.

By Lemma 4.7, (an)(a_{n}) has a monotone subsequence (ank)(a_{n_{k}}). But the range of (ank)(a_{n_{k}}) is a subset of the range of (an)(a_{n}), so (ank)(a_{n_{k}}) is also bounded (by the same upper and lower bounds as (an)(a_{n})), and hence converges by the Monotone Convergence Theorem (Theorem 3.20). ∎

Of course, the converse of this Theorem is false: just because a sequence has a convergent subsequence, it doesn’t follow that it’s bounded. For example

has a convergent subseqence, a2⁢k=0→0a_{2k}=0\rightarrow 0, but is clearly unbounded above.

4.3 The Cauchy property

One slightly inconvenient thing about Definition 2.4 is that, to prove a sequence converges from first principles, we must first know what its limit is. The Monotone Convergence Theorem allows us to prove convergence of some sequences without knowing their limits, but this only works in the special case where the sequence happens to be monotone (increasing or decreasing). In this section we will study a criterion which turns out to be equivalent to convergence, for all sequences, but which makes no mention of limits.

Definition 4.10

A real sequence (an)(a_{n}) is Cauchy (or has the Cauchy property) if, for each ε>0\varepsilon>0, there exists N∈ℤ+N\in{\mathbb{Z}}^{+} such that, for all n,m≥Nn,m\geq N, |an−am|<ε|a_{n}-a_{m}|<\varepsilon.

Remark This looks very similar to the definition of convergence. It says that, given any ε>0\varepsilon>0 (no matter how small), there is a point in the sequence beyond which all terms lie closer than distance ε\varepsilon from one another. Note that it makes no mention of any limit.

We will show that real sequences converge if and only if they have the Cauchy property.

If (an)(a_{n}) converges then (an)(a_{n}) is Cauchy.

Theorem 4.14

A real sequence converges if and only if it is Cauchy.

Proof.

If (an)(a_{n}) converges, it is Cauchy, by Lemma 4.11. Conversely, if (an)(a_{n}) is Cauchy, it is bounded (Lemma 4.12), so has a convergent subsequence (by the Bolzano-Weierstrass Theorem) and so converges (Lemma 4.13). ∎
