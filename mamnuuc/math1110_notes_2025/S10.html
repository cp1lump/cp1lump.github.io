10 The Mean Value Theorem

10.1 Maxima and minima

Recall (Definition 8.11) that a function f:D→ℝf:D\rightarrow{\mathbb{R}} attains a maximum at c∈Dc\in D if, for all x∈Dx\in D, f⁢(x)≤f⁢(c)f(x)\leq f(c), and attains a minimum at d∈Dd\in D if, for all x∈Dx\in D, f⁢(x)≥f⁢(d)f(x)\geq f(d). These definitions don’t make any reference to the derivative of ff. You may find this surprising: after all, in your previous studies of calculus, you’ve been trained, when asked to find the maximum or minimum values of a function, to find and examine its critical points (also called “turning” or “stationary” points), that is points xx were f′⁢(x)=0f^{\prime}(x)=0. The next theorem66 6 Extremum means “maximum or minimum”. Its plural is “extrema.” explains why.

Theorem 10.1 (Interior Extremum Theorem)

Let f:(a,b)→ℝf:(a,b)\rightarrow{\mathbb{R}} be differentiable, and attain a maximum or a minimum at c∈(a,b)c\in(a,b). Then f′⁢(c)=0f^{\prime}(c)=0.

Proof.

Assume first that ff attains a maximum at c∈(a,b)c\in(a,b). Consider the sequences

xn=c−(c−a)n+1andyn=c+b−cn+1.x_{n}=c-\frac{(c-a)}{n+1}\qquad\mbox{and}\qquad y_{n}=c+\frac{b-c}{n+1}.

These are sequences in (a,b)\{c}(a,b){\backslash}\{c\} which converge to cc, so

sn=f⁢(xn)−f⁢(c)xn−c→f′⁢(c)andtn=f⁢(yn)−f⁢(c)yn−c→f′⁢(c)s_{n}=\frac{f(x_{n})-f(c)}{x_{n}-c}\rightarrow f^{\prime}(c)\qquad\mbox{and}% \qquad t_{n}=\frac{f(y_{n})-f(c)}{y_{n}-c}\rightarrow f^{\prime}(c)

by the definition of f′⁢(c)f^{\prime}(c). But ff attains a maximum at cc, so f⁢(xn)≤f⁢(c)f(x_{n})\leq f(c) and f⁢(yn)≤f⁢(c)f(y_{n})\leq f(c), and xn<cx_{n}<c while yn>cy_{n}>c. Hence sn≥0s_{n}\geq 0 (it’s a non-positive number divided by a negative number) while tn≤0t_{n}\leq 0 (it’s a non-positive number divided by a positive number). But then

limn→∞sn≥0andlimn→∞tn≤0\lim_{n\rightarrow\infty}s_{n}\geq 0\qquad\mbox{and}\qquad\lim_{n\rightarrow% \infty}t_{n}\leq 0

by Theorem 2.14. Hence f′⁢(c)≥0f^{\prime}(c)\geq 0 and f′⁢(c)≤0f^{\prime}(c)\leq 0, so f′⁢(c)=0f^{\prime}(c)=0.

If ff attains a minimum at cc, we apply the argument above to g⁢(x)=−f⁢(x)g(x)=-f(x), which attains a maximum at cc. Then g′⁢(c)=0g^{\prime}(c)=0, so f′⁢(c)=−g′⁢(c)=−0f^{\prime}(c)=-g^{\prime}(c)=-0. ∎

Example 10.2

Find the maximum and minimum values attained by

f:[0,2]→ℝ,f⁢(x)=x3−x2.f:[0,2]\rightarrow{\mathbb{R}},\qquad f(x)=x^{3}-x^{2}.

Note that ff does attain both a maximum and a minimum, by the Extreme Value Theorem (the domain is a closed bounded interval, and the function is continuous).

Solution:  Either the extrema occur at the endpoints {0,2}\{0,2\} or one or both occur in (0,2)(0,2). If an extremum occurs at c∈(0,2)c\in(0,2) then f⁢(c)f(c) also an extremum of the function restricted to (0,2)(0,2), so f′⁢(c)=0f^{\prime}(c)=0 by the Interior Extremum Theorem. Hence, the extrema occur either at an endpoint 0, 22, or at some interior critical point, where f′⁢(c)=0f^{\prime}(c)=0. Now

f′⁢(c)=3⁢c2−2⁢c=c⁢(3⁢c−2)=0f^{\prime}(c)=3c^{2}-2c=c(3c-2)=0

if c=0c=0 or c=2/3c=2/3 (in this case, one of the critical points is actually also an endpoint). Now

f⁢(0)=0,f⁢(2/3)=−427,f⁢(2)=4.f(0)=0,\qquad f(2/3)=-\frac{4}{27},\qquad f(2)=4.

Hence, the maximum value of ff is 44, attained at the endpoint 22, and the minimum value is −427-\frac{4}{27} attained at the interior critical point 23\frac{2}{3}. Note that in this case, the critical point 0 (also an endpoint) is neither a maximum nor a minimum of ff. □\Box

Remark 10.3

Having shown that, for all x∈[0,2]x\in[0,2], −4/27=f⁢(2/3)≤f⁢(x)≤f⁢(2)=4-4/27=f(2/3)\leq f(x)\leq f(2)=4, it follows from the Intermediate Value Theorem that the range of this function is

f⁢([0,2])=[−4/27,4].f([0,2])=[-4/27,4].

Do you see why?

We can combine the Interior Extremum Theorem with the Extreme Value Theorem to give a surprisingly powerful result traditionally called Rolle’s Theorem:

Theorem 10.4 (Rolle’s Theorem)

Let ff be a real function which is continuous on [a,b][a,b] and differentiable on (a,b)(a,b), and assume that f⁢(a)=f⁢(b)f(a)=f(b). Then there exists c∈(a,b)c\in(a,b) such that f′⁢(c)=0f^{\prime}(c)=0.

Proof.

By the Extreme Value Theorem, ff attains both a maximum and a minimum on [a,b][a,b]. If both the maximum and the minimum occur at endpoints of [a,b][a,b], then since f⁢(a)=f⁢(b)f(a)=f(b), the maximum value equals the minimum value, so ff is constant on [a,b][a,b] and hence f′⁢(c)=0f^{\prime}(c)=0 for all c∈(a,b)c\in(a,b), and we’re done. Otherwise, either the maximum or the minimum occurs at an interior point c∈(a,b)c\in(a,b). But then f′⁢(c)=0f^{\prime}(c)=0 by the Interior Extremum Theorem. ∎

10.2 Increasing and decreasing functions

It is common to interpret the derivative of a function ff as the rate of change of f⁢(x)f(x) as its input variable xx is increased. So, if f′⁢(a)>0f^{\prime}(a)>0, we expect that ff is an increasing function, at least for xx sufficiently close to aa. We’ve already seen that this naive expectation can actually be wrong in general (see Example 9.9). To convert information about the sign of f′f^{\prime} into information about the increasing or decreasing behaviour of ff, we need to be a bit more careful. We start (of course!) by carefully defining our terms.

Definition 10.5

A function f:D→ℝf:D\rightarrow{\mathbb{R}} (where D⊆ℝD\subseteq{\mathbb{R}}) is increasing if, for all x,y∈Dx,y\in D, if y>xy>x then f⁢(y)≥f⁢(x)f(y)\geq f(x). It is strictly increasing if, for all x,y∈Dx,y\in D, if y>xy>x then f⁢(y)>f⁢(x)f(y)>f(x). It is decreasing if, for all x,y∈Dx,y\in D, if y>xy>x then f⁢(y)≤f⁢(x)f(y)\leq f(x), and strictly decreasing if, for all x,y∈Dx,y\in D, if y>xy>x then f⁢(y)<f⁢(x)f(y)<f(x).

Remark Note once again that these definitions have absolutely nothing to do with calculus! They use only the ordering relation on ℝ{\mathbb{R}}, and make sense even if the function is not differentiable (or, indeed, continuous). Note also that a function can be both increasing and decreasing – but only if it is constant! (Can you see why?)

Example 10.6

The step function

f:ℝ→ℝ,f⁢(x)={0,x<0.1,x≥0.f:{\mathbb{R}}\rightarrow{\mathbb{R}},\qquad f(x)=\left\{\begin{array}[]{cc}0,% &x<0.\\ 1,&x\geq 0.\end{array}\right.

is an increasing function.

The key result allowing us to deduce information about the decreasing/increasing behaviour of differentiable functions is called the Mean Value Theorem.

Theorem 10.7 (Extended Mean Value Theorem)

Let ff and gg be real functions which are continuous on [a,b][a,b] and differentiable on (a,b)(a,b), and assume that, for all x∈(a,b)x\in(a,b), g′⁢(x)≠0g^{\prime}(x)\neq 0. Then there exists c∈(a,b)c\in(a,b) such that

f′⁢(c)g′⁢(c)=f⁢(b)−f⁢(a)g⁢(b)−g⁢(a).\frac{f^{\prime}(c)}{g^{\prime}(c)}=\frac{f(b)-f(a)}{g(b)-g(a)}.
Proof.

First note that the conditions on gg imply that g⁢(a)≠g⁢(b)g(a)\neq g(b) (since if g⁢(b)=g⁢(a)g(b)=g(a) then g′⁢(x)=0g^{\prime}(x)=0 for some x∈(a,b)x\in(a,b) by Rolle’s Theorem). Hence,

α=f⁢(b)−f⁢(a)g⁢(b)−g⁢(a)\alpha=\frac{f(b)-f(a)}{g(b)-g(a)}

is a well-defined real constant. Let h:[a,b]→ℝh:[a,b]\rightarrow{\mathbb{R}} such that h⁢(x)=f⁢(x)−α⁢g⁢(x)h(x)=f(x)-\alpha g(x). Then hh is continuous on [a,b][a,b] and differentiable on (a,b)(a,b), and, as one may easily verify,

h⁢(a)=f⁢(a)⁢g⁢(b)−g⁢(a)⁢f⁢(b)g⁢(b)−g⁢(a)=h⁢(b).h(a)=\frac{f(a)g(b)-g(a)f(b)}{g(b)-g(a)}=h(b).

Hence hh satisfies the hypotheses of Rolle’s Theorem, and we conclude that there exists c∈(a,b)c\in(a,b) such that h′⁢(c)=0h^{\prime}(c)=0. But then f′⁢(c)=α⁢g′⁢(c)f^{\prime}(c)=\alpha g^{\prime}(c), whence the result immediately follows. ∎

The name “Mean Value Theorem” is usually reserved for a special case of Theorem 10.7:

Corollary 10.8 (Mean Value Theorem)

Let ff be continuous on [a,b][a,b] and differentiable on (a,b)(a,b). Then there exists c∈(a,b)c\in(a,b) such that

f′⁢(c)=f⁢(b)−f⁢(a)b−a.f^{\prime}(c)=\frac{f(b)-f(a)}{b-a}.
Proof.

Just apply Theorem 10.7 in the case where g:[a,b]→ℝg:[a,b]\rightarrow{\mathbb{R}}, g⁢(x)=xg(x)=x. ∎

Remarks

  • •
    ​

    The Mean Value Theorem has a nice geometric interpretation. Consider the graph y=f⁢(x)y=f(x) of a differentiable function f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}}. For each pair of distinct numbers a<ba<b, we can construct the chord (straight line) passing through the points (a,f⁢(a))(a,f(a)) and (b,f⁢(b))(b,f(b)) on the graph. Its slope is (f⁢(b)−f⁢(a))/(b−a)(f(b)-f(a))/(b-a). The Mean Value Theorem asserts that, at some point (c,f⁢(c))(c,f(c)) on the graph between these two points, the tangent line to the graph is parallel to the chord.

  • •
    ​

    The Mean Value Theorem guarantees the existence of a point cc with the required f′⁢(c)f^{\prime}(c). It says nothing about uniqueness: there could be more than one such point.

The Mean Value Theorem allows us (among many other things) to relate the sign of f′f^{\prime} to the increasing/decreasing behaviour of ff.

Proposition 10.9

Let II be an interval and f:I→ℝf:I\rightarrow{\mathbb{R}} be differentiable.

  • (a)
    ​

    If f′⁢(x)≥0f^{\prime}(x)\geq 0 for all x∈Ix\in I then ff is increasing.

  • (b)
    ​

    If f′⁢(x)>0f^{\prime}(x)>0 for all x∈Ix\in I then ff is strictly increasing.

  • (c)
    ​

    If f′⁢(x)≤0f^{\prime}(x)\leq 0 for all x∈Ix\in I then ff is decreasing.

  • (d)
    ​

    If f′⁢(x)<0f^{\prime}(x)<0 for all x∈Ix\in I then ff is strictly decreasing.

  • (e)
    ​

    ff is constant if and only if f′⁢(x)=0f^{\prime}(x)=0 for all x∈Ix\in I.

Proof.

The proofs of parts (a)–(d) are similar, so we will do part (a), and leave (b), (c), (d) as an exercise.

(a) Assume, towards a contradiction, that f′⁢(x)≥0f^{\prime}(x)\geq 0 but ff is not increasing. Then there exist a,b∈Ia,b\in I with a<ba<b such that f⁢(a)>f⁢(b)f(a)>f(b). But then, by the Mean Value Theorem applied to ff on [a,b][a,b], there exists c∈(a,b)c\in(a,b) such that

f′⁢(c)=f⁢(b)−f⁢(a)b−a<0,f^{\prime}(c)=\frac{f(b)-f(a)}{b-a}<0,

a contradiction.

(e) If f⁢(x)=cf(x)=c, some constant, then f′⁢(x)=0f^{\prime}(x)=0 for all x∈Ix\in I by Example 9.2. Conversely, assume f′⁢(x)=0f^{\prime}(x)=0 for all x∈Ix\in I. Then ff is increasing by part (a) and decreasing by part (b), so must be constant. ∎

Remarks

  1. (i)
    ​

    The converses of parts (a) and (c) are also true. That is, if f:I→ℝf:I\rightarrow{\mathbb{R}} is increasing and differentiable, then f′⁢(x)≥0f^{\prime}(x)\geq 0 for all x∈Ix\in I. Similarly if f:I→ℝf:I\rightarrow{\mathbb{R}} is decreasing and differentiable, then f′⁢(x)≤0f^{\prime}(x)\leq 0 for all x∈Ix\in I. One can prove this by adapting the idea of the proof of the Interior Extremum Theorem.

  2. (ii)
    ​

    The converses of parts (b) and (d), by contrast, are false! For example

    f:ℝ→ℝ,f⁢(x)=x3f:{\mathbb{R}}\rightarrow{\mathbb{R}},\qquad f(x)=x^{3}

    is strictly increasing, but its derivative is not strictly positive.

  3. (iii)
    ​

    It’s important to note that Proposition 10.9 concerns functions on intervals. For example

    f:ℝ\{0}→ℝ,f⁢(x)=1xf:{\mathbb{R}}{\backslash}\{0\}\rightarrow{\mathbb{R}},\qquad f(x)=\frac{1}{x}

    has f′⁢(x)=−1/x2f^{\prime}(x)=-1/x^{2} which is negative everywhere, but f⁢(−1)<f⁢(1)f(-1)<f(1), so ff is not a decreasing function. This doesn’t contradict Proposition 10.9 since ℝ\{0}{\mathbb{R}}{\backslash}\{0\} is not an interval.

  4. (iv)
    ​

    Strictly increasing (or decreasing) functions are always injective, so Proposition 10.9 gives us a sneaky trick to prove that (some) functions are injective.

Example 10.10

f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}}, f⁢(x)=2⁢x+sin⁡xf(x)=2x+\sin x has f′⁢(x)=2+cos⁡x≥2−1>0f^{\prime}(x)=2+\cos x\geq 2-1>0 for all xx, so is strictly increasing by Proposition 10.9. Hence, if f⁢(x)=f⁢(y)f(x)=f(y) then x≮yx\nless y and y≮xy\nless x, so x=yx=y. That is, ff is injective.

10.3 The Inverse Function Theorem

Recall that in section 8.3 we used the Intermediate Value Theorem to prove that

f:[0,∞)→[0,∞),f⁢(x)=x2f:[0,\infty)\rightarrow[0,\infty),\qquad f(x)=x^{2}

has an inverse function

g:[0,∞)→[0,∞),g⁢(x)=x.g:[0,\infty)\rightarrow[0,\infty),\qquad g(x)=\sqrt{x}.

What can we prove about gg? Is it continuous? Differentiable? Our next theorem77 7 This is actually a very special case of a much more general theorem of the same name, which works for functions f:ℝn→ℝnf:{\mathbb{R}}^{n}\rightarrow{\mathbb{R}}^{n} even (when suitably interpreted) in the case n=∞n=\infty! helps us answer these questions.

Theorem 10.11 (The Inverse Function Theorem)

Let ff be continuous on [a,b][a,b] and differentiable on (a,b)(a,b), and assume that, for all x∈(a,b)x\in(a,b), f′⁢(x)>0f^{\prime}(x)>0. Then ff is a bijective mapping

f:[a,b]→[f⁢(a),f⁢(b)],f:[a,b]\rightarrow[f(a),f(b)],

and its inverse function

g:[f⁢(a),f⁢(b)]→[a,b]g:[f(a),f(b)]\rightarrow[a,b]

is continuous on [f⁢(a),f⁢(b)][f(a),f(b)], and differentiable on (f⁢(a),f⁢(b))(f(a),f(b)), with

g′⁢(y)=1f′⁢(g⁢(y)).g^{\prime}(y)=\frac{1}{f^{\prime}(g(y))}.
Proof.

I first claim that ff is strictly increasing. Assume this is false. Then there exist c,d∈[a,b]c,d\in[a,b] such that c<dc<d but f⁢(c)≥f⁢(d)f(c)\geq f(d). But then, by the Mean Value Theorem applied to ff on [c,d][c,d], there exists x∈(c,d)x\in(c,d) such that

f′⁢(x)=f⁢(d)−f⁢(c)d−c<0,f^{\prime}(x)=\frac{f(d)-f(c)}{d-c}<0,

contradicting the fact that f′f^{\prime} is positive on (a,b)(a,b).

Since ff is strictly increasing, it is injective and bounded below by f⁢(a)f(a) and above by f⁢(b)f(b). Further, given any y∈[f⁢(a),f⁢(b)]y\in[f(a),f(b)], there exists x∈[a,b]x\in[a,b] such that f⁢(x)=yf(x)=y, by the Intermediate Value Theorem, so ff is surjective. Hence ff is bijective, and it has an inverse function

g:[f⁢(a),f⁢(b)]→[a,b].g:[f(a),f(b)]\rightarrow[a,b].

Recall that this function, by definition, satisfies

g⁢(f⁢(x))=x,f⁢(g⁢(y))=yg(f(x))=x,\qquad f(g(y))=y

for all x∈[a,b]x\in[a,b] and y∈[f⁢(a),f⁢(b)]y\in[f(a),f(b)].

I claim that gg is continuous. Let y∈[f⁢(a),f⁢(b)]y\in[f(a),f(b)] and yny_{n} be any sequence in [f⁢(a),f⁢(b)][f(a),f(b)] that converges to yy. We must show that g⁢(yn)→g⁢(y)g(y_{n})\rightarrow g(y). Since ff is bijective, there is a unique c∈[a,b]c\in[a,b] such that y=f⁢(c)y=f(c) and c=g⁢(y)c=g(y). So we must show that g⁢(yn)→cg(y_{n})\rightarrow c (see figure 8). Assume that this is false, g⁢(yn)↛cg(y_{n})\nrightarrow c, or equivalently, g⁢(yn)−c↛0g(y_{n})-c\nrightarrow 0. Then, as we showed in Example 6.13, there exists ε>0\varepsilon>0 and a subsequence g⁢(ynk)g(y_{n_{k}}) such that |g⁢(ynk)−c|≥ε|g(y_{n_{k}})-c|\geq\varepsilon for all kk. But g⁢(ynk)g(y_{n_{k}}) is bounded (it’s a sequence in [a,b][a,b]) so, by the Bolzano-Weierstrass Theorem, it has a convergent subsequence g⁢(ynkl)→c∗g(y_{n_{k_{l}}})\rightarrow c_{*} for some c∗∈ℝc_{*}\in{\mathbb{R}}. So, choose y∈(f⁢(a),f⁢(b))y\in(f(a),f(b)) and let (yn)(y_{n}) be any sequence in (f⁢(a),f⁢(b))\{y}(f(a),f(b)){\backslash}\{y\} converging to yy. sn:=g⁢(yn)−g⁢(y)yn−y.s_{n}:=\frac{g(y_{n})-g(y)}{y_{n}-y}.

converges to 1/f′⁢(g⁢(y))1/f^{\prime}(g(y)). Since gg is bijective, there is a unique c∈(a,b)c\in(a,b) such that y=f⁢(c)y=f(c). f⁢(xn)−f⁢(c)xn−c→f′⁢(c),\frac{f(x_{n})-f(c)}{x_{n}-c}\rightarrow f^{\prime}(c),

Remark 10.12

We can easily deduce an equivalent theorem in the case where f′⁢(x)<0f^{\prime}(x)<0 for all x∈(a,b)x\in(a,b). We just apply Theorem 10.11 to g⁢(x)=−f⁢(x)g(x)=-f(x). Try to formulate this theorem carefully, then prove it.

The Inverse Function Theorem allows us to deduce useful facts about lots of functions that are naturally defined as the inverse of some function we already understand, for example, square and other roots.

Corollary 10.13

Let pp be a fixed positive integer. Given any x∈[0,∞)x\in[0,\infty), there exists a unique number x1/p∈[0,∞)x^{1/p}\in[0,\infty), its 𝐩th\mathbf{p}^{\mbox{th}} root, whose pthp^{\mbox{th}} power is xx. The function

g:[0,∞)→[0,∞),g⁢(x)=x1/pg:[0,\infty)\rightarrow[0,\infty),\qquad g(x)=x^{1/p}

is continuous on [0,∞)[0,\infty) and differentiable on (0,∞)(0,\infty), and its derivative is

g′⁢(x)=1p⁢(x1/p)p−1.g^{\prime}(x)=\frac{1}{p(x^{1/p})^{p-1}}.
Proof.

Choose any R>0R>0 and consider the function f⁢(x)=xpf(x)=x^{p} on [0,R][0,R]. This satisfies the hypotheses of the Inverse Function Theorem (on [a,b]=[0,R][a,b]=[0,R]), so has an inverse function

g:[0,Rp]→[0,R]g:[0,R^{p}]\rightarrow[0,R]

which is continuous on [0,Rp][0,R^{p}] and differentiable on (0,Rp)(0,R^{p}), with derivative

g′⁢(x)=1f′⁢(g⁢(x)).g^{\prime}(x)=\frac{1}{f^{\prime}(g(x))}.

Now g⁢(x)=x1/pg(x)=x^{1/p} by definition, and f′⁢(x)=p⁢xp−1f^{\prime}(x)=px^{p-1}, so

g′⁢(x)=1p⁢(g⁢(x))p−1=1p⁢(x1/p)p−1.g^{\prime}(x)=\frac{1}{p(g(x))^{p-1}}=\frac{1}{p(x^{1/p})^{p-1}}.

Hence the claims hold on [0,Rp][0,R^{p}] for all R>0R>0, so hold on [0,∞)[0,\infty). ∎

10.4 Higher derivatives and Taylor’s Theorem

If a function f:D→ℝf:D\rightarrow{\mathbb{R}} is differentiable (everywhere on DD) then its derivative defines another function f′:D→ℝf^{\prime}:D\rightarrow{\mathbb{R}}, and we can ask whether this function, in turn, is differentiable. If it is, then its derivative is denoted f′′:D→ℝf^{\prime\prime}:D\rightarrow{\mathbb{R}} and called the second derivative of ff. Similarly, if f′′:D→ℝf^{\prime\prime}:D\rightarrow{\mathbb{R}} is differentiable, its derivative f′′′:D→ℝf^{\prime\prime\prime}:D\rightarrow{\mathbb{R}} is the third derivative of ff. Proceeding inductively, we can define the nthn^{\mbox{th}} derivative of ff, denoted f(n)f^{(n)} to be the derivative (if it exists), of f(n−1)f^{(n-1)}, where f(0)f^{(0)} is, by definition, just the function ff itself (so f(1)=f′f^{(1)}=f^{\prime}, f(2)=f′′f^{(2)}=f^{\prime\prime} etc.). If f(n):D→ℝf^{(n)}:D\rightarrow{\mathbb{R}} exists, we say that ff is nn times differentiable, and if ff is nn times differentiable for all n∈ℤ+n\in{\mathbb{Z}}^{+}, we say that ff is smooth.

An interesting interpretation of the Mean Value Theorem can be given as follows. Let I⊆ℝI\subseteq{\mathbb{R}} be an open interval, f:I→ℝf:I\rightarrow{\mathbb{R}} be differentiable, and choose and fix some a∈Ia\in I. Then for any other x∈Ix\in I, we may apply the Mean Value Theorem to f:[a,x]→ℝf:[a,x]\rightarrow{\mathbb{R}} (if x>ax>a) or f:[x,a]→ℝf:[x,a]\rightarrow{\mathbb{R}} (if x<ax<a) to deduce that there exists cc between aa and xx such that

f⁢(x)=f⁢(a)+f′⁢(c)⁢(x−a).f(x)=f(a)+f^{\prime}(c)(x-a).

Note that the same equation holds trivially in the case where x=ax=a (with c=x=ac=x=a). So, for xx close to aa (meaning that |x−a||x-a| is small) we can approximate ff by the constant f⁢(a)f(a), and the error in this approximation is of size |f′⁢(c)|⁢|x−a||f^{\prime}(c)||x-a|, where cc is somewhere between aa and xx (and, of course, depends on xx).

The moral is that differentiability (once) of ff on an open interval allows one to approximate ff by a degree 0 polynomial function (i.e. a constant), with an error controlled by f′f^{\prime}. It turns out that if ff is nn times differentiable one can do better: one can approximate ff by a degree (n−1)(n-1) polynomial function, with an error controlled by f(n)f^{(n)}.

Summary

  • •
    ​

    The Extreme Value Theorem: A continuous function on a closed, bounded interval f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} is bounded above and below, and attains both a maximum and a minimum value.

  • •
    ​

    The Interior Extremum Theorem: If a differentiable function f⁢(a,b)→ℝf(a,b)\rightarrow{\mathbb{R}} attains a maximum or a minimum at c∈(a,b)c\in(a,b) then f′⁢(c)=0f^{\prime}(c)=0.

  • •
    ​

    The Mean Value Theorem: If a function ff is continuous on [a,b][a,b] and differentiable on (a,b)(a,b), then there exists c∈(a,b)c\in(a,b) such that

    f′⁢(c)=f⁢(b)−f⁢(a)b−a.f^{\prime}(c)=\frac{f(b)-f(a)}{b-a}.
  • •
    ​

    We can use the Mean Value Theorem to show that functions f:I→ℝf:I\rightarrow{\mathbb{R}}, where I⊆ℝI\subseteq{\mathbb{R}} is an interval, are increasing if f′⁢(x)≥0f^{\prime}(x)\geq 0 for all x∈Ix\in I (and decreasing if f′⁢(x)≤0f^{\prime}(x)\leq 0 for all x∈Ix\in I).

  • •
    ​

    The definitions of increasing, decreasing, maximum and minimum have nothing to do with calculus. But we can use calculus to find/understand them.

  • •
    ​

    A function f:D→ℝf:D\rightarrow{\mathbb{R}} is nn times differentiable if all its derivatives up to the nthn^{\rm th} exists everywhere in DD. It is smooth if it is nn times differentiable for all n∈ℤ+n\in{\mathbb{Z}}^{+}.

  • •
    ​

    A function which is (n+1)(n+1) times differentiable can be approximated by a polynomial of degree nn, with an error controlled by f(n+1)f^{(n+1)}, its (n+1)th(n+1)^{\rm th} derivative. This follows from Taylor’s Theorem.
