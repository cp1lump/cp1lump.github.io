9 Differentiable functions

9.1 The main definition

OK, let’s build calculus. Now that we have a rigorous definition of the limit of a function, it is straightforward to define derivatives.

Definition 9.1

Let f:D→ℝf:D\rightarrow{\mathbb{R}}, where DD is some subset of ℝ{\mathbb{R}}, and a∈Da\in D be a cluster point of DD. Then ff is differentiable at aa if the limit

limx→af⁢(x)−f⁢(a)x−a\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}

exists. In this case, we denote the limit f′⁢(a)f^{\prime}(a) and call it the derivative of ff at aa. We say that ff is differentiable if it is differentiable at aa for all a∈Da\in D.

Remarks

  • •
    ​

    In general, in order to define limx→ag⁢(x)\lim_{x\rightarrow a}g(x), we only need aa to be a cluster point of the domain of gg: it isn’t necessary in general for aa to be in the domain of gg, so g⁢(a)g(a) may or may not exist. For example

    limx→1x2−1x−1\lim_{x\rightarrow 1}\frac{x^{2}-1}{x-1}

    exists, although the function is undefined at x=1x=1. Note, however, that the limit defining f′⁢(a)f^{\prime}(a) contains the number f⁢(a)f(a), so to be differentiable at aa, the point aa must be both an element and a cluster point of the domain of ff. For example, a function f:(0,∞)→ℝf:(0,\infty)\rightarrow{\mathbb{R}} cannot be differentiable at 0 (since 0∉(0,∞)0\notin(0,\infty)), and a function f:ℤ→ℝf:{\mathbb{Z}}\rightarrow{\mathbb{R}} cannot be differentiable anywhere (since ℤ{\mathbb{Z}} has no cluster points).

  • •
    ​

    The limit in Definition 9.1 is defined as in Definition 7.1. That is, f:D→ℝf:D\rightarrow{\mathbb{R}} is differentiable at aa if for all sequences (xn)(x_{n}) in D\{a}D{\backslash}\{a\} which converge to aa,

    f⁢(xn)−f⁢(a)xn−a→f′⁢(a).\frac{f(x_{n})-f(a)}{x_{n}-a}\rightarrow f^{\prime}(a).
  • •
    ​

    Geometrically, the quantity

    sn=f⁢(xn)−f⁢(a)xn−as_{n}=\frac{f(x_{n})-f(a)}{x_{n}-a}

    is the slope of the straight line passing through the points (a,f⁢(a))(a,f(a)) and (xn,f⁢(xn))(x_{n},f(x_{n})). Such a straight line is called a chord on the graph y=f⁢(x)y=f(x). As xnx_{n} approaches aa, the points defining the chord get “arbitrarily close” to one another, and the chords approach a straight line through (a,f⁢(a))(a,f(a)) with slope f′⁢(a)f^{\prime}(a), the so-called tangent line. This is why the derivative f′⁢(a)f^{\prime}(a) is often interpreted as the slope of the graph y=f⁢(x)y=f(x) at the point (a,f⁢(a))(a,f(a)). Note, however, that this is just an interpretation. The definition of f′⁢(a)f^{\prime}(a) is in terms of limits which are, in turn, defined precisely in Definition 7.1.

  • •
    ​

    If f:D→ℝf:D\rightarrow{\mathbb{R}} is differentiable, then its derivative is also a function f′:D→ℝf^{\prime}:D\rightarrow{\mathbb{R}}, mapping DD to ℝ{\mathbb{R}}. There is a very popular alternative notation for the derivative in this case, namely

    f′⁢(x)=d⁢fd⁢x.f^{\prime}(x)=\frac{df}{dx}.

    This notation is convenient in some circumstances, but it tends to blur the distinction between a function (in this case f′f^{\prime}) and the value of the function at a particular point (in this case f′⁢(x)f^{\prime}(x)), so we will tend to avoid it.

Let’s verify that some simple, familiar functions have the derivatives we expect.

Example 9.2

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} such that f⁢(x)=cf(x)=c, a constant. Then ff is differentiable (everywhere) and f′⁢(a)=0f^{\prime}(a)=0 for all a∈ℝa\in{\mathbb{R}}.

Proof.

Choose any a∈ℝa\in{\mathbb{R}} and let xnx_{n} be any sequence in ℝ\{a}{\mathbb{R}}{\backslash}\{a\} which converges to aa. Then

f⁢(xn)−f⁢(a)xn−a=c−cxn−a=0→0.\frac{f(x_{n})-f(a)}{x_{n}-a}=\frac{c-c}{x_{n}-a}=0\rightarrow 0.

Hence

limx→af⁢(x)−f⁢(a)x−a=0\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}=0

as was to be proved. ∎

Example 9.3

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} such that f⁢(x)=xf(x)=x. Then ff is differentiable (everywhere) and f′⁢(a)=1f^{\prime}(a)=1 for all a∈ℝa\in{\mathbb{R}}.

Proof.

Choose any a∈ℝa\in{\mathbb{R}} and let xnx_{n} be any sequence in ℝ\{a}{\mathbb{R}}{\backslash}\{a\} which converges to aa. Then

f⁢(xn)−f⁢(a)xn−a=xn−axn−a=1→1.\frac{f(x_{n})-f(a)}{x_{n}-a}=\frac{x_{n}-a}{x_{n}-a}=1\rightarrow 1.

Hence

limx→af⁢(x)−f⁢(a)x−a=1\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}=1

as was to be proved. ∎

Example 9.4

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} such that f⁢(x)=x2f(x)=x^{2}. Then ff is differentiable and f′⁢(a)=2⁢af^{\prime}(a)=2a for all a∈ℝa\in{\mathbb{R}}.

Proof.

Choose any a∈ℝa\in{\mathbb{R}} and let xnx_{n} be any sequence in ℝ\{a}{\mathbb{R}}{\backslash}\{a\} which converges to aa. Then

f⁢(xn)−f⁢(a)xn−a=xn2−a2xn−a=(xn−a)⁢(xn+a)xn−a=xn+a→a+a.\frac{f(x_{n})-f(a)}{x_{n}-a}=\frac{x_{n}^{2}-a^{2}}{x_{n}-a}=\frac{(x_{n}-a)(% x_{n}+a)}{x_{n}-a}={x_{n}+a}\rightarrow a+a.

Hence

limx→af⁢(x)−f⁢(a)x−a=2⁢a\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}=2a

as was to be proved. ∎

Example 9.5

Let f:ℝ\{0}→ℝf:{\mathbb{R}}{\backslash}\{0\}\rightarrow{\mathbb{R}} such that f⁢(x)=1/xf(x)=1/x. Then ff is differentiable and f′⁢(a)=−1/a2f^{\prime}(a)=-1/a^{2} for all a∈ℝ\{0}a\in{\mathbb{R}}{\backslash}\{0\}.

Proof.

Choose any a≠0a\neq 0 and let xnx_{n} be any sequence in ℝ\{0,a}{\mathbb{R}}{\backslash}\{0,a\} which converges to aa. Then

f⁢(xn)−f⁢(a)xn−a\displaystyle\frac{f(x_{n})-f(a)}{x_{n}-a} =\displaystyle= 1/xn−1/axn−a=a−xnxn⁢a⁢(xn−a)=−1xn⁢a→−1a2\displaystyle\frac{1/x_{n}-1/a}{x_{n}-a}=\frac{a-x_{n}}{x_{n}a(x_{n}-a)}=-% \frac{1}{x_{n}a}\rightarrow-\frac{1}{a^{2}}

by the Algebra of Limits (Theorem 2.15). Hence

limx→af⁢(x)−f⁢(a)x−a=−1a2\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}=-\frac{1}{a^{2}}

as was to be proved.

∎

OK, all is as we expected. What about functions which are not differentiable? How do we prove they aren’t? A function f:D→ℝf:D\rightarrow{\mathbb{R}} is not differentiable at aa precisely if the limit

limx→af⁢(x)−f⁢(a)x−a\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}

does not exist. To establish this. it suffices to find one example of a sequence (xn)(x_{n}) in D\{a}D{\backslash}\{a\} which converges to aa for which

f⁢(xn)−f⁢(a)xn−a\frac{f(x_{n})-f(a)}{x_{n}-a}

does not converge.

Example 9.6

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} such that f⁢(x)=|x|f(x)=|x|. Then ff is not differentiable at 0.

Proof.

Assume, towards a contradiction, that ff is differentiable at 0 with derivative f′⁢(0)f^{\prime}(0). Then

limx→0f⁢(x)−f⁢(0)x−0=f′⁢(0).\lim_{x\rightarrow 0}\frac{f(x)-f(0)}{x-0}=f^{\prime}(0).

Let xn=(−1)n/nx_{n}=(-1)^{n}/n. Note that this is a sequence in ℝ\{0}{\mathbb{R}}{\backslash}\{0\} converging to 0. Hence,

f⁢(xn)−f⁢(0)xn−0→f′⁢(0).\frac{f(x_{n})-f(0)}{x_{n}-0}\rightarrow f^{\prime}(0).

But

f⁢(xn)−f⁢(0)xn−0=|(−1)n/n|(−1)n/n=1(−1)n=(−1)n\frac{f(x_{n})-f(0)}{x_{n}-0}=\frac{|(-1)^{n}/n|}{(-1)^{n}/n}=\frac{1}{(-1)^{n% }}=(-1)^{n}

which does not converge, a contradiction. Hence ff is not differentiable at 0. ∎

Remark Alternatively, we could have defined yn=1/ny_{n}=1/n and zn=−1/nz_{n}=-1/n and noted that both are sequences in ℝ\{0}{\mathbb{R}}{\backslash}\{0\} converging to 0, so if ff is differentiable at 0, then both

f⁢(yn)−f⁢(0)yn−0andf⁢(zn)−f⁢(0)zn−0\frac{f(y_{n})-f(0)}{y_{n}-0}\quad\mbox{and}\quad\frac{f(z_{n})-f(0)}{z_{n}-0}

must converge to f′⁢(0)f^{\prime}(0). But the first converges to 11, while the second converges to −1-1, which contradicts the uniqueness of limits (Theorem 7.5).

There is a link between differentiablity and continuity, as we now show.

Proposition 9.8

Let f:D→ℝf:D\rightarrow{\mathbb{R}} be differentiable at a∈Da\in D. Then ff is continuous at aa.

Proof.

Let (xn)(x_{n}) be any sequence in DD which converges to aa. Choose any sequence (yn)(y_{n}) in D\{a}D{\backslash}\{a\} which converges to aa (such a sequence must exist since aa is a cluster point of DD) and let

zn:={xnif xn≠aynif xn=a.z_{n}:=\left\{\begin{array}[]{cc}x_{n}&\mbox{if $x_{n}\neq a$}\\ y_{n}&\mbox{if $x_{n}=a$.}\end{array}\right.

I leave it as an exercise to prove that zn→az_{n}\rightarrow a. Now

|f⁢(xn)−f⁢(a)|={|f⁢(zn)−f⁢(a)zn−a|⁢|zn−a|if xn≠a0if xn=a}≤|f⁢(zn)−f⁢(a)zn−a|⁢|zn−a|.|f(x_{n})-f(a)|=\left\{\begin{array}[]{cc}\left|\frac{f(z_{n})-f(a)}{z_{n}-a}% \right||z_{n}-a|&\mbox{if $x_{n}\neq a$}\\ 0&\mbox{if $x_{n}=a$}\end{array}\right\}\leq\left|\frac{f(z_{n})-f(a)}{z_{n}-a% }\right||z_{n}-a|.

Since ff is differentiable at aa and zn→az_{n}\rightarrow a,

f⁢(zn)−f⁢(a)zn−a→f′⁢(a).\frac{f(z_{n})-f(a)}{z_{n}-a}\rightarrow f^{\prime}(a).

Hence, by the Algebra of Limits,

|f⁢(zn)−f⁢(a)zn−a|⁢|zn−a|→|f′⁢(a)|×0=0,\left|\frac{f(z_{n})-f(a)}{z_{n}-a}\right||z_{n}-a|\rightarrow|f^{\prime}(a)|% \times 0=0,

so, by the Squeeze Rule |f⁢(xn)−f⁢(a)|→0|f(x_{n})-f(a)|\rightarrow 0, that is, f⁢(xn)→f⁢(a)f(x_{n})\rightarrow f(a). Since this holds for all such sequences (xn)(x_{n}), ff is continuous at aa. ∎

Remark. The converse of Proposition 9.8 is false: if ff is continuous at aa, it does not follow that ff is differentiable there. We’ve already seen a counterexample: f⁢(x)=|x|f(x)=|x| is continuous at 0 but is not differentiable at 0 (see Example 9.6). In this case, the function fails to be differentiable only at a single isolated point. It is straightforward to construct functions which are differentiable only at a single isolated point.

Example 9.9

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} be defined such that

f⁢(x)={xif x∈ℚ,x−x2if x∉ℚ.f(x)=\left\{\begin{array}[]{ll}x&\mbox{if $x\in{\mathbb{Q}}$,}\\ x-x^{2}&\mbox{if $x\notin{\mathbb{Q}}$.}\end{array}\right.

I claim that ff is differentiable at 0, with f′⁢(0)=1f^{\prime}(0)=1, but is not differentiable anywhere else.

Proof.

Let a∈ℝa\in{\mathbb{R}} and assume that ff is differentiable at aa. Then, by Proposition 9.8, ff is continuous at aa. For each n∈ℤ+n\in{\mathbb{Z}}^{+} there exist rn∈ℚr_{n}\in{\mathbb{Q}} and in∈ℝ\ℚi_{n}\in{\mathbb{R}}{\backslash}{\mathbb{Q}} such that a<rn<a+1/na<r_{n}<a+1/n and a<in<a+1/na<i_{n}<a+1/n. Clearly rn→ar_{n}\rightarrow a and in→ai_{n}\rightarrow a by the Squeeze Rule. Hence, by the definition of continuity, f⁢(rn)→f⁢(a)f(r_{n})\rightarrow f(a) and f⁢(in)→f⁢(a)f(i_{n})\rightarrow f(a). But rnr_{n} is rational, so f⁢(rn)=rn→af(r_{n})=r_{n}\rightarrow a, and ini_{n} is irrational, so f⁢(in)=in−in2→a−a2f(i_{n})=i_{n}-i_{n}^{2}\rightarrow a-a^{2}. Both these limits equal f⁢(a)f(a), so a=a−a2a=a-a^{2}, whence a=0a=0. So if ff is differentiable at aa, then a=0a=0. Equivalently, ff is not differentiable at any a≠0a\neq 0.

It remains to show that ff is differentiable at 0 with f′⁢(0)=1f^{\prime}(0)=1, that is,

limx→0f⁢(x)−f⁢(0)x−0=1.\lim_{x\rightarrow 0}\frac{f(x)-f(0)}{x-0}=1.

So, let (xn)(x_{n}) be any sequence in ℝ\{0}{\mathbb{R}}{\backslash}\{0\} converging to 0, and consider

sn=f⁢(xn)−f⁢(0)xn−0=f⁢(xn)xn.s_{n}=\frac{f(x_{n})-f(0)}{x_{n}-0}=\frac{f(x_{n})}{x_{n}}.

We must show that sn→1s_{n}\rightarrow 1. If xnx_{n} is rational, then |sn−1|=|xn/xn−1|=0|s_{n}-1|=|x_{n}/x_{n}-1|=0, whereas if xnx_{n} is irrational then

|sn−1|=|xn−xn2xn−1|=|xn|.|s_{n}-1|=\left|\frac{x_{n}-x_{n}^{2}}{x_{n}}-1\right|=|x_{n}|.

Hence, for all nn, 0≤|sn−1|≤|xn|0\leq|s_{n}-1|\leq|x_{n}|, so |sn−1|→0|s_{n}-1|\rightarrow 0 by the Squeeze Rule. Hence sn→1s_{n}\rightarrow 1. It follows that f′⁢(0)=1f^{\prime}(0)=1, as claimed. ∎

So this (admittedly rather bizarre) function is differentiable at 0, and has a positive derivative there. Naive intuition would suggest, therefore, that the function should be increasing, at least for values of xx sufficiently close to 0. In fact, this is false!

Example 9.9 continued Claim: the function ff is not increasing on any neighbourhood of 0. That is, there does not exist ε>0\varepsilon>0 such that ff is increasing on (−ε,ε)(-\varepsilon,\varepsilon).

Proof.

Let ε>0\varepsilon>0 be given. Then there exists an irrational number xx such that 0<x<ε0<x<\varepsilon. By definition, f⁢(x)=x−x2<xf(x)=x-x^{2}<x. Similarly, there exists a rational number yy such that x−x2<y<xx-x^{2}<y<x. But then y<xy<x and f⁢(y)=y>x−x2=f⁢(x)f(y)=y>x-x^{2}=f(x). Hence, ff is not increasing on the interval (−ε,ε)(-\varepsilon,\varepsilon). This is true no matter which positive number ε\varepsilon we choose, which establishes the claim. ∎

9.2 The rules of differentiation

In principle, if we are given an explicit function, like

f:ℝ→ℝ,f⁢(x)=(x3+1)2,f:{\mathbb{R}}\rightarrow{\mathbb{R}},\qquad f(x)=(x^{3}+1)^{2},

we can show it is differentiable and compute its derivative by directly applying Definition 9.1. This quickly becomes complicated and tedious, however. So we next develop some “derivative theorems” which will allow us to reduce differentiation of many functions to an algorithmic process. You are (I would hope) already familiar with these and quite fluent in applying them. The point of this section is to show that, now we have a mathematically precise definition of the derivative, we can rigorously prove them, thus putting differential calculus on a solid foundation.

Proposition 9.10 (Linearity)

Let f:D→ℝf:D\rightarrow{\mathbb{R}}, g:D→ℝg:D\rightarrow{\mathbb{R}} be differentiable at a∈Da\in D with derivatives f′⁢(a)f^{\prime}(a) and g′⁢(a)g^{\prime}(a), respectively, and cc be a real constant. Then

  1. (i)
    ​

    c⁢f:D→ℝcf:D\rightarrow{\mathbb{R}} is differentiable at aa with derivative c⁢f′⁢(a)cf^{\prime}(a).

  2. (ii)
    ​

    f+g:D→ℝf+g:D\rightarrow{\mathbb{R}} is differentiable at aa with derivative f′⁢(a)+g′⁢(a)f^{\prime}(a)+g^{\prime}(a).

Proof.

These both follow immediately from the Algebra of Limits for functions (Theorem 7.10):

limx→ac⁢f⁢(x)−c⁢f⁢(a)x−a\displaystyle\lim_{x\rightarrow a}\frac{cf(x)-cf(a)}{x-a} =\displaystyle= limx→ac⁢(f⁢(x)−f⁢(a)x−a)=c⁢f′⁢(a)\displaystyle\lim_{x\rightarrow a}c\left(\frac{f(x)-f(a)}{x-a}\right)=cf^{% \prime}(a)
limx→af⁢(x)+g⁢(x)−(f⁢(a)+g⁢(a))x−a\displaystyle\lim_{x\rightarrow a}\frac{f(x)+g(x)-(f(a)+g(a))}{x-a} =\displaystyle= limx→a(f⁢(x)−f⁢(a)x−a+g⁢(x)−g⁢(a)x−a)=f′⁢(a)+g′⁢(a)\displaystyle\lim_{x\rightarrow a}\left(\frac{f(x)-f(a)}{x-a}+\frac{g(x)-g(a)}% {x-a}\right)=f^{\prime}(a)+g^{\prime}(a)

∎

Combining parts (i) and (ii) of Proposition 9.10, we see that linear combinations of differentiable functions are differentiable: for f,gf,g differentiable at aa and any constants α,β\alpha,\beta,

h⁢(x)=α⁢f⁢(x)+β⁢g⁢(x)h(x)=\alpha f(x)+\beta g(x)

is differentiable at aa, with derivative

h′⁢(a)=α⁢f′⁢(a)+β⁢g′⁢(a).h^{\prime}(a)=\alpha f^{\prime}(a)+\beta g^{\prime}(a).

What about products of differentiable functions? If ff and gg are differentiable at aa, is h⁢(x)=f⁢(x)⁢g⁢(x)h(x)=f(x)g(x) differentiable at aa? And if so, is h′⁢(a)=f′⁢(a)⁢g′⁢(a)h^{\prime}(a)=f^{\prime}(a)g^{\prime}(a)? Yes (to the first) and no (to the second):

Proposition 9.11 (Product Rule)

Let f:D→ℝf:D\rightarrow{\mathbb{R}}, g:D→ℝg:D\rightarrow{\mathbb{R}} be differentiable at a∈Da\in D with derivatives f′⁢(a)f^{\prime}(a) and g′⁢(a)g^{\prime}(a), respectively. Then f⁢g:D→ℝfg:D\rightarrow{\mathbb{R}} is differentiable at aa with derivative f′⁢(a)⁢g⁢(a)+f⁢(a)⁢g′⁢(a)f^{\prime}(a)g(a)+f(a)g^{\prime}(a).

Proof.

Let h:D→ℝh:D\rightarrow{\mathbb{R}}, h⁢(x)=f⁢(x)⁢g⁢(x)h(x)=f(x)g(x). We must show that

limx→ah⁢(x)−h⁢(a)x−a=f′⁢(a)⁢g⁢(a)+f⁢(a)⁢g′⁢(a).\lim_{x\rightarrow a}\frac{h(x)-h(a)}{x-a}=f^{\prime}(a)g(a)+f(a)g^{\prime}(a).

So, let (xn)(x_{n}) be any sequence in D\{a}D{\backslash}\{a\} converging to aa. Then

h⁢(xn)−h⁢(a)xn−a\displaystyle\frac{h(x_{n})-h(a)}{x_{n}-a} =\displaystyle= f⁢(xn)⁢g⁢(xn)−f⁢(a)⁢g⁢(a)xn−a\displaystyle\frac{f(x_{n})g(x_{n})-f(a)g(a)}{x_{n}-a}
=\displaystyle= f⁢(xn)⁢g⁢(xn)−f⁢(a)⁢g⁢(xn)+f⁢(a)⁢g⁢(xn)−f⁢(a)⁢g⁢(a)xn−a\displaystyle\frac{f(x_{n})g(x_{n})-f(a)g(x_{n})+f(a)g(x_{n})-f(a)g(a)}{x_{n}-a}
=\displaystyle= f⁢(xn)−f⁢(a)xn−a⁢g⁢(xn)+f⁢(a)⁢g⁢(xn)−f⁢(a)⁢g⁢(a)xn−a.\displaystyle\frac{f(x_{n})-f(a)}{x_{n}-a}g(x_{n})+f(a)\frac{g(x_{n})-f(a)g(a)% }{x_{n}-a}.

Now ff is differentiable at aa, so

f⁢(xn)−f⁢(a)xn−a→f′⁢(a).\frac{f(x_{n})-f(a)}{x_{n}-a}\rightarrow f^{\prime}(a).

Similarly, gg is differentiable at aa, so

g⁢(xn)−g⁢(a)xn−a→g′⁢(a).\frac{g(x_{n})-g(a)}{x_{n}-a}\rightarrow g^{\prime}(a).

Furthermore, since gg is differentiable at aa, it is continuous at aa, so

g⁢(xn)→g⁢(a).g(x_{n})\rightarrow g(a).

Hence, by the Algebra of Limits (for sequences, Theorem 2.15),

h⁢(xn)−h⁢(a)xn−a→f′⁢(a)×g⁢(a)+f⁢(a)×g′⁢(a).\frac{h(x_{n})-h(a)}{x_{n}-a}\rightarrow f^{\prime}(a)\times g(a)+f(a)\times g% ^{\prime}(a).

Since this holds for all such sequences (xn)(x_{n}), the claim is proved. ∎

Propositions 9.10 and 9.11 followed easily from the Algebra of Limits. Our next differentiation rule, and its proof, are considerably more subtle. Recall that the composition of two functions f:A→Bf:A\rightarrow B and g:B→Cg:B\rightarrow C, denoted g∘f:A→Cg\circ f:A\rightarrow C is the function which first “does” ff, then feeds the result into gg, that is

(g∘f)⁢(x)=g⁢(f⁢(x)).(g\circ f)(x)=g(f(x)).

This only makes sense if the range of ff is a subset of the domain of gg. We’ve already proved that the composition of two continuous functions is continuous (Theorem 8.5). There’s a very famous rule which tells us that the composition of two differentiable functions is differentiable, and gives a formula for its derivative.

Theorem 9.13 (Chain Rule)

Let f:D→Ef:D\rightarrow E be differentiable at a∈Da\in D and g:E→ℝg:E\rightarrow{\mathbb{R}} be differentiable at f⁢(a)∈Ef(a)\in E. Then g∘f:D→ℝg\circ f:D\rightarrow{\mathbb{R}} is differentiable at aa, and

(g∘f)′⁢(a)=g′⁢(f⁢(a))⁢f′⁢(a).(g\circ f)^{\prime}(a)=g^{\prime}(f(a))f^{\prime}(a).
Proof.

We consider two cases separately: (i) f′⁢(a)≠0f^{\prime}(a)\neq 0 and (ii) f′⁢(a)=0f^{\prime}(a)=0.

(i) Let (xn)(x_{n}) be any sequence in D\{a}D{\backslash}\{a\} which converges to aa and consider the sequence

sn=(g∘f)⁢(xn)−(g∘f)⁢(a)xn−a.s_{n}=\frac{(g\circ f)(x_{n})-(g\circ f)(a)}{x_{n}-a}.

We must show that sn→g′⁢(f⁢(a))⁢f′⁢(a)s_{n}\rightarrow g^{\prime}(f(a))f^{\prime}(a).

Let’s do an example where we compute a derivative using the Chain Rule. The point is not so much to calculate the derivative, but rather to show how the Chain Rule (now that we’ve established it as a theorem) gives us a proof that the claimed formula is correct.

Remark You are probably more familiar with the Chain Rule expressed something like the following:

d⁢yd⁢x=d⁢yd⁢ud⁢ud⁢x.(∗)\qquad\qquad\qquad\qquad\frac{dy}{dx}=\frac{dy}{du}\frac{du}{dx}.\qquad\qquad% \qquad\qquad(*)

Certainly, this has a look of plausibility about it (don’t the “d⁢udu’s” just cancel?) and is arguably more memorable than

(g∘f)′(a)=g′(f(a))f′(a).(†)\qquad\qquad\qquad\qquad(g\circ f)^{\prime}(a)=g^{\prime}(f(a))f^{\prime}(a).% \qquad\qquad\qquad\qquad(\dagger)

Nonetheless, it is vastly inferior. Ask yourself what, exactly, does (∗)(*) mean? The derivative of a function yy (where? At xx?) equals the derivative (where? At uu?) of the same (?) function – but with respect to a different variable, whatever that means — times the derivative of some other function uu (but isn’t uu a variable? Can we use the same symbol to denote both a function and a variable?) evaluated somewhere (xx? Maybe?). Note that “the function yy thought of as a function of uu instead of xx,” where “uu is a function of xx,” is actually a different function! And to interpret (∗)(*) we’re forced to talk about how we think about the function yy, instead of what it actually is. This is the hallmark of imprecise, badly formulated mathematics.

By contrast (†)(\dagger) is a completely unambiguous, precise statement standing on its own. We don’t need to explain separately “how we’re thinking of” the terms it contains in order to make sense of it. Be warned: if ever I ask for a statement of the Chain Rule, it is Theorem 9.13 I want, not wooly, ambiguous stuff in the style of (∗)(*). □\Box
