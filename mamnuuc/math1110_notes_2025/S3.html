3 Sets of real numbers

You will have studied sets in Core Mathematics over the last few weeks, and we now remind ourselves of some set notation.

  • •
    ​

    Recall that a set is a collection of objects, real or of the imagination.

  • •
    ​

    Given a set AA, we write x∈Ax\in A to denote that object xx is in set AA. This is usually translated into words (when reading the expression) as “xx is an element of AA”. The expression x∉Ax\notin A means xx is not an element of AA.

  • •
    ​

    Another set BB is a subset of AA if every element of BB is an element of AA. This is denoted B⊆AB\subseteq A. Note that every set is a subset of itself, A⊆AA\subseteq A. Two sets AA, BB are equal if they have precisely the same elements, in which case A⊆BA\subseteq B and B⊆AB\subseteq A.

  • •
    ​

    We have already seen several sets of numbers such as the real numbers ℝ\mathbb{R}, the positive integers ℤ+\mathbb{Z}^{+}, the integers ℤ\mathbb{Z} and the rational numbers (or quotients) ℚ\mathbb{Q} (see section 1.1).

  • •
    ​

    A further very important set if the empty set, denoted ∅︀\emptyset. This is the set containing no objects. Do not confuse it with {∅︀}\{\emptyset\}, which certainly isn’t the empty set since it contains an object (namely, the empty set)!

One of the most common ways of specifying a set is by simply listing its elements inside curly brackets, for example

A={1,2,3,4,5,6,7,8}.A=\{1,2,3,4,5,6,7,8\}\ .

A second if by giving a condition which elements must satisfy,

A={x:x∈ℤ+,x≤8}.A=\{x\>:\>x\in{\mathbb{Z}}^{+},\,x\leq 8\}.

In general A={x:P⁢(x)}A=\{x\>:\>P(x)\} means the set of objects xx for which condition P⁢(x)P(x) is true. If part of condition P⁢(x)P(x) is that xx should be in a particular set, we often move that part to the left of the colon. For example, the set above could be written

A={x∈ℤ+:x≤8}A=\{x\in{\mathbb{Z}}^{+}\>:\>x\leq 8\}

and we could write the rational numbers as

ℚ={x∈ℝ:x=ab⁢ where ⁢a,b∈ℤ,b≠0}.\mathbb{Q}=\left\{x\in\mathbb{R}:x=\frac{a}{b}\text{ where }a,b\in\mathbb{Z},b\neq 0\right\}\ .

3.1 Boundedness and the axiom of completeness

When we first learn about real numbers, it is common to think of them as infinite decimal expansions, that is, expressions of the form

20142108.030118151205⁢…20142108.030118151205\ldots

with finitely many digits taken from {0,1,…,9}\{0,1,\ldots,9\} to the left of a dot (the “decimal point”) and infinitely many to the right (possibly all but finitely many of these being 0). This is OK, but it has some disadvantages: the choice of base 10 for the expansion is pretty arbitrary, and there’s the rather annoying fact that different expansions can represent the same number, for example,

0.999999⁢…=1.000000⁢….0.999999\ldots=1.000000\ldots\,.

For our purposes, it is better to take a radically different approach: rather than attempt to define the set of real numbers constructively, we will characterize it by stating the properties that we assume it has. These properties come in three stages – we have already seen two of these.

First, we assume that ℝ{\mathbb{R}} is a field, that is, a set on which the four basic operations of arithmetic (+,×,−,÷)(+,\times,-,\div) are well defined and have the familiar properties (e.g. x+y=y+xx+y=y+x and x×(y+z)=(x×y)+(x×z)x\times(y+z)=(x\times y)+(x\times z)).

Second, we assume that ℝ{\mathbb{R}} is an ordered field: it is equipped with an ordering relation <<. This is a structure which, given a pair of distinct elements x,y∈ℝx,y\in{\mathbb{R}}, x≠yx\neq y, determines which of them is the smaller. So, for any x,y∈ℝx,y\in\mathbb{R}, exactly one of x=yx=y, x<yx<y or y<xy<x is true. The ordering relation << is also assumed to satisfy the usual properties (e.g. if x<yx<y then x+z<y+zx+z<y+z for all real numbers zz).

These first two structures (the arithmetic operations and ordering relation) have exactly the properties you’ve been familar with since primary school, so we won’t labour them here.22 2 If you’d like to see a systematic development of the idea of an ordered field, you’ll find it in most textbooks on real analysis. A Sequential Introduction to Real Analysis by J.M. Speight (World Scientific, Singapore, 2016) would be a great choice (have a look at who is lecturing the second part of this course). In fact, you already know an example of an ordered field, namely the set of rational numbers ℚ{\mathbb{Q}}.

So what is the difference between ℚ{\mathbb{Q}} and ℝ{\mathbb{R}}? For our purposes, the key difference is that ℝ{\mathbb{R}} satisfies an extra property called completeness, while ℚ{\mathbb{Q}} does not. This property is not familiar from school level maths, so we will study it in some depth. To define completeness requires some preliminary ideas.

Definition 3.2

Let AA be a subset of ℝ{\mathbb{R}}. An upper bound on AA is any K∈ℝK\in{\mathbb{R}} such that, for all x∈Ax\in A, x≤Kx\leq K. A lower bound on AA is any L∈ℝL\in{\mathbb{R}} such that, for all x∈Ax\in A, x≥Lx\geq L. If AA has an upper bound, we say that AA is bounded above. If AA has a lower bound, we say that AA is bounded below. AA is bounded if it is bounded both above and below.

Example 3.3

The set A=[−1,2)A=[-1,2) is bounded above, by 22 for example, and below, by −1-1 for example. Hence AA is bounded. Note that upper and lower bounds are not unique. That is, 2,54,19345⁢π2,54,19345\pi are all upper bounds on AA. In fact, any K≥2K\geq 2 is an upper bound on AA, and any L≤−1L\leq-1 is a lower bound on AA.

Remark 3.4

It is important not to confuse bounded with finite. A finite set is one containing a finite number of distinct elements. Any finite subset of ℝ{\mathbb{R}} will certainly be bounded, but the converse is clearly false: not every bounded subset of ℝ{\mathbb{R}} is finite, as the above example shows.

Example 3.5

The set of positive integers ℤ+{\mathbb{Z}}^{+} is bounded below, by 11 for example (or −54-54, or −1000-1000). However, it is not bounded above.

Let’s think carefully about what it means to say that a set AA is unbounded above. It means that AA has no upper bound. Equivalently, every real number KK is not an upper bound on AA. Hence, given any K∈ℝK\in{\mathbb{R}}, there exists x∈Ax\in A such that x>Kx>K.

So an alternative formulation of the statement “ℤ+{\mathbb{Z}}^{+} is unbounded above” is “given any real number KK there exists a positive integer nn such that n>Kn>K.”. We have already seen this property - it is the Archimedean property of ℝ\mathbb{R}!

Note that ℚ{\mathbb{Q}} also has the Archimedean Property (given any K∈ℚK\in{\mathbb{Q}} there is some n∈ℤ+n\in{\mathbb{Z}}^{+} such that n>Kn>K).

Given a set of real numbers, we would like to be able to use properties of the largest and smallest element. Our first attempt at a “largest” and “smallest” value in a set could be the following:

Definition 3.6

The maximum of a subset AA of ℝ{\mathbb{R}} (if it exists) is the element a∈Aa\in A such that for any other b∈Ab\in A, b≤ab\leq a.

The minimum of a subset AA of ℝ{\mathbb{R}} (if it exists) is the element a∈Aa\in A such that for any other b∈Ab\in A, b≥ab\geq a.

This works well for finite sets (i.e. sets with only a finite number of elements).

Example 3.7

Consider the set A={1,2,3,5,7}A=\{1,2,3,5,7\}. Then max⁡A=7\max A=7 and min⁡A=1\min A=1.

In fact we have already been using this notation - we have written max⁡{N1,N2}\max\{N_{1},N_{2}\} for the largest of the numbers N1N_{1} and N2N_{2}, and we now see that really we were writing them as the maximum value of the 2-element set {N1,N2}\{N_{1},N_{2}\}.

However, this definition starts to break down pretty quickly - for example, we already get into problems with intervals. We would like to say that [−1,2)[-1,2) has maximum 2, but in fact, as 2∉[−1,2)2\notin[-1,2) it has no maximum! Clearly, we want a definition which gives a “maximum” for any set that is bounded above.

Definition 3.8

The supremum of a subset AA of ℝ{\mathbb{R}} is its least upper bound (if this exists), denoted supA\sup A. So supA\sup A is a real number with two properties:

  1. (i)
    ​

    supA\sup A is an upper bound on AA.

  2. (ii)
    ​

    No number less than supA\sup A is an upper bound on AA.

Similarly the infimum of AA is its greatest lower bound (if this exists), denoted infA\inf A.

It follows immediately from the definition that supA\sup A, if it exists, is unique. For if two numbers both satisfy the above two properties and are different, then the smaller of them, K1K_{1} say, is an upper bound on AA by (i), and is less than the larger one, K2K_{2} say, contradicting (ii). So it makes sense to speak of the supremum and/or infimum of a subset of ℝ{\mathbb{R}}, in contrast to upper/lower bounds.

Example 3.9

Let A=[−1,2)A=[-1,2). Then, immediately from the definition of this kind of interval, supA=2\sup A=2 and infA=−1\inf A=-1.

Example 3.10

Let A={1n:n∈ℤ+}A=\{\frac{1}{n}\>:\>n\in{\mathbb{Z}}^{+}\}. Every element of AA is positive, so AA is bounded below, by 0. In fact, infA=0\inf A=0. Let’s prove this. We already know that 0 is a lower bound on AA, so it just remains to show that every number bigger than 0 is not a lower bound on AA. So, let K>0K>0. Then 1/K1/K is a real number, so by the Archimedean Property, there exists some n∈ℤ+n\in{\mathbb{Z}}^{+} such that n>1/Kn>1/K. But then 1/n∈A1/n\in A and 1/n<K1/n<K, so KK is not a lower bound on AA.

The supremum is a bit less subtle. I claim that supA=1\sup A=1. Again, let’s prove this. Let x∈Ax\in A. Then x=1/nx=1/n for some n∈ℤ+n\in{\mathbb{Z}}^{+}. But n≥1n\geq 1, so x≤1x\leq 1. Hence 11 is an upper bound on AA. Any K<1K<1 is not an upper bound on AA, since 1∈A1\in A.

The existence of suprema (== plural of supremum) is the defining difference between ℝ{\mathbb{R}} and ℚ{\mathbb{Q}}.

Axiom 3.11 (The Axiom of Completeness)

Every nonempty subset of ℝ{\mathbb{R}} which is bounded above has a supremum in ℝ{\mathbb{R}}.

The equivalent statement for ℚ{\mathbb{Q}} (replace ℝ{\mathbb{R}} by ℚ{\mathbb{Q}} in both places) is false. For example A={x∈ℚ:x2<2}A=\{x\in{\mathbb{Q}}\>:\>x^{2}<2\} is a nonempty subset of ℚ{\mathbb{Q}} (e.g. 0∈A0\in A) which is bounded above (e.g. every x∈Ax\in A is less than 22), but it has no supremum in ℚ{\mathbb{Q}}, that is, there is no rational number rr with the required properties that (i) rr is an upper bound on AA and (ii) no rational number s<rs<r is an upper bound on AA. The problem is, of course, that 2\sqrt{2} is not rational, and there is no smallest rational number bigger than 2\sqrt{2}.

The axiom of completeness looks rather lopsided: it says that every nonempty subset of ℝ{\mathbb{R}} which is bounded above, has a supremum, but says nothing about sets which are bounded below. That’s because the equivalent statement for infima (== plural of infimum) follows directly from Axiom 3.11. Proving this is a good exercise in keeping clear sight of what upper/lower bound and sup/inf mean:

Proposition 3.12

Let A⊂ℝA\subset{\mathbb{R}} be nonempty and bounded below. Then AA has an infimum.

Proof.

Let B={−x:x∈A}B=\{-x\>:\>x\in A\}, which is certainly nonempty. Let KK be a lower bound on AA. Then −K-K is an upper bound on BB. (Check: if y∈By\in B then −y∈A-y\in A, and hence −y≥K-y\geq K. Hence y≤−Ky\leq-K.) Hence, BB is a nonempty subset of ℝ{\mathbb{R}} which is bounded above so, by the Axiom of Completeness, BB has a supremum, LL say. Now −L-L is a lower bound on AA. (Check: if x∈Ax\in A then −x∈B-x\in B, so −x≤L-x\leq L. Hence x≥−Lx\geq-L.)

Let MM be any real number greater than −L-L. Then −M<L-M<L, and LL is the least upper bound on BB. Hence −M-M is not an upper bound on BB, so there exists y∈By\in B such that y>−My>-M. But then −y∈A-y\in A and −y<M-y<M, so MM is not a lower bound on AA. Hence, no number greater than −L-L is a lower bound on AA. Hence −L-L is the infimum of AA. ∎

3.2 Some consequences of the axiom of completeness

In this section, we will deduce some basic properties of the set ℝ{\mathbb{R}} by using Axiom 3.11. The first of these we have already asserted as Fact: the set ℤ+{\mathbb{Z}}^{+} is unbounded above. This looks entirely obvious, but is it really? As far as we’re concerned, ℝ{\mathbb{R}} is, by definition a complete ordered field (so we are not defining it to be the set of all decimal expansions, for example). Is it really obvious that a complete ordered field cannot have a magic element, xx say, which is bigger than every positive integer?

Theorem 3.13 (The Archimedean Property of ℝ{\mathbb{R}})

Given any real number KK, there is some positive integer nn such that n>Kn>K.

Proof.

Assume, to the contrary, that ℤ+{\mathbb{Z}}^{+} is bounded above. Then, by Axiom 3.11, ℤ+{\mathbb{Z}}^{+} has a supremum, UU, say. Since U−1<UU-1<U, it is not an upper bound on ℤ+{\mathbb{Z}}^{+}, so there exists n∈ℤ+n\in{\mathbb{Z}}^{+} such that n>U−1n>U-1. But then n+1∈ℤ+n+1\in{\mathbb{Z}}^{+}, and n+1>Un+1>U, contradicting the fact that UU is an upper bound on ℤ+{\mathbb{Z}}^{+}. ∎

That’s a relief! Note that the only property of ℤ+{\mathbb{Z}}^{+} we used in the above proof was the “successor property”: if nn is in ℤ+{\mathbb{Z}}^{+}, then n+1n+1 is in ℤ+{\mathbb{Z}}^{+} also. By an almost identical argument, we can show that any subset of ℝ{\mathbb{R}} which has the “predecessor property” must be unbounded below.

Lemma 3.14

Let A⊆ℝA\subseteq{\mathbb{R}} be any nonempty subset of ℝ{\mathbb{R}} with the property that whenever m∈Am\in A then m−1∈Am-1\in A. Then AA is unbounded below.

Proof.

Assume, to the contrary, that AA is bounded below. Then, by Proposition 3.12, AA has an infimum, LL say. Since L+1>LL+1>L, it is not a lower bound on AA, so there exists m∈Am\in A such that m<L+1m<L+1. But then, by our assumption about AA, m−1∈Am-1\in A also, and m−1<Lm-1<L. This contradicts the fact that LL is a lower bound on AA. ∎

As a corollary of these two results, we can prove that the set of rational numbers must be dense in the set of reals, in the following sense:

Theorem 3.15 (ℚ{\mathbb{Q}} is dense in ℝ{\mathbb{R}})

Between any pair of distinct real numbers, there is a rational number.

Proof.

Let’s call the smaller of the two real numbers xx and the larger yy. By the Archimedean Property, there exists n∈ℤ+n\in{\mathbb{Z}}^{+} such that n>1y−xn>\frac{1}{y-x}, and hence, n⁢y−1>n⁢xny-1>nx. We will show that there exists a rational number of the form m/nm/n between xx and yy.

Consider the set

E={m∈ℤ:m>n⁢x}.E=\{m\in{\mathbb{Z}}\>:\>m>nx\}.

This is nonempty, by the Archimedean Property again (if it weren’t, n⁢xnx would be an upper bound on ℤ{\mathbb{Z}}, and hence on ℤ+⊂ℤ{\mathbb{Z}}^{+}\subset{\mathbb{Z}}). We claim there exists m∈Em\in E such that m<n⁢ym<ny. Assume this is false, so every m∈Em\in E satisfies m≥n⁢ym\geq ny. Then, given any m∈Em\in E, m−1≥n⁢y−1>n⁢xm-1\geq ny-1>nx, so m−1∈Em-1\in E. Hence, EE has the “predecessor property”, so by Lemma 3.14, EE is unbounded below. But EE is clearly bounded below (by n⁢xnx) by definition, a contradiction. Hence, the claim is true: there exists m∈Em\in E with m<n⁢ym<ny. But then mm is an integer and n⁢x<m<n⁢ynx<m<ny, so x<mn<yx<\frac{m}{n}<y, as required. ∎

Recall that a real number is irrational if it is not rational. Since ℚ{\mathbb{Q}} is countable, we deduce immediately that the set of irrational numbers, ℝ\ℚ{\mathbb{R}}{\backslash}{\mathbb{Q}}, is uncountable. In particular, there certainly are irrational numbers in [0,1][0,1], although we do not yet have a single explicit example of such a number! In MATH1000 you may have seen that 2\sqrt{2} can’t be rational, but you have not yet proved that it exists at all! However, if you have completed the second Problem Set, you will have shown this. For now, we will assume that 2\sqrt{2} exists, and we will see more on this later in the course. From the existence of a single irrational number, we can easily deduce that the irrational numbers are, like the rational numbers, dense in ℝ{\mathbb{R}}:

Theorem 3.16 (ℝ\ℚ{\mathbb{R}}{\backslash}{\mathbb{Q}} is dense in ℝ{\mathbb{R}})

Between any pair of distinct real numbers there is an irrational number.

Proof.

Let’s call the smaller of the two real numbers xx and the larger yy. Let ww be an irrational number in [0,1][0,1]. Then w>0w>0, so by Theorem 3.15, there exists a rational number mn\frac{m}{n} such that

xw<mn<yw.\frac{x}{w}<\frac{m}{n}<\frac{y}{w}.

Hence mn⁢w\frac{m}{n}w lies between xx and yy and is certainly irrational (since if mn⁢w=pq\frac{m}{n}w=\frac{p}{q} then w=p⁢nq⁢mw=\frac{pn}{qm}, which is rational). ∎

You will also see in Core Maths that while ℚ\mathbb{Q} is countable, ℝ∖ℚ\mathbb{R}\setminus\mathbb{Q} is not. So, between any pair of rationals there’s an irrational, and between any pair of irrationals there’s a rational. But there are more irrationals between 0 and 11 than there are rationals in ℝ{\mathbb{R}}! This set ℝ{\mathbb{R}} is very, very strange…

Definition 3.17

A sequence (an)(a_{n}) is increasing if an+1≥ana_{n+1}\geq a_{n} for all n∈ℤ+n\in{\mathbb{Z}}^{+}. It is decreasing if an+1≤ana_{n+1}\leq a_{n} for all n∈ℤ+n\in{\mathbb{Z}}^{+}. It is monotone if it is increasing or decreasing.

Remarks

  • •
    ​

    It follows immediately from the definition that, if (an)(a_{n}) is increasing then an≥ama_{n}\geq a_{m} whenever n≥mn\geq m. Similarly, a decreasing sequence has an≤ama_{n}\leq a_{m} for all n≥mn\geq m.

  • •
    ​

    It follows that an increasing sequence is automatically bounded below (by a1a_{1}) and a decreasing sequence is automatically bounded above (again, by a1a_{1}).

  • •
    ​

    Note that it’s possible for a sequence to be both increasing and decreasing! But only if it’s constant.

  • •
    ​

    We sometimes refer to a sequence as strictly increasing, meaning an+1>ana_{n+1}>a_{n} for all n∈ℤ+n\in{\mathbb{Z}}^{+}.

We have proved that all convergent sequences are bounded, and observed that not all bounded sequences are convergent (an=(−1)na_{n}=(-1)^{n} provides a counterexample). We next prove that all monotone bounded sequences are convergent. This is the first limit theorem in which we make direct use of the Axiom of Completeness.

Theorem 3.18

Let (an)(a_{n}) be increasing and bounded above. Then (an)(a_{n}) converges to

L:=sup{an:n∈ℤ+}.L:=\sup\{a_{n}\>:\>n\in{\mathbb{Z}}^{+}\}.
Proof.

The set A={an:n∈ℤ+}A=\{a_{n}\>:\>n\in{\mathbb{Z}}^{+}\} is nonempty and bounded above, and hence, by Axiom 3.11, has a supremum, L∈ℝL\in{\mathbb{R}}. We claim that an→La_{n}\rightarrow L.

Let ε>0\varepsilon>0. Then L−ε<LL-\varepsilon<L, so by the definition of supremum, L−εL-\varepsilon is not an upper bound on AA. Hence there exists N∈ℤ+N\in{\mathbb{Z}}^{+} such that aN>L−εa_{N}>L-\varepsilon. But (an)(a_{n}) is increasing, so for all n≥Nn\geq N, an≥aN>L−εa_{n}\geq a_{N}>L-\varepsilon. Further, LL is an upper bound on AA, so for all n∈ℤ+n\in{\mathbb{Z}}^{+}, an<L<L+εa_{n}<L<L+\varepsilon. Putting these two inequalities together, we see that for all n≥Nn\geq N

−ε<an−L<ε-\varepsilon<a_{n}-L<\varepsilon

and hence |an−L|<ε|a_{n}-L|<\varepsilon. ∎

Corollary 3.20 (Monotone Convergence Theorem)

Every bounded monotone sequence converges.

Example 3.21

Summary

  • •
    ​

    An upper bound on a set A⊂ℝA\subset{\mathbb{R}} is a number K∈ℝK\in{\mathbb{R}} such that x≤Kx\leq K for all x∈Ax\in A. A lower bound on AA is L∈ℝL\in{\mathbb{R}} such that x≥Lx\geq L for all x∈Ax\in A. AA is bounded above if it has an upper bound, bounded below if it has a lower bound, and bounded if it has both.

  • •
    ​

    The supremum of AA is its least upper bound. That is, K=supAK=\sup A if (i) KK is an upper bound on AA and (ii) no L<KL<K is an upper bound on AA.

  • •
    ​

    The infimum of AA is its greatest lower bound. That is, K=infAK=\inf A if (i) KK is a lower bound on AA and (ii) no L>KL>K is a lower bound on AA.

  • •
    ​

    ℝ{\mathbb{R}} is, by definition, an ordered field which additionally satisfies the Axiom of Completeness: every nonempty subset of ℝ{\mathbb{R}} which is bounded above has a supremum in ℝ{\mathbb{R}}.

  • •
    ​

    It follows from this definition that:

    • –
      ​

      ℝ{\mathbb{R}} has the Archimedean Property: given any K∈ℝK\in{\mathbb{R}}, there exists n∈ℤ+n\in{\mathbb{Z}}^{+} such that n>Kn>K.

    • –
      ​

      ℚ{\mathbb{Q}} is dense in ℝ{\mathbb{R}}: between any pair of distinct real numbers, there is a rational number.

    • –
      ​

      ℝ\ℚ{\mathbb{R}}{\backslash}{\mathbb{Q}} is dense in ℝ{\mathbb{R}}: between any pair of distinct real numbers, there is an irrational number.

  • •
    ​

    A sequence is increasing if an+1≥ana_{n+1}\geq a_{n} for all n∈ℤ+n\in\mathbb{Z}^{+}. A sequence is decreasing if an+1≤ana_{n+1}\leq a_{n} for all n∈ℤ+n\in\mathbb{Z}^{+}. A sequence is monotone if it is either increasing or decreasing.

  • •
    ​

    Monotone convergence theorem states that any bounded monotone sequence converges.
