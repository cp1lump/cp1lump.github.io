7 Limits of functions

7.1 Cluster points and limits of functions

We now have a very precise (and powerful) definition of limit for real sequences. In this section we will show how to use this to give a precise definition of the limit of a real function of a real variable. We will also see that basic properties of limits of functions, which are usually asserted without proof in introductory calculus courses, follow very easily from the corresponding limit theorems for sequences, which we proved in section 2.2.

Informally, limx→af⁢(x)=L\lim_{x\rightarrow a}f(x)=L means that f⁢(x)f(x) is “arbitrarily close” to LL for all xx “sufficiently close to, but different from” aa. So limits concern the behaviour of xx arbitrarily close to aa, but not at aa. There is no reason why aa has to be in the domain of ff, that is, f⁢(a)f(a) may, or may not, be defined. If it is, its value should be irrelevant to limx→af⁢(x)\lim_{x\rightarrow a}f(x). For example,

f⁢(x)=sin⁡xxf(x)=\frac{\sin x}{x}

is clearly undefined at 0 (its domain is (−∞,0)∪(0,∞)(-\infty,0)\cup(0,\infty)), but limx→0f⁢(x)\lim_{x\rightarrow 0}f(x) is well defined (it’s 11).

In order to probe the behaviour of f⁢(x)f(x) for values of xx close to, but different from, aa, we can consider the sequence f⁢(xn)f(x_{n}) where (xn)(x_{n}) is any sequence converging to aa with xn≠ax_{n}\neq a. From now on, given a set D⊆ℝD\subseteq\mathbb{R} we will say that a sequence ana_{n} is in DD if for every n∈ℤ+n\in\mathbb{Z}^{+}, an∈Da_{n}\in D.

This leads us to the main definition of this chapter:

Definition 7.1

Let D⊆ℝD\subseteq{\mathbb{R}} and f:D→ℝf:D\rightarrow{\mathbb{R}}. Then ff has limit LL at aa if, for all sequences (xn)(x_{n}) in D\{a}D{\backslash}\{a\} such that xn→ax_{n}\rightarrow a, f⁢(xn)→Lf(x_{n})\rightarrow L. In this case, we usually denote the number LL by limx→af⁢(x)\lim_{x\rightarrow a}f(x).

It’s important to note that the definition says that for all sequences (xn)(x_{n}) in D\{a}D{\backslash}\{a\} converging to aa, f⁢(xn)→Lf(x_{n})\rightarrow L. It’s not enough to show that f⁢(xn)→Lf(x_{n})\rightarrow L for just one specific sequence. Note that the sequence (xn)(x_{n}) gets arbitrarily close to aa (since xn→ax_{n}\rightarrow a), but never equals aa (since it takes values in D\{a}D{\backslash}\{a\}). So the value of ff at aa, if this exists, is irrelevant.

In order for Definition 7.1 to make sense, there must exist at least one sequence in D\{a}D{\backslash}\{a\} such that xn→ax_{n}\rightarrow a. For example, the function

f⁢(x)=1−x2f(x)=\sqrt{1-x^{2}}

has maximal domain D=[−1,1]D=[-1,1], so it doesn’t make sense to try to define limx→2f⁢(x)\lim_{x\rightarrow 2}f(x), its limit at 22, say, because there don’t exist any sequences in D\{2}=[−1,1]D{\backslash}\{2\}=[-1,1] which converge to 22.

To be precise, we should include this requirement in our definition of limit. This leads us to the following definition.

Definition 7.2

Let D⊂ℝD\subset{\mathbb{R}} and a∈ℝa\in{\mathbb{R}}. We say that aa is a cluster point of DD if there exists a sequence (xn)(x_{n}) in D\{a}D{\backslash}\{a\} such that xn→ax_{n}\rightarrow a.

Example 7.3

Claim: Every a∈ℝa\in{\mathbb{R}} is a cluster point of ℚ{\mathbb{Q}}.

Proof.

Choose any a∈ℝa\in{\mathbb{R}}. By Theorem 3.15, for each n∈ℤ+n\in{\mathbb{Z}}^{+} there exists a rational number xnx_{n} between aa and a+1/na+1/n. Now xn∈ℚ\{a}x_{n}\in{\mathbb{Q}}{\backslash}\{a\} and xn→ax_{n}\rightarrow a by the Squeeze Rule. Hence aa is a cluster point of ℚ{\mathbb{Q}}. ∎

Example 7.4

Claim: The set of cluster points of (0,∞)(0,\infty) is [0,∞)[0,\infty).

Proof.

Let a∈[0,∞)a\in[0,\infty). Then, for each n∈ℤ+n\in{\mathbb{Z}}^{+}, xn=a+1nx_{n}=a+\frac{1}{n} is different from aa and lies in (0,∞)(0,\infty). Clearly xn→ax_{n}\rightarrow a. Hence every a∈[0,∞)a\in[0,\infty) is a cluster point of (0,∞)(0,\infty), that is, the set of cluster points of (0,∞)(0,\infty) contains [0,∞)[0,\infty). Conversely, assume a∉[0,∞)a\notin[0,\infty). Then a<0a<0. Let (xn)(x_{n}) be any sequence in (0,∞)(0,\infty). If (xn)(x_{n}) converges at all, its limit must be non-negative, since xn>0x_{n}>0 for all nn (Proposition 2.14). Hence xn↛ax_{n}\nrightarrow a. Since no sequence in (0,∞)(0,\infty) converges to aa, aa is not a cluster point of (0,∞)(0,\infty). ∎

Many authors call cluster points limit points, or accumulation points. It makes sense to ask whether f:D→ℝf:D\rightarrow{\mathbb{R}} has a limit at aa precisely when aa is a cluster point of DD. With this is mind, we should really tighten up Definition 7.1.

Improved Definition 7.1 Let D⊆ℝD\subseteq{\mathbb{R}}, f:D→ℝf:D\rightarrow{\mathbb{R}} and aa be a cluster point of DD. Then ff has limit LL at aa if, for all sequences (xn)(x_{n}) in D\{a}D{\backslash}\{a\} such that xn→ax_{n}\rightarrow a, f⁢(xn)→Lf(x_{n})\rightarrow L. In this case, we usually denote the number LL by limx→af⁢(x)\lim_{x\rightarrow a}f(x).

An important fact about limits is that, if they exist, they’re unique:

Proposition 7.5

If limx→af⁢(x)\lim_{x\rightarrow a}f(x) exists, then it is unique.

Proof.

Let f:D→ℝf:D\rightarrow{\mathbb{R}} and assume, to the contrary, that KK and LL are non-equal real numbers satisfying the definition of limx→af⁢(x)\lim_{x\rightarrow a}f(x). Let (xn)(x_{n}) be a sequence in D\{a}D{\backslash}\{a\} such that xn→ax_{n}\rightarrow a. Then f⁢(xn)→Kf(x_{n})\rightarrow K and f⁢(xn)→Lf(x_{n})\rightarrow L and f⁢(xn)→L≠Kf(x_{n})\rightarrow L\neq K. But this contradicts Proposition 2.11. ∎

So it makes sense to speak of the limit of a function at aa.

Let f:ℝ\{1}→ℝf:{\mathbb{R}}{\backslash}\{1\}\rightarrow{\mathbb{R}} such that

Let f:ℝ\{0}→ℝf:{\mathbb{R}}{\backslash}\{0\}\rightarrow{\mathbb{R}} such that

f⁢(x)=x|x|.f(x)=\frac{x}{|x|}.

Let f:ℝ\{0}→ℝf:{\mathbb{R}}{\backslash}\{0\}\rightarrow{\mathbb{R}} such that

There is also an alternative (and more commonly used) definition of limits. This is the so-called ε\varepsilon–δ\delta criterion for limits:

Theorem 7.9

Let aa be a cluster point of D⊆ℝD\subseteq{\mathbb{R}} and f:D→ℝf:D\rightarrow{\mathbb{R}}. Then limx→af⁢(x)=L\lim_{x\rightarrow a}f(x)=L if and only if, for each ε>0\varepsilon>0, there exists δ>0\delta>0 such that, for all x∈Dx\in D such that 0<|x−a|<δ0<|x-a|<\delta, |f⁢(x)−L|<ε|f(x)-L|<\varepsilon.

Proof.

(⇐\Leftarrow, the “if” direction): Assume that f,a,Lf,a,L satisfy the ε\varepsilon–δ\delta criterion. We must show that limx→af⁢(x)=L\lim_{x\rightarrow a}f(x)=L. So, let (xn)(x_{n}) be any sequence in D\{a}D{\backslash}\{a\} such that xn→ax_{n}\rightarrow a. We must show that f⁢(xn)→Lf(x_{n})\rightarrow L. Let ε∈(0,∞)\varepsilon\in(0,\infty) be given. Then, by the ε\varepsilon–δ\delta criterion, there exists δ∈(0,∞)\delta\in(0,\infty) such that, for all x∈(D\{a})∩(a−δ,a+δ)x\in(D{\backslash}\{a\})\cap(a-\delta,a+\delta), |f⁢(x)−L|<ε|f(x)-L|<\varepsilon. But xn→ax_{n}\rightarrow a so given any positive real number, for example, δ\delta, there exists N∈ℤ+N\in{\mathbb{Z}}^{+} such that, for all n≥Nn\geq N, xn∈(a−δ,a+δ)x_{n}\in(a-\delta,a+\delta). By definition, xn∈D\{a}x_{n}\in D{\backslash}\{a\}, so, for all n≥Nn\geq N, xn∈(D\{a})∩(a−δ,a+δ)x_{n}\in(D{\backslash}\{a\})\cap(a-\delta,a+\delta), whence |f⁢(xn)−L|<ε|f(x_{n})-L|<\varepsilon. Hence f⁢(xn)→Lf(x_{n})\rightarrow L.

(⇒\Rightarrow, the “only if” direction): We will prove the contrapositive, that is, we will show that if f,a,Lf,a,L do not satisfy the ε\varepsilon–δ\delta criterion, then limx→af⁢(x)≠L\lim_{x\rightarrow a}f(x)\neq L, that is, there exists a sequence (xn)(x_{n}) in D\{a}D{\backslash}\{a\} converging to aa whose image under ff, f⁢(xn)f(x_{n}) does not converge to LL.

Our first task is to negate the ε\varepsilon–δ\delta criterion:

¬[∀ε∈(0,∞),∃δ∈(0,∞),∀x∈(D\{a})∩(a−δ,a+δ),|f⁢(x)−L|<ε]\displaystyle\neg[\forall\varepsilon\in(0,\infty),\exists\delta\in(0,\infty),% \forall x\in(D{\backslash}\{a\})\cap(a-\delta,a+\delta),|f(x)-L|<\varepsilon]
⇔\displaystyle\Leftrightarrow ∃ε∈(0,∞)⁢¬[∃δ∈(0,∞),∀x∈(D\{a})∩(a−δ,a+δ),|f⁢(x)−L|<ε]\displaystyle\exists\varepsilon\in(0,\infty)\neg[\exists\delta\in(0,\infty),% \forall x\in(D{\backslash}\{a\})\cap(a-\delta,a+\delta),|f(x)-L|<\varepsilon]
⇔\displaystyle\Leftrightarrow ∃ε∈(0,∞),∀δ∈(0,∞)⁢¬[∀x∈(D\{a})∩(a−δ,a+δ),|f⁢(x)−L|<ε]\displaystyle\exists\varepsilon\in(0,\infty),\forall\delta\in(0,\infty)\neg[% \forall x\in(D{\backslash}\{a\})\cap(a-\delta,a+\delta),|f(x)-L|<\varepsilon]
⇔\displaystyle\Leftrightarrow ∃ε∈(0,∞),∀δ∈(0,∞),∃x∈(D\{a})∩(a−δ,a+δ)⁢¬[|f⁢(x)−L|<ε]\displaystyle\exists\varepsilon\in(0,\infty),\forall\delta\in(0,\infty),% \exists x\in(D{\backslash}\{a\})\cap(a-\delta,a+\delta)\neg[|f(x)-L|<\varepsilon]
⇔\displaystyle\Leftrightarrow ∃ε∈(0,∞),∀δ∈(0,∞),∃x∈(D\{a})∩(a−δ,a+δ),|f⁢(x)−L|≥ε.\displaystyle\exists\varepsilon\in(0,\infty),\forall\delta\in(0,\infty),% \exists x\in(D{\backslash}\{a\})\cap(a-\delta,a+\delta),|f(x)-L|\geq\varepsilon.

So, we assume that there exists some positive real number ε\varepsilon such that, for all δ>0\delta>0 there is some x∈(D\{a})∩(a−δ,a+δ)x\in(D{\backslash}\{a\})\cap(a-\delta,a+\delta) such that |f⁢(x)−L|≥ε|f(x)-L|\geq\varepsilon. This holds for every δ>0\delta>0. In particular, it holds for δ=1/n\delta=1/n where n∈ℤ+n\in{\mathbb{Z}}^{+}: for each n∈ℤ+n\in{\mathbb{Z}}^{+} there exists xn∈(D\{a})∩(a−1/n,a+1/n)x_{n}\in(D{\backslash}\{a\})\cap(a-1/n,a+1/n) such that |f⁢(xn)−L|≥ε|f(x_{n})-L|\geq\varepsilon. Consider the sequence (xn)(x_{n}). By definition, it lies in D\{a}D{\backslash}\{a\} and

a−1n<xn<a+1n,a-\frac{1}{n}<x_{n}<a+\frac{1}{n},

so xn→ax_{n}\rightarrow a by the Squeeze Rule. But |f⁢(xn)−L|≥ε|f(x_{n})-L|\geq\varepsilon for all nn, so certainly |f⁢(xn)−L|↛0|f(x_{n})-L|\nrightarrow 0 (this would contradict Proposition 2.14), and so f⁢(xn)↛Lf(x_{n})\nrightarrow L.

In summary, we have shown that if f,a,Lf,a,L do not satisfy the ε\varepsilon–δ\delta criterion, then we can construct a sequence xn∈D\{a}x_{n}\in D{\backslash}\{a\} which converges to aa such that f⁢(xn)f(x_{n}) does not converge to LL. It follows, in this case, that ff does not have limit LL at aa. ∎

Many (in fact, almost all) textbooks on Real Analysis define limits using the ε\varepsilon–δ\delta criterion, rather than sequences. The Theorem above assures us that this approach is equivalent to ours. One reason to prefer our sequential approach to limits is that, having defined limits purely in terms of convergence of sequences, it is very easy to establish many of their basic properties, since these follow directly from theorems about sequences which we’ve already proved. We’ve already seen an example of this: limits of functions are unique (Proposition 7.5) because limits of convergent sequences are unique (Proposition 2.11). Another very useful example is the Algebra of Limits for functions:

Theorem 7.10 (The Algebra of Limits for functions)

.
Let f:D→ℝf:D\rightarrow{\mathbb{R}}, g:D→ℝg:D\rightarrow{\mathbb{R}}, limx→af⁢(x)=L\lim_{x\rightarrow a}f(x)=L and limx→ag⁢(x)=K\lim_{x\rightarrow a}g(x)=K. Then

  • (i)
    ​

    limx→a(f⁢(x)+g⁢(x))=L+K\displaystyle{\lim_{x\rightarrow a}(f(x)+g(x))=L+K},

  • (ii)
    ​

    limx→af⁢(x)⁢g⁢(x)=L⁢K\displaystyle{\lim_{x\rightarrow a}f(x)g(x)=LK},

and if, in addition, f⁢(x)≠0f(x)\neq 0 for all x∈D\{a}x\in D{\backslash}\{a\} and L≠0L\neq 0, then

  • (iii)
    ​

    limx→a1f⁢(x)=1L\displaystyle{\lim_{x\rightarrow a}\frac{1}{f(x)}=\frac{1}{L}}.

Proof.

Let (xn)(x_{n}) be any sequence in D\{a}D{\backslash}\{a\} such that xn→ax_{n}\rightarrow a. Then f⁢(xn)→Lf(x_{n})\rightarrow L and g⁢(xn)→Kg(x_{n})\rightarrow K by assumption. Hence

  1. (i)
    ​

    f⁢(xn)+g⁢(xn)→L+Kf(x_{n})+g(x_{n})\rightarrow L+K by Theorem 2.15(i).

  2. (ii)
    ​

    f⁢(xn)⁢g⁢(xn)→L⁢Kf(x_{n})g(x_{n})\rightarrow LK by Theorem 2.15(ii).

  3. (iii)
    ​

    f⁢(xn)≠0f(x_{n})\neq 0 and L≠0L\neq 0 so 1/f⁢(xn)→1/L1/f(x_{n})\rightarrow 1/L by Theorem 2.15(iii).

∎

Note that the theorem follows almost immediately from Theorem 2.15, once we’ve defined limits of functions in the right way (Definition 7.1).

Hey! You now know what limits are!

It’s worthwhile to pause and reflect on what we’ve achieved in the first semester. We now have a completely precise and rigorous definition of limx→af⁢(x)\lim_{x\rightarrow a}f(x) for functions f:D→ℝf:D\rightarrow{\mathbb{R}}, where DD is a subset of ℝ{\mathbb{R}}. It follows, that we now know, with no hand-waving or weasel words, precisely what the limit

f′⁢(a)=limx→af⁢(x)−f⁢(a)x−af^{\prime}(a)=\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}

means, that is, we have put the definition of the derivative of a function on a proper mathematical footing. Note that this is not a limit of f:D→ℝf:D\rightarrow{\mathbb{R}}, but of the associated function

h:D\{a}→ℝ,h⁢(x)=f⁢(x)−f⁢(a)x−a.h:D{\backslash}\{a\}\rightarrow{\mathbb{R}},\qquad h(x)=\frac{f(x)-f(a)}{x-a}.

The interesting, and somewhat curious, thing about this is that we managed to do this by studying sequences and, in particular, making precise what it means for sequences to converge. So from the rather modest and unassuming definition of convergence of sequences (Definition 2.4) something very large and powerful has grown.
