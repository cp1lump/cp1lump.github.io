6 Some symbolic logic

6.1 Statements and their symbolic manipulation

In this module we have dealt with many mathematical statements, some of which were quite logically subtle. For example, consider the statement

“The real sequence (an)(a_{n}) converges to LL.”

We learned that this statement really means

“For each positive real number ε\varepsilon there exists a positive integer NN such that, for all n≥Nn\geq N, |an−L|<ε|a_{n}-L|<\varepsilon.”

What does the statement

“The real sequence (an)(a_{n}) converges”

mean? It means that (an)(a_{n}) converges to LL for some real number LL. That is,

“There exists a real number LL such that, for each positive number ε\varepsilon, there exists a positive integer NN such that, for all n≥Nn\geq N, |an−L|<ε|a_{n}-L|<\varepsilon.”

Recall that a sequence diverges if it doesn’t converge. So what does the statement

“The real sequence (an)(a_{n}) diverges”

really mean? We will see that it means precisely

“For each real number LL there exists a positive real number ε\varepsilon such that for all positive integers NN there exists n≥Nn\geq N such that |an−L|≥ε|a_{n}-L|\geq\varepsilon.”

This is, to put it mildly, not obvious! In this section we will introduce some basic ideas from symbolic logic which will allow us to manipulate and analyze complicated mathematical statements like “(an)(a_{n}) converges,” and obtain from them related statements like “(an)(a_{n}) diverges.”

The key idea is to represent mathematical statements by symbols which can be combined and algebraically manipulated. For our purposes, a statement is any declaration which is unambiguously either true or false. For example, the declarations

PP: (−2)2=4\displaystyle{(-2)^{2}=4}
QQ: Leeds is the third largest city in the UK by population
RR: −2>2\displaystyle{-2>2}
SS: Every even integer greater than 2 can be expressed
as the sum of two primes

are all statements: PP and QQ are true, RR is false, and no-one knows (at the time of writing) whether SS is true or false, but it is clearly one or the other55 5 This is the infamous Goldbach Conjecture.. On the other hand, declarations like

P′P^{\prime}: Tough on crime, tough on the causes of crime!
Q′Q^{\prime}: Lucy in the sky with diamonds

are not statements because neither can be meaningfully said to be true, or false.

Given one or more statements P,Q,…P,Q,\ldots, we can produce new statements from them by three basic constructions:

  • •
    ​

    Negation: not PP, denoted ¬P\neg P, is the statement which is true if and only if PP is false. For the examples above, we have

    ¬P\neg P: (−2)2≠4\displaystyle{(-2)^{2}\neq 4} False
    ¬Q\neg Q: Leeds is not the third largest city in the UK by population False
    ¬R\neg R −2≤2\displaystyle{-2\leq 2} True
  • •
    ​

    Conjunction: PP and QQ, denoted P∧QP\wedge Q, is the statement which is true if both PP and QQ are true, and false otherwise. So, for the examples above, P∧QP\wedge Q is true, but P∧RP\wedge R is false (because RR is false).

  • •
    ​

    Disjunction: PP or QQ, denoted P∨QP\vee Q, is the statement which is true if PP is true, or QQ is true, or both, but is false if both PP and QQ are false. So, for the examples above, P∨QP\vee Q is true, as is P∨RP\vee R (because PP is true). However, (¬P)∨R(\neg P)\vee R is false (because both ¬P\neg P and RR are false).

You should bear in mind that in mathematics we use the word “or” in its non-exclusive sense. That is “PP or QQ” means “PP or QQ (or both),” which is different from its use in everyday English. For example, a child who is told

“Finish your broccoli or you can’t have any ice cream”

could reasonably expect to get some ice cream if he/she/they finishes his/her/their broccoli. If the nagging adult is a mathematician, the child would be wise to seek clarification.

6.2 Implications

It is convenient to define another connective between two statements P,QP,Q in addition to conjunction (P∧QP\wedge Q) and disjunction (P∨QP\vee Q), namely implication, denoted P⇒QP\Rightarrow Q, pronounced “if PP then QQ”, or “PP implies QQ”. (Actually, the standard logic notation is P→QP\rightarrow Q, and P⇒QP\Rightarrow Q is reserved for something subtly different. However, we are already making heavy use of the symbol →\rightarrow to denote convergence of sequences, so we will stick to ⇒\Rightarrow for implication.) The truth table for this statement is

PP QQ P⇒QP\Rightarrow Q
0 0 1
0 1 1
1 0 0
1 1 1

As an example, consider the statement

“If everyone in this class passes the final exam, I will eat my hat.”

This is clearly false if everyone passes the exam, and I refuse to eat my hat. On the other hand, it’s clearly true if everyone passes and I do eat my hat. What if not everyone passes? Then the question of whether I eat my hat becomes moot: the statement is true if I don’t, but it’s also true if I do (I never said I’d eat my hat only if everyone passes - maybe I just like eating hats!). So P⇒QP\Rightarrow Q is false only when PP is true and QQ is false. In every other case, it is true.

Unlike P∧QP\wedge Q and P∨QP\vee Q, the order of the statements P,QP,Q in P⇒QP\Rightarrow Q matters, as can be seen immediately from its truth table. For example, the statement

“If you are vegetarian then you don’t eat pork”

is true, whereas

“If you don’t eat pork then you are vegetarian”

is false (except in Spain). Given an implication P⇒QP\Rightarrow Q, the implication Q⇒PQ\Rightarrow P is called its converse. As we have just seen, an implication and its converse are not logically equivalent. We have seen many examples of this fact in this module. For example

“If a real sequence converges then it is bounded”

is true (see Proposition 2.13), but its converse

“If a real sequence is bounded then it converges”

is false (e.g. an=(−1)na_{n}=(-1)^{n}). Similarly

“If a real sequence is bounded then it has a convergent subsequence”

is true (this is the Bolzano-Weierstrass Theorem), but its converse

“If a real sequence has a convergent subsequence then it is bounded”

is false (e.g. an=(1+(−1)n)⁢na_{n}=(1+(-1)^{n})n). Another important, and often misused, example is

“If ∑n=1∞an\sum_{n=1}^{\infty}a_{n} converges then an→0a_{n}\rightarrow 0”

which is true (this is the Divergence Test), but its converse

“If an→0a_{n}\rightarrow 0 then ∑n=0∞an\sum_{n=0}^{\infty}a_{n} converges”

is false (e.g. ∑n=1∞1/n\sum_{n=1}^{\infty}1/n).

It is possible to rewrite P⇒QP\Rightarrow Q using only ¬\neg and ∨\vee, and this is often useful when analyzing the precise meaning of an implication. In fact

(6.8) (6.8) [P⇒Q]⇔[(¬P)∨Q].[P\Rightarrow Q]\quad\iff\quad[(\neg P)\vee Q].

We can check this by computing the truth table of the right hand side:

PP QQ ¬P\neg P (¬P)∨Q(\neg P)\vee Q
0 0 1 1
0 1 1 1
1 0 0 0
1 1 0 1

From this, we deduce that

[P⇒Q]\displaystyle[P\Rightarrow Q] ⇔\displaystyle\iff [(¬P)∨Q]\displaystyle[(\neg P)\vee Q]
⇔\displaystyle\iff [Q∨(¬P)]\displaystyle[Q\vee(\neg P)]
⇔\displaystyle\iff [(¬(¬Q))∨(¬P)]by (6.5)\displaystyle[(\neg(\neg Q))\vee(\neg P)]\qquad\mbox{by (\ref{plsk})}
⇔\displaystyle\iff [(¬Q)⇒(¬P)]by (6.8)\displaystyle[(\neg Q)\Rightarrow(\neg P)]\qquad\mbox{by (\ref{scd})}

So, for any pair of statements P,QP,Q,

[P⇒Q]⇔[(¬Q)⇒(¬P)].[P\Rightarrow Q]\quad\iff\quad[(\neg Q)\Rightarrow(\neg P)].

Given an implication P⇒QP\Rightarrow Q, the implication (¬Q)⇒(¬P)(\neg Q)\Rightarrow(\neg P) is called its contrapositive. We have just shown that an implication and its contrapositive are logically equivalent. Looking back at the above examples, we see that Proposition 2.13 could equally well have been stated

“If a real sequence is unbounded then it does not converge,”

and the Bolzano-Weierstrass Theorem is equivalent to

“If a real sequence has no convergent subsequence then it is unbounded,”

and the Divergence Test is equivalent to

“If ana_{n} does not converge to 0 then ∑n=1∞an\sum_{n=1}^{\infty}a_{n} diverges,”

which is, in fact, the form in which one actually uses it.

If you are asked to prove an implication, you should always consider whether proving its contrapositive will be easier. Since the contrapositive is logically equivalent to the original implication, this will suffice.

Example 6.5

Claim: Let nn be an integer. If n2n^{2} is divisible by 3 then nn is divisible by 3.

Proof.

We will prove the contrapositive: if nn is not divisible by 3 then n2n^{2} is not divisible by 3.  So, assume that nn is not divisible by 3. Then its remainder on division by 3 is either 1 or 2, that is, either n=3⁢k+1n=3k+1 or n=3⁢k+2n=3k+2, where kk is an integer. In the first case

n2=9⁢k2+6⁢k+1=3⁢k′+1n^{2}=9k^{2}+6k+1=3k^{\prime}+1

where k′=3⁢k2+2⁢k∈ℤk^{\prime}=3k^{2}+2k\in{\mathbb{Z}}, and in the second case

n2=9⁢k2+12⁢k+4=3⁢k′+1n^{2}=9k^{2}+12k+4=3k^{\prime}+1

where k′=3⁢k2+4⁢k+1∈ℤk^{\prime}=3k^{2}+4k+1\in{\mathbb{Z}}. In either case, we see that n2n^{2} is not divisible by 3 (in fact, its remainder is 11 in both cases), as was to be proved. ∎

Another important fact follows immediately from (6.8): the negation of an implication is NOT and implication!

¬[P⇒Q]⇔¬[Q∨¬P]⇔(¬Q)∧(¬¬P)⇔P∧¬Q.\neg[P\Rightarrow Q]\quad\iff\quad\neg[Q\vee\neg P]\quad\iff\quad(\neg Q)% \wedge(\neg\neg P)\quad\iff\quad P\wedge\neg Q.

Note that this is entirely different from (¬P)⇒(¬Q)(\neg P)\Rightarrow(\neg Q). For example, the negation of the statement “if nn is prime then nn is odd” is the statement “nn is prime and nn is not odd”, not the statement “if nn is not prime then nn is not odd.”

6.3 Quantifiers

We have been a little bit sloppy up to now in dealing with statements which contain variables. Consider, for example, the statement

x>2.x>2.

Is this really a statement? Until one specifies a particular value for xx, one cannot say whether it is true. So it’s true for x=3x=3, and false for x=−2x=-2, for example. What about if x=1+3⁢ix=1+3i? The “statement” 1+3⁢i>21+3i>2 is pretty meaningless, and it isn’t very helpful to declare it either true or false (e.g. if you decide it’s false, then you’d expect 1+3⁢i≤21+3i\leq 2 to be true, and this “statement” is equally meaningless). If we’re being careful, we should make explicit the condition that xx is a real number, and if we denote the statement x>2x>2 with a symbol, we should choose one like P⁢(x)P(x), rather than PP, to remind ourselves that whether P⁢(x)P(x) is true depends on the particular xx under consideration. So P⁢(3)P(3) is true, in this case, while P⁢(2)P(2) is false.

There are two very useful ways to turn a statement with a variable into an unambiguous true-or-false statement, using quantifiers. Let P⁢(x)P(x) be any statement which makes sense for all values of xx in some specified set AA (e.g. A=ℝA={\mathbb{R}} and P⁢(x):x>2P(x):x>2, as above). Then

∀x⁢P⁢(x)\forall x\,P(x) means “for all x∈Ax\in A, P⁢(x)P(x) is true”

and

∃x⁢P⁢(x)\exists x\,P(x) means “there exists some x∈Ax\in A such that P⁢(x)P(x) is true”.

The symbol ∀\forall is called the universal quantifier and the symbol ∃\exists is called the existential quantifier. For the example above, ∀x⁢P⁢(x)\forall x\,P(x) is false, because, for example, P⁢(−1)P(-1) is false (since −1-1 is not greater than 22), but ∃x⁢P⁢(x)\exists x\,P(x) is true because, for example, P⁢(3)P(3) is true. Note that, once we’ve coupled it with a quantifier, the statement no longer contains a variable, and is a simple true or false assertion. Note also that it’s entirely irrelevant what we call the variable. So ∀y⁢P⁢(y)\forall y\,P(y) is precisely the same statement as ∀x⁢P⁢(x)\forall x\,P(x).

In practice, rather than specifying the set AA in advance, we often specify it explicitly with the quantifier. In the above example, we would write

∀x∈ℝ,x>2\forall x\in{\mathbb{R}},\,x>2

which is false, and

(6.9) (6.9) ∃x∈ℝ,x>2\exists x\in{\mathbb{R}},\,x>2

which is true. You should feel free to add punctuation and/or ordinary English to expressions like this, if you think it clarifies the meaning. For example, I think

∃x∈ℝ⁢ such that ⁢x>2\exists x\in{\mathbb{R}}\mbox{ such that }x>2

is clearer than (6.9). Indeed, you should feel free to avoid using the symbols ∀\forall and ∃\exists entirely if you think they obscure the mathematical meaning of the sentences you are writing. As we will see, they can be extremely useful, but you shouldn’t chuck them into your mathematical prose willy-nilly. They won’t make your proofs any more rigorous, elegant or impressive.

The situation in which quantifiers really come into their own is when you want to understand the negation of a statement involving several quantifiers. Let’s ask ourselves, what does it mean to say that ∀x⁢P⁢(x)\forall x\,P(x) is false? It means that it is not true that for all xx in AA, P⁢(x)P(x) is true. That is, there is at least one element xx in AA for which P⁢(x)P(x) is not true. In other words, there exists x∈Ax\in A such that ¬P⁢(x)\neg P(x) is true. That is

(6.10) (6.10) ¬[∀x⁢P⁢(x)]⇔∃x⁢¬P⁢(x).\neg[\forall x\,P(x)]\qquad\iff\qquad\exists x\,\neg P(x).
Example 6.9

Consider the assertion that “all politicians are corrupt.” I think this is false. In saying that, I am certainly not asserting that “all politicians are not corrupt” (which is equivalent to “no politician is corrupt”). Rather, I am asserting that “there exists at least one politician who is not corrupt.” In symbols, let XX denote the set of politicians and P⁢(x)P(x) be the statement “xx is corrupt.” Then the negation of ∀x∈X⁢P⁢(x)\forall x\in X\,P(x) is not ∀x∈X⁢¬P⁢(x)\forall x\in X\,\neg P(x), but rather ∃x∈X⁢¬P⁢(x)\exists x\in X\,\neg P(x).

Note that to prove that [∀x∈A⁢P⁢(x)][\forall x\in A\,\,P(x)] is false, we need to show that [∃x∈A⁢¬P⁢(x)][\exists x\in A\,\,\neg P(x)] is true, that is, we need to show that there exists (at least) one element xx in AA such that P⁢(x)P(x) is false. Such an element xx is called a counterexample to [∀x∈A⁢P⁢(x)][\forall x\in A\,\,P(x)].

Example 6.10

Consider the statement ∀x∈ℝ⁢x2<10\forall x\in{\mathbb{R}}\,\,x^{2}<10. This is clearly false. To prove that it is false, all we need to do is exhibit a single real number whose square is not less than 1010. For example, x=10x=10 will do (since 102≥1010^{2}\geq 10). To prove the statement is false we do not have to find all real numbers xx such that x2≥10x^{2}\geq 10. Giving a single counterexample is quicker, clearer and more elegant. □\Box

The rule for negating ∃x⁢P⁢(x)\exists x\,P(x) actually follows from (6.10) and rule (6.5), but let’s derive it by thinking. What does it mean to say that ∃x⁢P⁢(x)\exists x\,P(x) is false? It means that there does not exist xx in AA such that P⁢(x)P(x) is true. Hence, for each and every x∈Ax\in A, P⁢(x)P(x) must be false. That is

¬[∃x⁢P⁢(x)]⇔∀x⁢¬P⁢(x).\neg[\exists x\,P(x)]\qquad\iff\qquad\forall x\,\neg P(x).
Example 6.11

Consider the assertion “someone in this room speaks fluent Mandarin.” Let’s assume this is false. What does this mean? It certainly does not merely mean “someone in this room does not speak fluent Mandarin.” Rather it means “no-one in this room speaks fluent Mandarin,” or, equivalently, “everyone in this room does not speak fluent Mandarin.” In symbols, let XX denote the set of people in this room and P⁢(x)P(x) be the statement “xx speaks fluent Mandarin.” Then the negation of ∃x∈X⁢P⁢(x)\exists x\in X\,P(x) is not ∃x∈X⁢¬P⁢(x)\exists x\in X\,\neg P(x), but rather ∀x∈X⁢¬P⁢(x)\forall x\in X\,\neg P(x). □\Box

You’ve already seen these negation rules in many different contexts. For example, consider the Archimedean Property of ℝ{\mathbb{R}}. Our first statement of this property was simply that “ℤ+{\mathbb{Z}}^{+} is unbounded above” (see Example 3.5). We then reformulated this as

“Given any real number KK, there is some positive integer nn such that n>Kn>K.”

We can now see symbolically that these two statements are logically equivalent:

ℤ+{\mathbb{Z}}^{+} is unbounded above ⇔\displaystyle\iff ¬[ℤ+ is bounded above]\displaystyle\neg\,[\mbox{${\mathbb{Z}}^{+}$ is bounded above}]\qquad\qquad\qquad\qquad

⇔\displaystyle\iff ∀K∈ℝ⁢∃n∈ℤ+⁢¬[n≤K]\displaystyle\forall K\in{\mathbb{R}}\,\,\exists n\in{\mathbb{Z}}^{+}\,\neg[n% \leq K]
⇔\displaystyle\iff ∀K∈ℝ⁢∃n∈ℤ+⁢n>K.\displaystyle\forall K\in{\mathbb{R}}\,\,\exists n\in{\mathbb{Z}}^{+}\,n>K.

In this case, using symbolic logic was really overkill: we could figure out the negation just by thinking carefully. A good example where symbolic logic really helps is the statement “(an)(a_{n}) does not converge to LL”. To illustrate, consider the following question:

Example 6.13

We consider three statements

S1S_{1}: “the sequence ana_{n} doesn’t converge to zero”,

We find this by negating the definition of “ana_{n} does converge to zero”

Translating back into English, an↛0a_{n}\nrightarrow 0 means

there exists ε>0\varepsilon>0 such that, for all N∈ℤ+N\in{\mathbb{Z}}^{+} there is some integer n≥Nn\geq N such that |an|≥ε|a_{n}|\geq\varepsilon.

So S1⇔S2S_{1}\iff S_{2}.

Example 6.14

Let (an)(a_{n}) be a real sequence with the property that every subsequence of (an)(a_{n}) has a subsequence which converges to 0. Show that (an)(a_{n}) converges to 0.

Proof.

We will prove this by contradiction. So, let (an)(a_{n}) be a sequence with the specified property, but assume that (an)(a_{n}) does not converge to 0. Then we know that there exists ε>0\varepsilon>0 and a subsequence bj=anjb_{j}=a_{n_{j}} of (an)(a_{n}) so that |bj|=|anj|>ϵ|b_{j}|=|a_{n_{j}}|>\epsilon. Then let ck=bjkc_{k}=b_{j_{k}} be a subsequence of (bj)(b_{j}) which converges to zero, which exists by our assumption. Hence there is an NN so that for all k≥Nk\geq N, |ck|<ϵ|c_{k}|<\epsilon. So, ϵ<|bjN|=|cN|<ϵ\epsilon<|b_{j_{N}}|=|c_{N}|<\epsilon. But this is false, so an↛0a_{n}\nrightarrow 0 must also be false. ∎

Remark This result is quite subtle. It does not say that

If every subsequence of (an)(a_{n}) converges to 0 then (an)(a_{n}) converges to 0

although this is trivially true, since (an)(a_{n}) is a subsequence of itself! On the other hand, by Theorem 4.3, Example 6.14 has the following corollary:

If every subsequence of (an)(a_{n}) has a subsequence which converges to 0, then every subsequence of (an)(a_{n}) converges to 0.

This is true despite the fact that, in general, having a subsequence which converges to 0 does not imply that a sequence converges to 0.

    Is this true? □\Box

Summary

  • •
    ​

    We can represent statements by symbols PP, QQ etc. We say that PP and QQ are logically equivalent if PP is true if and only if QQ is true, denoted P⇔QP\iff Q.

  • •
    ​

    We obtain new statements from existing ones using the following operations

    • –
      ​

      ¬P\neg P denotes the negation of PP, “not PP”, which is true if and only if PP is false

    • –
      ​

      P∧QP\wedge Q denotes the conjunction of PP and QQ, “PP and QQ”, which is true if and only if both PP and QQ are true

    • –
      ​

      P∨QP\vee Q denotes the disjunction of PP and QQ, “PP or QQ”, which is true if PP is true or QQ is true, or both.

  • •
    ​

    These operations obey algebraic rules:

    ¬(¬P)⇔P,¬(P∧Q)⇔(¬P)∨(¬Q),¬(P∨Q)⇔(¬P)∧(¬Q).\neg(\neg P)\iff P,\qquad\neg(P\wedge Q)\iff(\neg P)\vee(\neg Q),\qquad\neg(P% \vee Q)\iff(\neg P)\wedge(\neg Q).
  • •
    ​

    P⇒QP\Rightarrow Q denotes the implication “if PP then QQ”. By definition, this is logically equivalent to (¬P)∨Q(\neg P)\vee Q.

    • –
      ​

      The converse of P⇒QP\Rightarrow Q is Q⇒PQ\Rightarrow P.

    • –
      ​

      The contrapositive of P⇒QP\Rightarrow Q is (¬Q)⇒(¬P)(\neg Q)\Rightarrow(\neg P).

    An implication is logically equivalent to its contrapositive, but not to its converse.

  • •
    ​

    The symbols ∀\forall and ∃\exists are the universal and existential quantifiers.

    • –
      ​

      ∀x∈X⁢P⁢(x)\forall x\in X\,\,P(x) means “for all x∈Xx\in X, P⁢(x)P(x) is true”.

    • –
      ​

      ∃x∈X⁢P⁢(x)\exists x\in X\,\,P(x) means “there exists some x∈Xx\in X such that P⁢(x)P(x) is true”.

  • •
    ​

    These symbols behave under negation as follows

    ¬[∀x∈X⁢P⁢(x)]⇔∃x∈X⁢¬P⁢(x),¬[∃x∈X⁢P⁢(x)]⇔∀x∈X⁢¬P⁢(x).\neg[\forall x\in X\,\,P(x)]\iff\exists x\in X\,\,\neg P(x),\qquad\neg[\exists x% \in X\,\,P(x)]\iff\forall x\in X\,\,\neg P(x).
