The Interior Extremum Theorem

3.1 The Interior Extremum Theorem

Recall that f:D→ℝf:D\rightarrow{\mathbb{R}} attains a maximum at a∈Da\in D if f⁢(x)≤f⁢(a)f(x)\leq f(a) for all x∈Dx\in D. Similarly, ff attains a minimum at a∈Da\in D if f⁢(x)≥f⁢(a)f(x)\geq f(a) for all x∈Dx\in D. Note that these definitions have absolutely nothing to do with calculus! In fact, they make perfectly good sense whatever set DD, the domain of ff, is: it need not be a subset of ℝ{\mathbb{R}}!

We say that ff attains an extremum at a∈Da\in D if it attains either a maximum or minimum at aa. (The word “extremum” means “maximum or minimum”, in much the same way that “monotonic” means “increasing or decreasing”. Its plural is “extrema”.)

Theorem 3.1 (The Interior Extremum Theorem)

Let f:(a,b)→ℝf:(a,b)\rightarrow{\mathbb{R}} be differentiable and ff attain an extremum at c∈(a,b)c\in(a,b). Then f′⁢(c)=0f^{\prime}(c)=0.

Proof.

Consider first the case that ff attains a maximum at c∈(a,b)c\in(a,b). By assumption, f′⁢(c)f^{\prime}(c) exists. Hence, by Theorem 1.36, given any sequence (xn)(x_{n}) in (a,b)\{c}(a,b){\backslash}\{c\} which converges to cc,

s⁢(xn):=f⁢(xn)−f⁢(c)xn−c→f′⁢(c).s(x_{n}):=\frac{f(x_{n})-f(c)}{x_{n}-c}\rightarrow f^{\prime}(c). (3.1)

This holds, in particular, for the sequence xn=c+(b−c)/(n+1)∈(c,b)x_{n}=c+(b-c)/(n+1)\in(c,b). Note that xn>cx_{n}>c for all cc, so xn−c>0x_{n}-c>0, and f⁢(c)f(c) is the maximum value of ff, so f⁢(xn)−f⁢(c)≤0f(x_{n})-f(c)\leq 0. Hence s⁢(xn)≤0s(x_{n})\leq 0 for all nn so, by Proposition 1.7, its limit is non-positive, i.e. f′⁢(c)≤0f^{\prime}(c)\leq 0.

But (3.1) also holds in the case where xn=c−(c−a)/(n+1)∈(a,c)x_{n}=c{-(c-a)/(n+1)}\in(a,c). Now xn−c<0x_{n}-c<0 for all nn, and f⁢(xn)−f⁢(c)≤0f(x_{n})-f(c)\leq 0 (since, as before, ff attains its maximum at cc), so s⁢(xn)≥0s(x_{n})\geq 0 for all nn. Hence, by Proposition 1.7, its limit is non-negative, i.e. f′⁢(c)≥0f^{\prime}(c)\geq 0.

Hence, f′⁢(c)=0f^{\prime}(c)=0, as was to be shown.

Consider now the case that ff attains a minimum at c∈(a,b)c\in(a,b). Let g=−fg=-f. Then gg is differentiable on (a,b)(a,b) and attains a maximum at cc, so, as we just proved, g′⁢(c)=0g^{\prime}(c)=0. Hence f′⁢(c)=−g′⁢(c)=0f^{\prime}(c)=-g^{\prime}(c)=0. ∎

Definition 3.2

Let f:D→ℝf:D\rightarrow{\mathbb{R}} be differentiable. Then c∈Dc\in D is a critical point of ff if f′⁢(c)=0f^{\prime}(c)=0.

Given a differentiable function f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}}, we know it attains both a maximum and a minimum by the Extreme Value Theorem. Either or both of these extrema might occur at an endpoint of the interval [a,b][a,b]. But if they don’t, they occur at some interior point c∈(a,b)c\in(a,b), and this point must be a criticial point of ff by the Interior Extremum Theorem. So to find the maximum and minimum values attained by ff, we just need to evaluate it at the endpoints a,ba,b and any interior critical points, and extract the extreme values.

Example 3.3

Find the maximum and minimum values attained by the function

f:[0,3]→ℝ,f⁢(x)=x2−2⁢x+7.f:[0,3]\rightarrow{\mathbb{R}},\qquad f(x)=x^{2}-2x+7.

Solution: First note that, since ff is continuous, it certainly attains both a maximum and a minimum by the Extreme Value Theorem (Theorem 1.49). Since ff is differentiable on (0,3)(0,3), it follows from the Interior Extremum Theorem that each extremum occurs either at an endpoint, that is, 0 or 33, or at an interior critical point of ff. Now

f′⁢(x)=2⁢x−2=0f^{\prime}(x)=2x-2=0

if and only if x=1x=1, so 11 is the only critical point of ff. Since f⁢(1)=6f(1)=6, f⁢(0)=7f(0)=7, and f⁢(3)=10f(3)=10, we deduce that ff attains a maximum value of 1010 at the right endpoint 33, and a minimum value of 66 at the interior critical point 11.

Remark In this case, using calculus was really overkill, since we could have deduced the same information by simply completing the square:

f⁢(x)=(x−1)2+6.f(x)=(x-1)^{2}+6.

We can also use information about the derivative to show, in some circumstances, that the extrema of a differentiable function on [a,b][a,b] do not occur at its endpoints.

Proposition 3.4

Let f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} be differentiable.

  1. (i)
    ​

    If f′⁢(a)>0f^{\prime}(a)>0, then f⁢(a)f(a) is not the maximum value attained by ff.

  2. (ii)
    ​

    If f′⁢(a)<0f^{\prime}(a)<0, then f⁢(a)f(a) is not the minimum value attained by ff.

  3. (iii)
    ​

    If f′⁢(b)>0f^{\prime}(b)>0, then f⁢(b)f(b) is not the minimum value attained by ff.

  4. (iv)
    ​

    If f′⁢(b)<0f^{\prime}(b)<0, then f⁢(b)f(b) is not the maximum value attained by ff.

Proof.

The proofs of each part are very similar, so we will just prove (i).

Assume f′⁢(a)>0f^{\prime}(a)>0. Assume further, towards a contradiction, that ff does attain its maximum at aa. Since ff is differentiable at aa,

sn=f⁢(xn)−f⁢(a)xn−a→f′⁢(a)>0,s_{n}=\frac{f(x_{n})-f(a)}{x_{n}-a}\rightarrow f^{\prime}(a)>0,

for any sequence (xn)(x_{n}) in (a,b](a,b] converging to aa (by Theorem 1.36). But f⁢(xn)≤f⁢(a)f(x_{n})\leq f(a) by assumption, and xn>ax_{n}>a, so sn≤0s_{n}\leq 0 for all nn. Hence f′⁢(a)≤0f^{\prime}(a)\leq 0 by Proposition 1.7, a contradiction. ∎
