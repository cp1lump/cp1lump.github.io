A sequential characterization of integrability

4.4 A sequential characterization of integrability

We can generalize the argument used in Example 4.15 to give a rather elegant (and useful) characterization of Riemann integrability in terms of sequences of dissections.

Theorem 4.16

Let f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} be a bounded function. Then ff is Riemann integrable if and only if there exists a sequence (𝒟n)(\mathscr{D}_{n}) of dissections of [a,b][a,b] such that

u𝒟n⁢(f)−l𝒟n⁢(f)→0.u_{\mathscr{D}_{n}}(f)-l_{\mathscr{D}_{n}}(f)\rightarrow 0.

In this case,

∫abf=liml𝒟n⁢(f)=limu𝒟n⁢(f).\int_{a}^{b}f=\lim l_{\mathscr{D}_{n}}(f)=\lim u_{\mathscr{D}_{n}}(f).

To illustrate the power of Theorem 4.16, let’s use it to prove that every monotonic (i.e. increasing or decreasing) function is Riemann integrable.

Theorem 4.17

Let f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} be increasing. Then ff is Riemann integrable.

Proof.

First note that since ff is increasing, it is bounded (below by f⁢(a)f(a) and above by f⁢(b)f(b), so l⁢(f)l(f) and u⁢(f)u(f) exist. Note that Theorem 4.17 partially covers both Examples 4.14 and 4.15: in both these examples the function in question is increasing so Theorem 4.17 implies that they’re both Riemann integrable. It does not, however, tell us anything about the value of ∫abf\int_{a}^{b}f, only that it exists. Note also that the function in Example 4.14 is discontinuous, at a single point (x=1x=1), which illustrates that while a discontinuous function cannot be differentiable (Proposition 2.8), it certainly can be integrable. In fact, we can construct examples of functions which are discontinuous at infinitely many points in [a,b][a,b] and yet are still Riemann integrable on [a,b][a,b].

Example 4.19

Consider the function f:[0,1]→ℝf:[0,1]\rightarrow{\mathbb{R}} defined so that, f⁢(0)=0f(0)=0 and, for all x∈(1n+1,1n]x\in(\frac{1}{n+1},\frac{1}{n}], where n∈ℤ+n\in{\mathbb{Z}}^{+}, f⁢(x)=1nf(x)=\frac{1}{n}. By construction, ff is increasing and so is Riemann integrable on [0,1][0,1] by Theorem 4.17. Note, however, that ff is discontinuous at every point 1n\frac{1}{n} for n≥2n\geq 2.

If even such an extreme example as this is Riemann integrable, you might wonder whether there are any bounded functions which fail to be integrable. Fear not: such functions certainly do exist. Here is an example.

Example 4.20

Let f:[0,1]→ℝf:[0,1]\rightarrow{\mathbb{R}} such that f⁢(x)=0f(x)=0 if x∈ℚx\in{\mathbb{Q}} and f⁢(x)=1f(x)=1 if x∉ℚx\notin{\mathbb{Q}}. I claim that ff is not Riemann integrable.

Theorem 4.17 gives us one interesting class of functions f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} that are Riemann integrable: those that are increasing. Our next theorem gives us another rather more useful class: continuous functions.

Theorem 4.21

Let f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} be continuous. Then ff is Riemann integrable.

Proof.

First note that, since ff is continuous, it is certainly bounded (by the Extreme Value Theorem), so l⁢(f)l(f) and u⁢(f)u(f) exist.

Consider the sequence (𝒟n)(\mathscr{D}_{n}) of regular dissections of [a,b][a,b] of size 2n2^{n}. Hence l𝒟n⁢(f)→Kl_{\mathscr{D}_{n}}(f)\rightarrow K and u𝒟n⁢(f)→Lu_{\mathscr{D}_{n}}(f)\rightarrow L for some numbers KK and LL by the Monotone Convergence Theorem, so u𝒟n⁢(f)−l𝒟n⁢(f)→K−L≥0u_{\mathscr{D}_{n}}(f)-l_{\mathscr{D}_{n}}(f)\rightarrow K-L\geq 0 by the Algebra of Limits and Proposition 1.7.
