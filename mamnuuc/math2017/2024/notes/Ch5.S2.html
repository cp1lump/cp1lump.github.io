5.2 The second form and a practical method for computing integrals

Theorem 5.1 immediately implies a second theorem, also called the Fundamental Theorem of the Calculus, which renders the job of computing many Riemann integrals almost trivial:

Theorem 5.4 (Fundamental Theorem of the Calculus version 2)

Let
f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} be continuous and F:[a,b]→ℝF:[a,b]\rightarrow{\mathbb{R}} be any differentiable function such that F′=fF^{\prime}=f. Then

∫abf=F⁢(b)−F⁢(a).\int_{a}^{b}f=F(b)-F(a).
Proof.

Define the function g:[a,b]→ℝg:[a,b]\rightarrow{\mathbb{R}},

g⁢(x)=(∫axf)−F⁢(x).g(x)=\left(\int_{a}^{x}f\right)-F(x).

Then, by Theorem 5.1 and the definition of FF,

g′⁢(x)=f⁢(x)−f⁢(x)=0g^{\prime}(x)=f(x)-f(x)=0

for all x∈[a,b]x\in[a,b]. Hence, by Proposition 3.11, gg is constant, so g⁢(b)=g⁢(a)g(b)=g(a), that is

(∫abf)−F⁢(b)=(∫aaf)−F⁢(a)=0−F⁢(a),\left(\int_{a}^{b}f\right)-F(b)=\left(\int_{a}^{a}f\right)-F(a)=0-F(a),

and the result immediately follows. ∎

∫0πf=F⁢(π)−F⁢(0)=−cos⁡π+cos⁡0=2.\int_{0}^{\pi}f=F(\pi)-F(0)=-\cos\pi+\cos 0=2.

Sometimes we can use the Fundamental Theorem of the Calculus to compute an integral even when we can’t write down an antiderivative of the integrand. Before giving an example, we will need a quick proposition:

Proposition 5.6

Let f:[−a,a]→ℝf:[-a,a]\rightarrow{\mathbb{R}} be a continuous odd function (meaning f⁢(−x)=−f⁢(x)f(-x)=-f(x) for all x∈[−a,a]x\in[-a,a]). Then

∫−aaf=0.\int_{-a}^{a}f=0.
Proof.

Define F:[0,a]→ℝF:[0,a]\rightarrow{\mathbb{R}} by

F⁢(x)=∫−xxf=∫−x0f+∫0xfF(x)=\int_{-x}^{x}f=\int_{-x}^{0}f+\int_{0}^{x}f

by Theorem 4.22. Then, by Theorem 5.1, Corollary 5.2, and the Chain Rule

F′⁢(x)=−(−f⁢(−x))+f⁢(x)=f⁢(−x)+f⁢(x)=0,F^{\prime}(x)=-(-f(-x))+f(x)=f(-x)+f(x)=0,

since ff is odd. Hence, by Proposition 3.11, FF is contant, so

∫−aaf=F⁢(a)=F⁢(0)=0.\int_{-a}^{a}f=F(a)=F(0)=0.

∎

Example 5.7

f⁢(x)=sin⁡x⁢sin⁡(x2)f(x)=\sin x\sin(x^{2}) is an odd continuous function, so

∫−ππf=0\int_{-\pi}^{\pi}f=0

by Proposition 5.6. Note that we deduced this from the Fundamental Theorem of the Calculus even though we have no idea how to write down an antiderivative of f⁢(x)f(x).

Having established Theorem 5.4, we see that we can explicitly compute

∫abf⁢(x)⁢𝑑x\int_{a}^{b}f(x)\,dx

if we can think of an antiderivative of ff, that is, any function whose derivative is f⁢(x)f(x). This trick is so pervasive in integral calculus that it leads the unwary to identify integration of ff (that is, the process of computing a Riemann integral ∫abf\int_{a}^{b}f) with the process of writing down an antiderivative of ff. Indeed, it is common practice to call an antiderivative of ff an indefinite integral of ff and to denote any such function by the symbol

∫f⁢(x)⁢𝑑x.\int f(x)\,dx.

This notation has some practical advantages, but it also, unfortunately, generates a huge amount of confusion. Note that

∫abf⁢(x)⁢𝑑x\int_{a}^{b}f(x)\,dx

is a single number and that its definition has nothing whatsoever to do with antiderivatives of ff: it is the unique real number which is no greater than any upper Riemann sum of ff on [a,b][a,b] and no less than any lower Riemann sum of ff on [a,b][a,b]. In particular, it is not a function of the “variable” xx. Indeed, we could equally well have written it ∫abf⁢(y)⁢𝑑y\int_{a}^{b}f(y)\,dy or ∫abf⁢(Γℵ^)⁢𝑑Γℵ^\int_{a}^{b}f(\Gamma_{\widehat{\aleph}})\,d\Gamma_{\widehat{\aleph}}, which is one reason to prefer the simpler notation ∫abf\int_{a}^{b}f. It is a function of aa and bb, however.

By contrast, ∫f⁢(x)⁢𝑑x\int f(x)\,dx is just a (slightly ambiguous) symbol denoting any function whose derivative (at xx) is f⁢(x)f(x). It is a function of xx, not a single number, and its definition has nothing to do with Riemann sums. That these two things turn out to be closely related is (as its name suggests) a very important theorem: the Fundamental Theorem of the Calculus. To understand calculus properly it is important to maintain a clear conceptual distinction between the Riemann integral ∫abf\int_{a}^{b}f and any antiderivative that one might use to compute it.
