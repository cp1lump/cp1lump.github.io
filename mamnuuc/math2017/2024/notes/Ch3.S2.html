The Mean Value Theorem

3.2 The Mean Value Theorem

Our next theorem has hypotheses (assumptions on ff) which look very restrictive – so restrictive that one might wonder whether the theorem is of any practical use at all. In fact, as we shall see, it has very powerful and useful consequences.

Theorem 3.6 (Rolle’s Theorem)

Let ff be continuous on [a,b][a,b] and differentiable on (a,b)(a,b). Assume f⁢(a)=f⁢(b)f(a)=f(b). Then there exists c∈(a,b)c\in(a,b) such that f′⁢(c)=0f^{\prime}(c)=0.

Proof.

By the Extreme Value Theorem, ff attains both a maximum and a minimum value on [a,b][a,b]. If both of these occur at the endpoints of [a,b][a,b] then, since f⁢(a)=f⁢(b)f(a)=f(b), the maximum value coincides with the minimum value, whence it follows that f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} is constant. But then f′⁢(x)=0f^{\prime}(x)=0 for all x∈(a,b)x\in(a,b), so c=(b+a)/2c=(b+a)/2, for example, has f′⁢(c)=0f^{\prime}(c)=0.

Hence, we may assume that either the maximum or the minimum value does not occur at an endpoint. But then ff attains an extremum at some interior point c∈(a,b)c\in(a,b), so f′⁢(c)=0f^{\prime}(c)=0 by the Interior Extremum Theorem. ∎

Theorem 3.7 (The Mean Value Theorem)

Let ff be continuous on [a,b][a,b] and differentiable on (a,b)(a,b). Then there exists c∈(a,b)c\in(a,b) such that

f′⁢(c)=f⁢(b)−f⁢(a)b−a.f^{\prime}(c)=\frac{f(b)-f(a)}{b-a}.
Proof.

Let α=(f⁢(b)−f⁢(a))/(b−a)\alpha=(f(b)-f(a))/(b-a) and define g:[a,b]→ℝg:[a,b]\rightarrow{\mathbb{R}} such that

g⁢(x)=f⁢(x)−α⁢(x−a).g(x)=f(x)-\alpha(x-a).

Then gg is continuous, and is differentiable on (a,b)(a,b). Furthermore g⁢(a)=f⁢(a)g(a)=f(a) and g⁢(b)=f⁢(a)g(b)=f(a). So gg satisfies the hypotheses of Rolle’s Theorem, and we deduce that there exists c∈(a,b)c\in(a,b) such that g′⁢(c)=0g^{\prime}(c)=0. But

g′⁢(c)=f′⁢(c)−αg^{\prime}(c)=f^{\prime}(c)-\alpha

which completes the proof. ∎

Remarks

  • •
    ​

    The Mean Value Theorem has a nice geometric interpretation. Consider the graph y=f⁢(x)y=f(x) of a differentiable function f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}}. For each pair of distinct numbers a<ba<b, we can construct the chord (straight line) passing through the points (a,f⁢(a))(a,f(a)) and (b,f⁢(b))(b,f(b)) on the graph. Its slope is (f⁢(b)−f⁢(a))/(b−a)(f(b)-f(a))/(b-a). The Mean Value Theorem asserts that, at some point (c,f⁢(c))(c,f(c)) on the graph between these two points, the tangent line to the graph is parallel to the chord.

  • •
    ​

    The Mean Value Theorem guarantees the existence of a point cc with the required f′⁢(c)f^{\prime}(c). It says nothing about uniqueness: there could be more than one such point.

  • •
    ​

    Clearly Rolle’s Theorem is a special case of the Mean Value Theorem (geometrically, the case where the chord is horizontal). It’s slightly surprising at first sight, therefore, that Rolle’s Theorem actually implies the Mean Value Theorem.

You perhaps don’t realize it, but you have probably been using the Mean Value Theorem rather a lot. We have seen (Example 2.2) that if a function is constant then its derivative is zero everywhere. When one solves a differential equation, one often uses the converse fact: if a function has zero derivative everywhere, then it must be constant. But how do we know this? It’s a consequence of the Mean Value Theorem!

Example 3.8

Suppose we know that f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} is differentiable and satisfies f′⁢(x)=x2f^{\prime}(x)=x^{2} for all xx, and f⁢(0)=2f(0)=2. Then surely

f⁢(x)=x33+2,f(x)=\frac{x^{3}}{3}+2,

right? But how do we know this? Certainly, F⁢(x):=x33+2F(x):=\frac{x^{3}}{3}+2 is an example of a function whose derivative at each xx is x2x^{2}, and whose value at 0 is 22, so it’s possible that f=Ff=F. But, perhaps there’s more than one function satisfying those properties?

In fact, we can prove that f=Ff=F, as follows. Let g:ℝ→ℝg:{\mathbb{R}}\rightarrow{\mathbb{R}} be

g⁢(x)=f⁢(x)−F⁢(x)=f⁢(x)−x33−2.\displaystyle g(x)=f(x)-F(x)=f(x)-\frac{x^{3}}{3}-2.

Then, for all xx,

g′⁢(x)=f′⁢(x)−x2=0.\displaystyle g^{\prime}(x)=f^{\prime}(x)-x^{2}=0.

Now assume, towards a contradiction, that gg is not constant. Then there exist a,b∈ℝa,b\in{\mathbb{R}} with a<ba<b such that g⁢(a)≠g⁢(b)g(a)\neq g(b). But then, by the Mean Value Theorem applied to gg on [a,b][a,b], there exists c∈(a,b)c\in(a,b) such that

g′⁢(c)=g⁢(b)−g⁢(a)b−a≠0g^{\prime}(c)=\frac{g(b)-g(a)}{b-a}\neq 0

a contradiction since g′⁢(x)=0g^{\prime}(x)=0 everywhere. Hence, gg is constant, so for all x∈ℝx\in{\mathbb{R}} g⁢(x)=g⁢(0)=0g(x)=g(0)=0, that is, f=Ff=F. □\Box

When we first learn calculus, we are taught to think of f′⁢(x)f^{\prime}(x) as the “rate of change of ff with respect to the variable xx”. So if, at a particular point a∈Da\in D, f′⁢(a)>0f^{\prime}(a)>0, we expect that ff should be strictly increasing, at least sufficiently close to aa. You’ve already seen (Example 2.9) that this expectation can be false: the relationship between f′f^{\prime} and the increasing/decreasing behaviour of ff can be subtle. Before proceeding further, let’s make absolutely precise what we mean by increasing and decreasing functions.

Definition 3.9

f:D→ℝf:D\rightarrow{\mathbb{R}} is increasing if for all x,y∈Dx,y\in D with x<yx<y, f⁢(x)≤f⁢(y)f(x)\leq f(y). It is strictly increasing if for all x,y∈Dx,y\in D with x<yx<y, f⁢(x)<f⁢(y)f(x)<f(y). Similarly, ff is decreasing if for all x,y∈Dx,y\in D with x<yx<y, f⁢(x)≥f⁢(y)f(x)\geq f(y), and strictly decreasing if for all x,y∈Dx,y\in D with x<yx<y, f⁢(x)>f⁢(y)f(x)>f(y).

f⁢(x)−f⁢(y)\displaystyle f(x)-f(y) =\displaystyle= (x−y)⁢(x2+x⁢y+y2)\displaystyle(x-y)(x^{2}+xy+y^{2})

Note that Definition 3.9 makes no mention of the derivative of ff and may, in principle, apply when ff is not differentiable, or even continuous. For example, the step function defined in Example 1.39, is an increasing function. Our next result uses the Mean Value Theorem to establish some useful links between f′f^{\prime} and the increasing/decreasing behaviour of ff.

Proposition 3.11

Let I⊆ℝI\subseteq{\mathbb{R}} be an interval and f:I→ℝf:I\rightarrow{\mathbb{R}} be differentiable. Then

  1. (i)
    ​

    ff is increasing if and only if f′⁢(x)≥0f^{\prime}(x)\geq 0 for all x∈Ix\in I.

  2. (ii)
    ​

    ff is decreasing if and only if f′⁢(x)≤0f^{\prime}(x)\leq 0 for all x∈Ix\in I.

  3. (iii)
    ​

    ff is constant if and only if f′⁢(x)=0f^{\prime}(x)=0 for all x∈Ix\in I.

  4. (iv)
    ​

    If f′⁢(x)>0f^{\prime}(x)>0 for all x∈Ix\in I then ff is strictly increasing.

  5. (v)
    ​

    If f′⁢(x)<0f^{\prime}(x)<0 for all x∈Ix\in I then ff is strictly decreasing.

Proof.
  1. (i)
    ​

    Assume f′⁢(x)≥0f^{\prime}(x)\geq 0 for all xx in II, but that ff is not increasing. Then there exist a,b∈Ia,b\in I with a<ba<b such that f⁢(a)>f⁢(b)f(a)>f(b). But then, by the MVT, at some c∈(a,b)c\in(a,b),

    f′⁢(c)=f⁢(b)−f⁢(a)b−a<0f^{\prime}(c)=\frac{f(b)-f(a)}{b-a}<0

    contradicting the assumption that f′≥0f^{\prime}\geq 0. Hence ff is increasing.

    Conversely, assume that ff is increasing. If II has a left endpoint a∈Ia\in I, then ff attains a minimum at aa, so f′⁢(a)≥0f^{\prime}(a)\geq 0 by Proposition 3.4. Similarly, if II has a right endpoint b∈Ib\in I then ff attains a maximum at bb so f′⁢(b)≥0f^{\prime}(b)\geq 0 by Proposition 3.4. Let xx be an interior point of II, so x∈(a,b)⊆Ix\in(a,b)\subseteq I.     But sn≥0s_{n}\geq 0 for all nn, since ff is increasing and xn>xx_{n}>x. Hence it’s limit f′⁢(x)≥0f^{\prime}(x)\geq 0 by Proposition 1.7.

    Since strictly increasing and strictly decreasing functions are automatically injective, Proposition 3.11 gives us a sneaky way of showing that some functions are injective.

Example 3.12

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} such that f⁢(x)=2⁢x+sin⁡xf(x)=2x+\sin x. Then ff is strictly increasing, and hence injective.

Proof.

The function ff is differentiable, with derivative11 1 Strictly speaking, you haven’t yet proved that sin\sin is differentiable, with derivative cos\cos, so this argument requires you to suspend disbelief for the time being.

f′⁢(x)=2+cos⁡x≥1>0.\displaystyle f^{\prime}(x)=2+\cos x\geq 1>0.

Hence, by Proposition 3.11, ff is strictly increasing, and hence injective. ∎

We finish this section by studying a function f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} with some very counterintuitive properties.

Example 3.14

Consider the function f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} defined so that

f⁢(x)={x+2⁢x2⁢sin⁡(1/x)if x≠00if x=0f(x)=\left\{\begin{array}[]{cc}x+2x^{2}\sin(1/x)&\mbox{if $x\neq 0$}\\ 0&\mbox{if $x=0$}\end{array}\right.

Claim: ff is differentiable everywhere, has f′⁢(0)=1>0f^{\prime}(0)=1>0, but is not increasing on any neighbourhood of 0.

Proof.

ff coincides with the function g:U→ℝg:U\rightarrow{\mathbb{R}}, g⁢(x)=x+2⁢x2⁢sin⁡(1/x)g(x)=x+2x^{2}\sin(1/x) on the open set U=ℝ\{0}U={\mathbb{R}}{\backslash}\{0\}. gg is differentiable by the Product and Chain Rules, so ff is differentiable on UU by the Localization Lemma, and for all a≠0a\neq 0,

f′⁢(a)=g′⁢(a)=1+4⁢a⁢sin⁡(1/a)−2⁢cos⁡(1/a).f^{\prime}(a)=g^{\prime}(a)=1+4a\sin(1/a)-2\cos(1/a). (3.2)

Consider now the case a=0a=0. Let (xn)(x_{n}) be any sequence in ℝ\{0}{\mathbb{R}}{\backslash}\{0\} converging to 0. Then

sn:=f⁢(xn)−f⁢(0)xn−0=1+2⁢xn⁢sin⁡(1/xn).s_{n}:=\frac{f(x_{n})-f(0)}{x_{n}-0}=1+2x_{n}\sin(1/x_{n}).

Now −|xn|≤xn⁢sin⁡(1/xn)≤|xn|-|x_{n}|\leq x_{n}\sin(1/x_{n})\leq|x_{n}|, so xn⁢sin⁡(1/xn)→0x_{n}\sin(1/x_{n})\rightarrow 0 by the Squeeze Rule, and hence sn→1s_{n}\rightarrow 1 (by the Algebra of Limits). Hence, ff is differentiable at 0 and f′⁢(0)=1f^{\prime}(0)=1.

For the final part, assume, to the contrary, that there exists ε>0\varepsilon>0 such that f:(−ε,ε)→ℝf:(-\varepsilon,\varepsilon)\rightarrow{\mathbb{R}} is increasing. Then, by Proposition 3.11, f′⁢(x)≥0f^{\prime}(x)\geq 0 for all x∈(−ε,ε)x\in(-\varepsilon,\varepsilon). But there exists m∈ℤ+m\in{\mathbb{Z}}^{+} such that m>1/(2⁢π⁢ε)m>1/(2\pi\varepsilon). Then x=1/(2⁢π⁢m)∈(0,ε)x=1/(2\pi m)\in(0,\varepsilon) and, by equation (3.2),

f′⁢(x)=1+4⁢x×0−2×1=−1<0,f^{\prime}(x)=1+4x\times 0-2\times 1=-1<0,

a contradiction. ∎

Note that we proved ff can’t be increasing on any interval containing 0 by showing that its derivative takes negative values on every such interval. So we are again using Proposition 3.11.
