The first form and the Intermediate Value Theorem

5.1 The first form and the Intermediate Value Theorem

It’s important to realize that the Riemann integral of a function f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} is a single number ∫abf\int_{a}^{b}f, not a function. If f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} is continuous, it is Riemann integrable on any interval [a,b]⊂ℝ[a,b]\subset{\mathbb{R}}. If we allow (say) the right endpoint of the interval [a,b][a,b] to vary while keeping aa fixed, we can think of ∫abf\int_{a}^{b}f as a function of bb. It’s natural to ask what analytic properties this new function has. The (first form of) the Fundamental Theorem of the Calculus says that b↦∫abfb\mapsto\int_{a}^{b}f is differentiable, and its derivative is ff.

Theorem 5.1 (Fundamental Theorem of the Calculus version 1)

Let
f:I→ℝf:I\rightarrow{\mathbb{R}} be continuous, where I⊆ℝI\subseteq{\mathbb{R}} is an interval. Choose a∈Ia\in I and define F:I→ℝF:I\rightarrow{\mathbb{R}} by

F⁢(x)=∫axf.F(x)=\int_{a}^{x}f.

Then FF is differentiable, and F′=fF^{\prime}=f.

Proof.

First note that ff is continuous on [a,x]⊆I[a,x]\subseteq I (or [x,a][x,a] if x<ax<a) for all x∈[a,b]x\in[a,b], so FF is well-defined by Theorem 4.21. We wish to compute

limy→xF⁢(y)−F⁢(x)y−x.\lim_{y\rightarrow x}\frac{F(y)-F(x)}{y-x}.

We will do this using the sequential criterion for limits (Theorem 1.36). So, let (yn)(y_{n}) be any sequence in I\{x}I{\backslash}\{x\} such that yn→xy_{n}\rightarrow x, and

sn=F⁢(yn)−F⁢(x)yn−x.s_{n}=\frac{F(y_{n})-F(x)}{y_{n}-x}.

We must show that sn→f⁢(x)s_{n}\rightarrow f(x).

For each nn, either yn>xy_{n}>x or yn<xy_{n}<x. If yn>xy_{n}>x then, by Theorem 4.22,

sn=1yn−x⁢∫xynf,s_{n}=\frac{1}{y_{n}-x}\int_{x}^{y_{n}}f,

whereas if yn<xy_{n}<x,

sn=1x−yn⁢∫ynxf.s_{n}=\frac{1}{x-y_{n}}\int_{y_{n}}^{x}f.

In either case, by the Extreme Value Theorem, there exist wnw_{n} and znz_{n} between xx and yny_{n} such that f⁢(wn)f(w_{n}) is the minimum value of ff on the closed interval with endpoints xx and yny_{n}, and f⁢(zn)f(z_{n}) is the maximum value of ff on this interval. Hence, by Proposition 4.8,

1yn−x⁢f⁢(wn)⁢(yn−x)≤\displaystyle\frac{1}{y_{n}-x}f(w_{n})(y_{n}-x)\leq sn\displaystyle s_{n} ≤1yn−x⁢f⁢(zn)⁢(yn−x)if yn>x,\displaystyle\leq\frac{1}{y_{n}-x}f(z_{n})(y_{n}-x)\qquad\mbox{if $y_{n}>x$,}
and1x−yn⁢f⁢(wn)⁢(x−yn)≤\displaystyle\mbox{and}\qquad\frac{1}{x-y_{n}}f(w_{n})(x-y_{n})\leq sn\displaystyle s_{n} ≤1x−yn⁢f⁢(zn)⁢(x−yn)if yn<x.\displaystyle\leq\frac{1}{x-y_{n}}f(z_{n})(x-y_{n})\qquad\mbox{if $y_{n}<x$.}

In either case, we conclude that

f⁢(wn)≤sn≤f⁢(zn).f(w_{n})\leq s_{n}\leq f(z_{n}).

Now, yn→xy_{n}\rightarrow x so wn→xw_{n}\rightarrow x and zn→xz_{n}\rightarrow x also, by the Squeeze Rule, and ff is continuous, so f⁢(wn)→f⁢(x)f(w_{n})\rightarrow f(x) and f⁢(zn)→f⁢(x)f(z_{n})\rightarrow f(x). Hence, by the Squeeze Rule again, sn→f⁢(x)s_{n}\rightarrow f(x), which completes the proof. ∎

This theorem tells us something interesting and far from obvious: any function which is continuous on an interval is the derivative of some differentiable function on that interval. Note that the converse of this is false: the derivative of a function may fail to be a continuous function (see Example 3.14 for a counterexample). Combining Theorem 5.1 with Darboux’s Theorem (Theorem 3.15) we get a (frankly rather ridiculous) proof of the Intermediate Value Theorem:

Theorem 1.48 (The Intermediate Value Theorem) Let f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} be continuous and yy be a number between f⁢(a)f(a) and f⁢(b)f(b). Then there exists c∈[a,b]c\in[a,b] such that f⁢(c)=yf(c)=y.

Proof.

Let F:[a,b]→ℝF:[a,b]\rightarrow{\mathbb{R}}, F⁢(x)=∫axfF(x)=\int_{a}^{x}f. Then FF is differentiable with F′=fF^{\prime}=f (by Theorem 5.1), so yy is a number between F′⁢(a)=f⁢(a)F^{\prime}(a)=f(a) and F′⁢(b)=f⁢(b)F^{\prime}(b)=f(b). Hence, there exists c∈[a,b]c\in[a,b] such that F′⁢(c)=yF^{\prime}(c)=y (Darboux’s Theorem), that is, f⁢(c)=yf(c)=y. ∎

Theorem 5.1 says that (provided ff is continuous!) ∫abf\int_{a}^{b}f is a differentiable function of the upper integration limit bb, and its derivative is ff. What if we fix bb and think of it as a function of aa? Is this function also differentiable? If so, what is its derivative?

Corollary 5.2

Let f:I→ℝf:I\rightarrow{\mathbb{R}} be continuous, where I⊆ℝI\subseteq{\mathbb{R}} is an interval. Choose b∈Ib\in I and define F:I→ℝF:I\rightarrow{\mathbb{R}} by

F⁢(x)=∫xbf.F(x)=\int_{x}^{b}f.

Then FF is differentiable, and F′=−fF^{\prime}=-f.

Proof.

Choose any a∈Ia\in I and define G:I→ℝG:I\rightarrow{\mathbb{R}}, G⁢(x)=∫axfG(x)=\int_{a}^{x}f. Then by Proposition 4.22

G⁢(x)+F⁢(x)=∫axf+∫xbf=∫abf=CG(x)+F(x)=\int_{a}^{x}f+\int_{x}^{b}f=\int_{a}^{b}f=C

a constant. GG is differentiable and G′=fG^{\prime}=f by Theorem 5.1, so F⁢(x)=C−G⁢(x)F(x)=C-G(x) is differentiable and F′⁢(x)=0−F′⁢(x)=−f⁢(x)F^{\prime}(x)=0-F^{\prime}(x)=-f(x). ∎

Of course, we can apply all the usual rules of differentiation to functions defined in this way (as integrals of other functions).
