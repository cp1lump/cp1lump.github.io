Analyticity versus smoothness

7.6 Analyticity versus smoothness

So far we have only considered power series based at 0, but the whole theory extends immediately to power series with some other base point x0∈ℝx_{0}\in{\mathbb{R}}, that is, series of the form

f⁢(x)=∑n=0∞an⁢(x−x0)n.f(x)=\sum_{n=0}^{\infty}a_{n}(x-x_{0})^{n}.

The point is that f⁢(x)=g⁢(x−x0)f(x)=g(x-x_{0}), where g⁢(x)=∑n=0∞an⁢xng(x)=\sum_{n=0}^{\infty}a_{n}x^{n}, so (by the Chain Rule) ff is smooth on (x0−R,x0+R)(x_{0}-R,x_{0}+R) where RR is the radius of convergence of gg, and the derivative of ff is again its termwise derivative. We will refer to RR as the radius of convergence of ff.

A function ff is analytic if it locally coincides with some convergent power series, everywhere in its domain. More precisely:

Definition 7.33

Let U⊆ℝU\subseteq{\mathbb{R}} be open. Then f:U→ℝf:U\rightarrow{\mathbb{R}} is analytic if, for each x0∈Ux_{0}\in U, there exists ε>0\varepsilon>0 and a power series

∑n=0∞an⁢(x−x0)n\sum_{n=0}^{\infty}a_{n}(x-x_{0})^{n}

which converges to f⁢(x)f(x) for all x∈(x0−ε,x0+ε)x\in(x_{0}-\varepsilon,x_{0}+\varepsilon).

Example 7.34

exp:ℝ→ℝ\exp:{\mathbb{R}}\rightarrow{\mathbb{R}} is analytic since

exp⁡(x)=exp⁡(x0)⁢exp⁡(x−x0)=∑n=0∞exp⁡(x0)n!⁢(x−x0)n\exp(x)=\exp(x_{0})\exp(x-x_{0})=\sum_{n=0}^{\infty}\frac{\exp(x_{0})}{n!}(x-x% _{0})^{n}

for all x,x0∈ℝx,x_{0}\in{\mathbb{R}} (so for any x0x_{0} we can take any ε>0\varepsilon>0 in the definition above).

If f:U→ℝf:U\rightarrow{\mathbb{R}} is analytic, it is certainly smooth, by Corollary 7.23. Rather remarkably, the converse is false, that is, not all smooth functions are analytic. This section is devoted to constructing a class of functions f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} which are smooth but not analytic.

Definition 7.36

Given a polynomial p⁢(x)=a0+a1⁢x+⋯+ak⁢xkp(x)=a_{0}+a_{1}x+\cdots+a_{k}x^{k}, we define the function fp:ℝ→ℝf_{p}:{\mathbb{R}}\rightarrow{\mathbb{R}} such that

fp⁢(x)={0,x≤0p⁢(1/x)⁢exp⁡(−1/x),x>0.f_{p}(x)=\left\{\begin{array}[]{cc}0,&x\leq 0\\ p(1/x)\exp(-1/x),&x>0.\end{array}\right.

Lemma 7.37 (Exponentials beat powers – sequences)

For any k∈ℕk\in{\mathbb{N}} the sequence

an=nk⁢exp⁡(−n)→0.a_{n}=n^{k}\exp(-n)\rightarrow 0.
Proof.
an+1an=(1+1n)k⁢exp⁡(−1)→1e<1\frac{a_{n+1}}{a_{n}}=\left(1+\frac{1}{n}\right)^{k}\exp(-1)\rightarrow\frac{1% }{e}<1

Lemma 7.38 (Exponentials beat powers – functions)

Corollary 7.41

The function fpf_{p} defined in Definition 7.36 is smooth and has fp(n)⁢(0)=0f_{p}^{(n)}(0)=0 for all n≥0n\geq 0.

Proof.

Denote by XX the set of functions {fp:p a polynomial}\{f_{p}\>:\>\mbox{$p$ a polynomial}\}. It follows that almost every function fpf_{p} is smooth, but not analytic. (The only exception is f0f_{0}, where we take p⁢(x)p(x) to be the trivial polynomial p=0p=0. Clearly f0≡0f_{0}\equiv 0 which is trivially analytic.) Let’s consider the simplest example.

Example 7.42

Consider the special case p⁢(x)=1p(x)=1. The corresponding function is

f⁢(x)={exp⁡(−1/x),x>00,x≤0.f(x)=\left\{\begin{array}[]{cc}\exp(-1/x),&x>0\\ 0,&x\leq 0.\end{array}\right.

Assume, towards a contradiction, that ff is analytic. Then there exist ε>0\varepsilon>0 and {an:n∈ℕ}\{a_{n}:n\in{\mathbb{N}}\} such that

□\Box

It’s not hard to construct from Example 7.42 smooth functions which vanish exactly outside some bounded interval (a,b)(a,b) but are strictly positive on (a,b)(a,b). Such functions are often called bump functions.

Example 7.43
