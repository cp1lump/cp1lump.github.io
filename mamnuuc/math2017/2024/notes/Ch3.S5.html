Higher derivatives and Taylor’s Theorem

3.5 Higher derivatives and Taylor’s Theorem

If a function f:D→ℝf:D\rightarrow{\mathbb{R}} is differentiable (everywhere on DD) then its derivative defines another function f′:D→ℝf^{\prime}:D\rightarrow{\mathbb{R}}, and we can ask whether this function, in turn, is differentiable (recall that, in general, it might not even be continuous). If f′f^{\prime} is differentiable, then its derivative is denoted f′′:D→ℝf^{\prime\prime}:D\rightarrow{\mathbb{R}} and called the second derivative of ff. Similarly, if f′′:D→ℝf^{\prime\prime}:D\rightarrow{\mathbb{R}} is differentiable, its derivative f′′′:D→ℝf^{\prime\prime\prime}:D\rightarrow{\mathbb{R}} is the third derivative of ff. Proceeding inductively, we can define the nthn^{\mbox{th}} derivative of ff, denoted f(n)f^{(n)} to be the derivative (if it exists), of f(n−1)f^{(n-1)}, where f(0)f^{(0)} is, by definition, just the function ff itself (so f(1)=f′f^{(1)}=f^{\prime}, f(2)=f′′f^{(2)}=f^{\prime\prime} etc.). If f(n):D→ℝf^{(n)}:D\rightarrow{\mathbb{R}} exists, we say that ff is nn times differentiable, and if ff is nn times differentiable for all n∈ℤ+n\in{\mathbb{Z}}^{+}, we say that ff is smooth. It is convenient to develop some terminology and notation for describing the differentiability of functions:

Definition 3.19

A function f:D→ℝf:D\rightarrow{\mathbb{R}} is continuously differentiable if it is differentiable and f′:D→ℝf^{\prime}:D\rightarrow{\mathbb{R}} is continuous. The function is nn times continuously differentiable if all its derivatives up to the nt⁢hn^{th} exist everywhere on DD, and are continuous. We denote the set of such functions by Cn⁢(D)C^{n}(D) and say that “ff is CnC^{n}” if f∈Cn⁢(D)f\in C^{n}(D). The function ff is smooth if it is CnC^{n} for all n∈ℤ+n\in{\mathbb{Z}}^{+}. The set of smooth functions on DD is denoted C∞⁢(D)C^{\infty}(D). For completeness, we denote the set of continuous functions D→ℝD\rightarrow{\mathbb{R}} by C0⁢(D)C^{0}(D).

An interesting interpretation of the Mean Value Theorem can be given as follows. Let I⊆ℝI\subseteq{\mathbb{R}} be an open interval, f:I→ℝf:I\rightarrow{\mathbb{R}} be differentiable, and choose and fix some a∈Ia\in I. Then for any other x∈Ix\in I, we may apply the Mean Value Theorem to f:[a,x]→ℝf:[a,x]\rightarrow{\mathbb{R}} (if x>ax>a) or f:[x,a]→ℝf:[x,a]\rightarrow{\mathbb{R}} (if x<ax<a) to deduce that there exists cc between aa and xx such that

f⁢(x)=f⁢(a)+f′⁢(c)⁢(x−a).f(x)=f(a)+f^{\prime}(c)(x-a).

Note that the same equation holds trivially in the case where x=ax=a (with c=x=ac=x=a). So, for xx close to aa (meaning that |x−a||x-a| is small) we can approximate ff by the constant f⁢(a)f(a), and the error in this approximation is of size |f′⁢(c)|⁢|x−a||f^{\prime}(c)||x-a|, where cc is somewhere between aa and xx (and, of course, depends on xx).

The moral is that differentiability (once) of ff on an open interval allows one to approximate ff by a degree 0 polynomial function (i.e. a constant), with an error controlled by f′f^{\prime}. It turns out that if ff is nn times differentiable one can do better: one can approximate ff by a degree (n−1)(n-1) polynomial function, with an error controlled by f(n)f^{(n)}. To prove this, we will again use the Extended Mean Value Theorem (Theorem 3.16).

Theorem 3.21 (Taylor’s Theorem)

Let I⊆ℝI\subseteq{\mathbb{R}} be an interval, f:I→ℝf:I\rightarrow{\mathbb{R}} be (n+1)(n+1) times differentiable, and a,x∈Ia,x\in I. Then there exists cc between aa and xx such that

f⁢(x)\displaystyle f(x) =\displaystyle= f⁢(a)+f′⁢(a)⁢(x−a)+f′′⁢(a)2!⁢(x−a)2+⋯+f(n)⁢(a)n!⁢(x−a)n\displaystyle f(a)+f^{\prime}(a)(x-a)+\frac{f^{\prime\prime}(a)}{2!}(x-a)^{2}+% \cdots+\frac{f^{(n)}(a)}{n!}(x-a)^{n}
+f(n+1)⁢(c)(n+1)!⁢(x−a)n+1.\displaystyle\qquad\qquad+\frac{f^{(n+1)}(c)}{(n+1)!}(x-a)^{n+1}.
Proof.

First note that the claimed equation holds trivially (with c=ac=a) in the case where x=ax=a. Choose and fix a,x∈Ia,x\in I, x≠ax\neq a, and consider the function F:I→ℝF:I\rightarrow{\mathbb{R}},

F⁢(t)=f⁢(x)−f⁢(t)−f′⁢(t)⁢(x−t)−f′′⁢(t)2!⁢(x−t)2−⋯−f(n)⁢(t)n!⁢(x−t)n.F(t)=f(x)-f(t)-f^{\prime}(t)(x-t)-\frac{f^{\prime\prime}(t)}{2!}(x-t)^{2}-% \cdots-\frac{f^{(n)}(t)}{n!}(x-t)^{n}.

Then the claim to be proved is that there exists cc between aa and xx such that

F⁢(a)=f(n+1)⁢(c)(n+1)!⁢(x−a)n+1.F(a)=\frac{f^{(n+1)}(c)}{(n+1)!}(x-a)^{n+1}.

Now, by the Product Rule, FF is differentiable, and, as is easily verified,

F′⁢(t)=−f(n+1)⁢(t)n!⁢(x−t)n.F^{\prime}(t)=-\frac{f^{(n+1)}(t)}{n!}(x-t)^{n}.

Let G⁢(t)=(x−t)n+1G(t)=(x-t)^{n+1}. Then F,GF,G satisfy the hypotheses of Theorem 3.16 on the interval with endpoints a,xa,x (note that G′⁢(t)=0G^{\prime}(t)=0 only if t=xt=x), so there exists cc between aa and xx such that

F′⁢(c)G′⁢(c)\displaystyle\frac{F^{\prime}(c)}{G^{\prime}(c)} =\displaystyle= F⁢(x)−F⁢(a)G⁢(x)−G⁢(a)=F⁢(a)G⁢(a)\displaystyle\frac{F(x)-F(a)}{G(x)-G(a)}=\frac{F(a)}{G(a)}
i.e. −f(n+1)⁢(c)⁢(x−c)nn!⁢[−(n+1)⁢(x−c)n]\displaystyle\mbox{i.e.\ }\qquad\frac{-f^{(n+1)}(c)(x-c)^{n}}{n![-(n+1)(x-c)^{% n}]} =\displaystyle= F⁢(a)(x−a)n+1,\displaystyle\frac{F(a)}{(x-a)^{n+1}},

whence the claimed expression for F⁢(a)F(a) immediately follows. ∎

The polynomial

pn⁢(x)=f⁢(a)+f′⁢(a)⁢(x−a)+f′′⁢(a)2!⁢(x−a)2+⋯+f(n)⁢(a)n!⁢(x−a)np_{n}(x)=f(a)+f^{\prime}(a)(x-a)+\frac{f^{\prime\prime}(a)}{2!}(x-a)^{2}+% \cdots+\frac{f^{(n)}(a)}{n!}(x-a)^{n}

is often called the nth{n^{\rm th}} Taylor approximant of the function ff about aa, and

f(n+1)⁢(c)(n+1)!⁢(x−a)n+1\frac{f^{(n+1)}(c)}{(n+1)!}(x-a)^{n+1}

is called the remainder.

You already knew how to prove that the Alternating Harmonic Series converges: you can apply the Alternating Series Test. But this test, like all the others (Comparison Test, Ratio Test etc), only tells you that the series converges, it doesn’t tell you anything about its limit. Taylor’s Theorem allowed us to prove convergence and compute the limit. Note that the argument made crucial use of the remainder term: a Taylor expansion without remainder isn’t much good for proving anything.

Summary

  • •
    ​

    The Interior Extremum Theorem: If f:(a,b)→ℝf:(a,b)\rightarrow{\mathbb{R}} is differentiable, and has an extremum (maximum or minimum) at c∈(a,b)c\in(a,b), then f′⁢(c)=0f^{\prime}(c)=0.

  • •
    ​

    The Mean Value Theorem: If f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} is continuous, and differentiable on (a,b)(a,b), then there exists c∈(a,b)c\in(a,b) such that

    f′⁢(c)=f⁢(b)−f⁢(a)b−a.f^{\prime}(c)=\frac{f(b)-f(a)}{b-a}.
  • •
    ​

    Let f:I→ℝf:I\rightarrow{\mathbb{R}} be differentiable, where I⊆ℝI\subseteq{\mathbb{R}} is an interval. We can use the Mean Value Theorem to prove that

    • ∙\bullet
      ​

      If f′⁢(x)=0f^{\prime}(x)=0 for all x∈Ix\in I then ff is constant.

    • ∙\bullet
      ​

      If f′⁢(x)>0f^{\prime}(x)>0 for all x∈Ix\in I then ff is strictly increasing.

    • ∙\bullet
      ​

      If f′⁢(x)<0f^{\prime}(x)<0 for all x∈Ix\in I then ff is strictly decreasing.

    We often use the first of these when we solve ordinary differential equations.

  • •
    ​

    The derivative f′f^{\prime} of a differentiable function may fail to be continuous. However…

  • •
    ​

    Darboux’s Theorem: Let f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} be differentiable and kk be any number between f′⁢(a)f^{\prime}(a) and f′⁢(b)f^{\prime}(b). Then there exists c∈[a,b]c\in[a,b] such that f′⁢(c)=kf^{\prime}(c)=k.

  • •
    ​

    The Extended Mean Value Theorem: If f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} and g:[a,b]→ℝg:[a,b]\rightarrow{\mathbb{R}} are continuous, and differentiable on (a,b)(a,b), and g′⁢(x)≠0g^{\prime}(x)\neq 0 for all x∈(a,b)x\in(a,b), then there exists c∈(a,b)c\in(a,b) such that

    f′⁢(c)g′⁢(c)=f⁢(b)−f⁢(a)g⁢(b)−g⁢(a).\frac{f^{\prime}(c)}{g^{\prime}(c)}=\frac{f(b)-f(a)}{g(b)-g(a)}.
  • •
    ​

    We can use the Extended Mean Value Theorem to prove Taylor’s Theorem.

  • •
    ​

    Taylor’s Theorem: If ff is (n+1)(n+1) times differentiable on some open interval II and a,x∈Ia,x\in I, then there exists cc between aa and xx such that

    f⁢(x)=f⁢(a)+f′⁢(a)1!⁢(x−a)+⋯+f(n)⁢(a)n!⁢(x−a)n+f(n+1)⁢(c)(n+1)!⁢(x−a)n+1.f(x)=f(a)+\frac{f^{\prime}(a)}{1!}(x-a)+\cdots+\frac{f^{(n)}(a)}{n!}(x-a)^{n}+% \frac{f^{(n+1)}(c)}{(n+1)!}(x-a)^{n+1}.
  • •
    ​

    Taylor’s Theorem allows us to approximate (n+1)(n+1) times differentiable functions by a polynomial of degree nn, with an error which is controlled by f(n+1)f^{(n+1)}.
