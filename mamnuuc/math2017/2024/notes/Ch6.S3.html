Uniform convergence and calculus

6.3 Uniform convergence and calculus

We began this chapter by considering (see Example 6.2) a sequence of continuous, and hence Riemann integrable, functions fn:[0,1]→ℝf_{n}:[0,1]\rightarrow{\mathbb{R}} which converge pointwise to the constant function f=0f=0, but whose integrals

∫01fn↛∫01f=0.\int_{0}^{1}f_{n}\nrightarrow\int_{0}^{1}f=0.

So pointwise convergence is insufficient for us to justify swapping the order of a limit (as n→∞n\rightarrow\infty) and an integral. However:

Theorem 6.16

Let fn:[a,b]→ℝf_{n}:[a,b]\rightarrow{\mathbb{R}} be a sequence of continuous functions converging uniformly to f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}}. Then

∫abfn→∫abf.\int_{a}^{b}f_{n}\rightarrow\int_{a}^{b}f.
Proof.

First note that ff is necessarily continuous (Theorem 6.14) and hence Riemann integrable (Theorem 4.21), so the proposed limit ∫abf\int_{a}^{b}f certainly exists. Now, for all nn,

0≤|∫abfn−∫abf|\displaystyle 0\leq\left|\int_{a}^{b}f_{n}-\int_{a}^{b}f\right| =\displaystyle= |∫ab(fn−f)|(Theorem 4.24)\displaystyle\left|\int_{a}^{b}(f_{n}-f)\right|\qquad\mbox{(Theorem \ref{% intlin})}
≤\displaystyle\leq ∫ab|fn−f|(Theorem 4.26)\displaystyle\int_{a}^{b}|f_{n}-f|\qquad\mbox{(Theorem \ref{int-triangle})}
≤\displaystyle\leq (b−a)⁢‖fn−f‖(Theorem 4.25).\displaystyle(b-a)\|f_{n}-f\|\qquad\mbox{(Theorem \ref{aliang}).}

Hence, ∫abfn→∫abf\int_{a}^{b}f_{n}\rightarrow\int_{a}^{b}f by the Squeeze Rule. ∎

We deduce from this that the sequence fn:[0,1]→ℝ,f_{n}:[0,1]\rightarrow{\mathbb{R}}, of “triangle” functions defined in Example 6.2 does not converge uniformly. Note that its pointwise limit (the constant function 0) is continuous, so we couldn’t deduce this from Theorem 6.14.

Example 6.17

Claim: limn→∞∫0πsin⁡(x+x2n)⁢𝑑x=2.\displaystyle{\lim_{n\rightarrow\infty}\int_{0}^{\pi}\sin(x+\frac{x^{2}}{n})dx% =2.}

Proof.

Let fn:[0,π]→ℝf_{n}:[0,\pi]\rightarrow{\mathbb{R}}, fn⁢(x)=sin⁡(x+x2/n)f_{n}(x)=\sin(x+x^{2}/n). Hopefully it’s clear that fnf_{n} converges pointwise to the function sin\sin since, for each fixed x∈[0,π]x\in[0,\pi], x+x2/n→xx+x^{2}/n\rightarrow x and so fn⁢(x)→sin⁡xf_{n}(x)\rightarrow\sin x (assuming, that is, that sin\sin is continuous – which we actually haven’t proved yet. It’s coming in the next chapter). If we can show that fnf_{n} converges to sin\sin uniformly, the claim immediately follows, since ∫0πsin=2\int_{0}^{\pi}\sin=2.

So, it remains to show that fn→sinf_{n}\rightarrow\sin uniformly. Fix nn and xx, and apply the Mean Value Theorem to the function sin\sin on the interval [x,x+x2/n][x,x+x^{2}/n]: there exists c∈[x,x+x2/n]c\in[x,x+x^{2}/n] such that

fn⁢(x)−sin⁡x(x+x2/n)−x=cos⁡c.\displaystyle\frac{f_{n}(x)-\sin x}{(x+x^{2}/n)-x}=\cos c.

Hence

|fn⁢(x)−sin⁡x|\displaystyle|f_{n}(x)-\sin x| =\displaystyle= |cos⁡c|⁢|x2/n|≤x2n.\displaystyle|\cos c||x^{2}/n|\leq\frac{x^{2}}{n}.

This inequality holds for all xx, and hence

‖fn−sin‖\displaystyle\|f_{n}-\sin\| =\displaystyle= sup{|fn(x)−sinx|: 0≤x≤π}≤π2n.\displaystyle\sup\{|f_{n}(x)-\sin x|\>:\>0\leq x\leq\pi\}\leq\frac{\pi^{2}}{n}.

So fnf_{n} converges to sin:[0,π]→ℝ\sin:[0,\pi]\rightarrow{\mathbb{R}} uniformly (by the Squeeze Rule). Hence, by Theorem 6.16,

∫0πfn\displaystyle\int_{0}^{\pi}f_{n} →\displaystyle\rightarrow ∫0πsin⁡x⁢d⁢x\displaystyle\int_{0}^{\pi}\sin x\,dx
=\displaystyle= [−cos⁡x]0π=2\displaystyle\left[-\cos x\right]_{0}^{\pi}=2

by the Fundamental Theorem of the Calculus (version 2). ∎

Note we have no hope of computing the integrals ∫0πfn\int_{0}^{\pi}f_{n} exactly (I invite you to try to write down an antiderivative for fnf_{n}), so this limit can’t be computed directly.

Our next goal is to prove an analogue of Theorem 6.16 for derivatives. Under what circumstances can we be sure that a sequence fnf_{n} of differentiable functions converges to a differentiable function? The functions fnf_{n} in the following theorem are assumed to be continuously differentiable, meaning that each function fn:[a,b]→ℝf_{n}:[a,b]\rightarrow{\mathbb{R}} is differentiable and that its derivative fn′:[a,b]→ℝf_{n}^{\prime}:[a,b]\rightarrow{\mathbb{R}} is continuous. (Recall that every differentiable function is continuous, but its derivative isn’t necessarily continuous: see Example 3.14 for a counterexample.)

Theorem 6.19

Let fn:[a,b]→ℝf_{n}:[a,b]\rightarrow{\mathbb{R}} be a sequence of continuously differentiable functions which converge pointwise to f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}}, and whose derivatives fn′:[a,b]→ℝf_{n}^{\prime}:[a,b]\rightarrow{\mathbb{R}} converge uniformly to g:[a,b]→ℝg:[a,b]\rightarrow{\mathbb{R}}. Then ff is continuously differentiable and f′=gf^{\prime}=g.

Proof.

For each n∈ℤ+n\in{\mathbb{Z}}^{+} let Fn:[a,b]→ℝF_{n}:[a,b]\rightarrow{\mathbb{R}} be defined by

Fn⁢(x)=∫axfn′.F_{n}(x)=\int_{a}^{x}f_{n}^{\prime}.

Note that this exists since fn′f_{n}^{\prime} is, by assumption, continuous. By version 2 of the Fundamental Theorem of the Calculus,

Fn⁢(x)=fn⁢(x)−fn⁢(a).F_{n}(x)=f_{n}(x)-f_{n}(a).

By Theorem 6.16, for each fixed xx,

fn⁢(x)−fn⁢(a)=Fn⁢(x)→∫axg,f_{n}(x)-f_{n}(a)=F_{n}(x)\rightarrow\int_{a}^{x}g,

since fn′f_{n}^{\prime} converges uniformly to gg on [a,x][a,x]. But fnf_{n} converges pointwise to ff, so fn⁢(x)→f⁢(x)f_{n}(x)\rightarrow f(x) and fn⁢(a)→f⁢(a)f_{n}(a)\rightarrow f(a). Hence, fn⁢(x)−fn⁢(a)→f⁢(x)−f⁢(a)f_{n}(x)-f_{n}(a)\rightarrow f(x)-f(a) (by the Algebra of Limits). But limits are unique, so

f⁢(x)−f⁢(a)=∫axg.f(x)-f(a)=\int_{a}^{x}g.

Hence, by version 1 of the Fundamental Theorem of the Calculus, ff is differentiable and f′⁢(x)=g⁢(x)f^{\prime}(x)=g(x). ∎

At the moment, it’s hard to see why Theorem 6.19 is useful, because it’s hard to imagine a situation where we can check that fn′f_{n}^{\prime} converges uniformly to a continuous function gg without already knowing that the limit ff of fnf_{n} is differentiable. For example, if we apply this Theorem to the sequence of functions in Exercise 6.18 we can deduce (at some effort) that the constant function f⁢(x)=1f(x)=1 is differentiable and has derivative 0 – hardly a startling revelation!

We will see in the next chapter that Theorem 6.19 can be extremely powerful when applied to the sequence of partial sums of a convergent power series. But before we can explore that application, we need to develop a way to prove that a sequence fnf_{n} of functions converges uniformly to something ff even when we can’t really say explicitly what ff is (except tautologically: f⁢(x)=limn→∞fn⁢(x)f(x)=\lim_{n\rightarrow\infty}f_{n}(x)).
