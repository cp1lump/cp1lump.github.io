Pointwise convergence versus uniform convergence

6.2 Pointwise convergence versus uniform convergence

Given a sequence of functions, fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}}, where D⊆ℝD\subseteq{\mathbb{R}} is some fixed set, what does it mean to say that (fn)(f_{n}) converges to f:D→ℝf:D\rightarrow{\mathbb{R}}? The obvious answer is:

Definition 6.4

A sequence of functions fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} converges pointwise to a function f:D→ℝf:D\rightarrow{\mathbb{R}} if, for each fixed x∈Dx\in D, the real sequence (fn⁢(x))(f_{n}(x)) converges (in the sense of Definition 1.1) to the real number f⁢(x)f(x).

This is the sense in which the sequence of functions fn:[0,1]→ℝf_{n}:[0,1]\rightarrow{\mathbb{R}} in Example 6.2 converges to the function g⁢(x)=0g(x)=0. Here’s another instructive example:

Example 6.5

fn⁢(x)=xn→0.f_{n}(x)=x^{n}\rightarrow 0.

If x=1x=1, however,

fn⁢(1)=1n=1→1.f_{n}(1)=1^{n}=1\rightarrow 1.

f⁢(x)={0,0≤x<1,1,x=1.f(x)=\left\{\begin{array}[]{cc}0,&{0\leq x<1},\\ 1,&x=1.\end{array}\right. (6.1)

This illustrates a key weakness of pointwise convergence: it doesn’t preserve continuity. Nonetheless, pointwise limits do have some nice properties.

Proposition 6.7

If (fn)(f_{n}) converges pointwise, its limit is unique.

Proof.

Assume, towards a contradiction, that (fn)(f_{n}) converges to both ff and gg, where f≠gf\neq g. Then, by assumption, there exists c∈Dc\in D such that f⁢(c)≠g⁢(c)f(c)\neq g(c). But fn⁢(c)→f⁢(c)f_{n}(c)\rightarrow f(c) and fn⁢(c)→g⁢(c)f_{n}(c)\rightarrow g(c), which contradicts the uniqueness of the limit of a convergent real sequence. ∎

A much stronger, and more useful, notion of convergence is that of uniform convergence. To define this, it’s convenient to make a preliminary definition.

Definition 6.9

The sup norm of a bounded function f:D→ℝf:D\rightarrow{\mathbb{R}} is

∥f∥:=sup{|f(x)|:x∈D}.\|f\|:=\sup\{|f(x)|\>:\>x\in D\}.

In the case where f,g:[a,b]→ℝf,g:[a,b]\rightarrow{\mathbb{R}} are continuous, we can give ‖f−g‖\|f-g\| a geometric interpretation: it’s the maximum separation between (x,f⁢(x))(x,f(x)) and (x,g⁢(x))(x,g(x)) as xx ranges over [a,b][a,b]. We may think of this as a “distance” between the functions ff and gg.

So ‖f−g‖\|f-g\| defines a sort of “distance” between bounded functions, in the same way that |a−b||a-b| defines a distance between real numbers aa and bb. This line of thought leads to the main definition of this chapter:

Definition 6.11

A sequence of bounded functions fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} converges uniformly to a function f:D→ℝf:D\rightarrow{\mathbb{R}} if the real sequence ‖fn−f‖→0\|f_{n}-f\|\rightarrow 0.

So pointwise convergence certainly does not imply uniform convergence. However, the converse holds:

Theorem 6.12

If a sequence fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} converges uniformly to f:D→ℝf:D\rightarrow{\mathbb{R}}, it converges pointwise to ff.

Proof.

Assume the sequence of functions fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} converges uniformly to f:D→ℝf:D\rightarrow{\mathbb{R}}. Then, for each fixed x∈Dx\in D,

0≤|fn(x)−f(x)|≤sup{|fn(y)−f(y)|:y∈D}=∥fn−f∥→0,0\leq|f_{n}(x)-f(x)|\leq\sup\{|f_{n}(y)-f(y)|\>:\>y\in D\}=\|f_{n}-f\|% \rightarrow 0,

so |fn⁢(x)−f⁢(x)|→0|f_{n}(x)-f(x)|\rightarrow 0 by the Squeeze Rule. Hence fn⁢(x)→f⁢(x)f_{n}(x)\rightarrow f(x), Since this holds for each x∈Dx\in D, fnf_{n} converges pointwise to ff. ∎

Corollary 6.13

If (fn)(f_{n}) converges uniformly, its limit is unique.

Proof.

Assume fnf_{n} converges uniformly to both ff and gg. Then fnf_{n} converges pointwise to ff and fnf_{n} converges pointwise to gg (Theorem 6.12). Hence f=gf=g (Proposition 6.7). ∎

We have seen that pointwise convergence doesn’t always preserve continuity. That is, a sequence of continuous functions can converge pointwise to a discontinuous function (Example 6.5). A key advantage of uniform convergence is that it does preserve continuity.
