Darboux’s Theorem

3.3 Darboux’s Theorem

If a function f:D→ℝf:D\rightarrow{\mathbb{R}} is differentiable, it is continuous (Proposition 2.8), and its derivative defines a function f′:D→ℝf^{\prime}:D\rightarrow{\mathbb{R}}. Note that it does not follow that f′:D→ℝf^{\prime}:D\rightarrow{\mathbb{R}} is continuous! In fact, Example 3.14 provides a counterexample: we saw that this function is differentiable everywhere and has f′⁢(0)=1f^{\prime}(0)=1. We also observed that, for any positive integer mm, f′⁢(xm)=−1f^{\prime}(x_{m})=-1, where xm=1/(2⁢π⁢m)x_{m}=1/(2\pi m). Clearly xm→0x_{m}\rightarrow 0, but f′⁢(xm)↛f′⁢(0)=1f^{\prime}(x_{m})\nrightarrow f^{\prime}(0)=1, so f′f^{\prime} is discontinuous at 0.

So, the derivative of a differentiable function f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}}, may fail to be a continuous function. Nonetheless, f′:[a,b]→ℝf^{\prime}:[a,b]\rightarrow{\mathbb{R}} is guaranteed to satisfy one of the key properties of continuous functions on [a,b][a,b]: it has the intermediate value property. That is, given any number kk between f′⁢(a)f^{\prime}(a) and f′⁢(b)f^{\prime}(b), there exists c∈[a,b]c\in[a,b] such that f′⁢(c)=kf^{\prime}(c)=k. This holds even though, in general, f′f^{\prime} may fail to be a continuous function!

Theorem 3.15 (Darboux’s Theorem)

Let ff be differentiable on [a,b][a,b] and kk be any number between f′⁢(a)f^{\prime}(a) and f′⁢(b)f^{\prime}(b). Then there exists c∈[a,b]c\in[a,b] such that f′⁢(c)=kf^{\prime}(c)=k.

Proof.

If k=f′⁢(a)k=f^{\prime}(a) we may take c=ac=a. If k=f′⁢(b)k=f^{\prime}(b) we may take c=bc=b. It remains to consider the cases (i) f′⁢(a)<k<f′⁢(b)f^{\prime}(a)<k<f^{\prime}(b) and (ii) f′⁢(b)<k<f′⁢(a)f^{\prime}(b)<k<f^{\prime}(a). In case (i), let g⁢(x)=k⁢x−f⁢(x)g(x)=kx-f(x). This is differentiable, hence continuous, on [a,b][a,b] so, by the Extreme Value Theorem, it attains a maximum somewhere on [a,b][a,b]. Now g′⁢(a)=k−f′⁢(a)>0g^{\prime}(a)=k-f^{\prime}(a)>0 so the maximum can’t be at aa (Proposition 3.4). Similarly, g′⁢(b)=k−f′⁢(b)<0g^{\prime}(b)=k-f^{\prime}(b)<0 so the maximum can’t be at bb (Proposition 3.4 again). Hence, the maximum is attained at some interior point c∈(a,b)c\in(a,b). But then g′⁢(c)=0g^{\prime}(c)=0 by the Interior Extremum Theorem, and g′⁢(c)=k−f′⁢(c)g^{\prime}(c)=k-f^{\prime}(c), which completes case (i). In case (ii), let f~⁢(x)=−f⁢(x)\widetilde{f}(x)=-f(x). Then f~\widetilde{f} is differentiable on [a,b][a,b] and f~′⁢(a)<−k<f~′⁢(b)\widetilde{f}^{\prime}(a)<-k<\widetilde{f}^{\prime}(b) so, by part (i), there exists c∈(a,b)c\in(a,b) such that f~′⁢(c)=−k\widetilde{f}^{\prime}(c)=-k. But then f′⁢(c)=kf^{\prime}(c)=k. ∎
