Swapping limits

6.1 Swapping limits

There are many occasions in calculus where we must compute a double limit. For example, say we wish to compute the derivative of the function f=expf=\exp. We know that f⁢(x)f(x) is the limit of a convergent series, that is,

f⁢(x)=limk→∞∑n=0kxnn!.f(x)=\lim_{k\rightarrow\infty}\sum_{n=0}^{k}\frac{x^{n}}{n!}.

Its derivative at a∈ℝa\in{\mathbb{R}}, therefore, is

f′⁢(a)=limx→af⁢(x)−f⁢(a)x−a=limx→alimk→∞1x−a⁢∑n=0kxn−ann!,f^{\prime}(a)=\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}=\lim_{x\rightarrow a}% \lim_{k\rightarrow\infty}\frac{1}{x-a}\sum_{n=0}^{k}\frac{x^{n}-a^{n}}{n!},

assuming this exists. Wouldn’t it be nice if we could swap the order of these limits? Since

limx→axn−anx−a=n⁢an−1\lim_{x\rightarrow a}\frac{x^{n}-a^{n}}{x-a}=na^{n-1}

we would then deduce that

f′⁢(a)=limk→∞∑n=0k1n!⁢n⁢an−1=limk→∞∑m=0k−1amm!=f⁢(a),f^{\prime}(a)=\lim_{k\rightarrow\infty}\sum_{n=0}^{k}\frac{1}{n!}na^{n-1}=\lim% _{k\rightarrow\infty}\sum_{m=0}^{k-1}\frac{a^{m}}{m!}=f(a),

establishing a fundamental property of the exponential function (it’s differentiable, and equals its own derivative).

Sadly, this argument is fatally flawed because, in general, limits do not commute.

Example 6.1

Consider the function

f:ℝ2\{(0,0)}→ℝ,f⁢(x,y)=x2−y2x2+y2.f:{\mathbb{R}}^{2}{\backslash}\{(0,0)\}\rightarrow{\mathbb{R}},\qquad f(x,y)=% \frac{x^{2}-y^{2}}{x^{2}+y^{2}}.

limy→0f⁢(x,y)=limy→0x2−y2x2+y2=x2x2=1.\lim_{y\rightarrow 0}f(x,y)=\lim_{y\rightarrow 0}\frac{x^{2}-y^{2}}{x^{2}+y^{2% }}=\frac{x^{2}}{x^{2}}=1.

limx→0(limy→0f⁢(x,y))=limx→01=1.\lim_{x\rightarrow 0}\left(\lim_{y\rightarrow 0}f(x,y)\right)=\lim_{x% \rightarrow 0}1=1.

limx→0f⁢(x,y)=limx→0x2−y2x2+y2=−y2y2=−1.\lim_{x\rightarrow 0}f(x,y)=\lim_{x\rightarrow 0}\frac{x^{2}-y^{2}}{x^{2}+y^{2% }}=\frac{-y^{2}}{y^{2}}=-1.

limy→0(limx→0f⁢(x,y))=limy→0−1=−1.\lim_{y\rightarrow 0}\left(\lim_{x\rightarrow 0}f(x,y)\right)=\lim_{y% \rightarrow 0}-1=-1.

Clearly, the order in which we evaluate the limits matters! □\Box
