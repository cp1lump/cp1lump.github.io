Properties of the exponential function

7.5 Properties of the exponential function

In this section we will prove that the function exp:ℝ→ℝ\exp:{\mathbb{R}}\rightarrow{\mathbb{R}}, defined as a power series, has all the properties you’re familiar with. We begin with the following fundamental identity.

Lemma 7.27

For all x,y∈ℝx,y\in{\mathbb{R}}, exp⁡(x+y)=exp⁡(x)⁢exp⁡(y)\exp(x+y)=\exp(x)\exp(y).

Proof.

For each fixed number b∈ℝb\in{\mathbb{R}}, define the function f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}},

f⁢(x)=exp⁡(x)⁢exp⁡(b−x).f(x)=\exp(x)\exp(b-x).

This is a product of differentiable functions, so is differentiable, and by the Product and Chain Rules, and Proposition 7.24,

f′⁢(x)=exp′⁡(x)⁢exp⁡(b−x)−exp⁡(x)⁢exp′⁡(b−x)=0.f^{\prime}(x)=\exp^{\prime}(x)\exp(b-x)-\exp(x)\exp^{\prime}(b-x)=0.

Since f′⁢(x)=0f^{\prime}(x)=0 for all xx in the interval ℝ{\mathbb{R}}, it follows that ff is constant (Proposition 3.11). Hence, for all x∈ℝx\in{\mathbb{R}} and any b∈ℝb\in{\mathbb{R}}, f⁢(x)=f⁢(b)f(x)=f(b), that is,

exp⁡(x)⁢exp⁡(b−x)=exp⁡(b).\exp(x)\exp(b-x)=\exp(b).

Applying this in the case where b=x+yb=x+y establishes the claim. ∎

Remark Using similar arguments, it’s quite straightforward to prove trigonometric identities such as

cos2⁡x+sin2⁡x=1.\cos^{2}x+\sin^{2}x=1.

Just define f⁢(x)=cos2⁡x+sin2⁡xf(x)=\cos^{2}x+\sin^{2}x and show that f′⁢(x)=0f^{\prime}(x)=0 for all xx, whence ff is constant (Proposition 3.11), and observe that f⁢(0)=1f(0)=1.

cos⁡(x+y)\displaystyle\cos(x+y) =\displaystyle= cos⁡x⁢cos⁡y−sin⁡x⁢sin⁡y,\displaystyle\cos x\cos y-\sin x\sin y,
sin⁡(x+y)\displaystyle\sin(x+y) =\displaystyle= sin⁡x⁢cos⁡y+cos⁡x⁢sin⁡y.\displaystyle\sin x\cos y+\cos x\sin y.

Lemma 7.27 explains why exp⁡(x)\exp(x) is often denoted exe^{x} and thought of as the constant e=exp⁡(1)=∑n=0∞1n!e=\exp(1)=\sum_{n=0}^{\infty}\frac{1}{n!} “raised to the power xx”. The point is that, written in this alternative notation, Lemma 7.27 looks like one of the standard algebraic rules of integer exponents:

ex+y=ex×eyin analogy withan+m=an×am.e^{x+y}=e^{x}\times e^{y}\qquad\mbox{in analogy with}\qquad a^{n+m}=a^{n}% \times a^{m}.

This analogy, and the associated notation, are a useful mnemonic, but it’s important to realize that they are just that, an analogy. We don’t literally define exp⁡(x)\exp(x) to be the irrational number ee “raised to the power xx”, for obvious reasons: what on earth does it mean to raise ee to the power 2\sqrt{2}, for example? In fact, the logic works in the opposite direction. Having proved that the function exp\exp, defined as a power series, satisfies a rule analogous to the behaviour of integer (and rational) exponents, we define exe^{x}, for any exponent xx (rational or irrational) to be exp⁡(x)\exp(x).

Proposition 7.28

For all x∈ℝx\in{\mathbb{R}}, exp⁡(x)>0\exp(x)>0.

Proof.

We first note that, for all x∈ℝx\in{\mathbb{R}}, exp⁡(x)≠0\exp(x)\neq 0 since, if there exists x∈ℝx\in{\mathbb{R}} such that exp⁡(x)=0\exp(x)=0, then Lemma 7.27 in the case y=−xy=-x implies that

1=exp⁡(0)=0×exp⁡(−x)=0.1=\exp(0)=0\times\exp(-x)=0.

Now exp⁡(0)=1>0\exp(0)=1>0, and exp\exp is differentiable, hence continuous, so it follows from the Intermediate Value Theorem that exp⁡(x)>0\exp(x)>0 for all x∈ℝx\in{\mathbb{R}} (if exp⁡(x)<0\exp(x)<0 then there exists cc between xx and 0 such that exp⁡(c)=0\exp(c)=0). ∎

So exp:ℝ→(0,∞)\exp:{\mathbb{R}}\rightarrow(0,\infty). I claim that exp\exp is bijective and that ln:(0,∞)→ℝ\ln:(0,\infty)\rightarrow{\mathbb{R}} is its inverse function. Recall (Definition 5.8) that

ln:(0,∞)→ℝ,ln⁡x:=∫1x1t⁢𝑑t.\ln:(0,\infty)\rightarrow{\mathbb{R}},\qquad\ln x:=\int_{1}^{x}\frac{1}{t}dt.
Proposition 7.29

exp:ℝ→(0,∞)\exp:{\mathbb{R}}\rightarrow(0,\infty) is bijective and ln:(0,∞)→ℝ\ln:(0,\infty)\rightarrow{\mathbb{R}} is its inverse function.

Proof.

We must show that, for all x∈ℝx\in{\mathbb{R}}, ln⁡(exp⁡(x))=x\ln(\exp(x))=x, and for all y∈(0,∞)y\in(0,\infty), exp⁡(ln⁡y)=y\exp(\ln y)=y. Consider the function

f:ℝ→ℝ,f⁢(x)=ln⁡(exp⁡(x))−x.f:{\mathbb{R}}\rightarrow{\mathbb{R}},\qquad f(x)=\ln(\exp(x))-x.

By the Chain Rule,

f′⁢(x)=exp⁡(x)exp⁡(x)−1=0f^{\prime}(x)=\frac{\exp(x)}{\exp(x)}-1=0

for all xx, so ff is constant (Corollary 3.11). Hence, for all xx, f⁢(x)=f⁢(0)=ln⁡1−0=0f(x)=f(0)=\ln 1-0=0, that is, ln⁡(exp⁡(x))=x\ln(\exp(x))=x. Now, for all y∈(0,∞)y\in(0,\infty),

ln⁡(exp⁡(ln⁡y))=ln⁡y,\ln(\exp(\ln y))=\ln y,

as we have just shown. But ln\ln is injective, so if ln⁡z=ln⁡y\ln z=\ln y then z=yz=y. Hence exp⁡(ln⁡y)=y\exp(\ln y)=y, as was to be shown.

It follows immediately that exp\exp is surjective:

∀y∈(0,∞),exp⁡(ln⁡(y))=y,\forall y\in(0,\infty),\quad\exp(\ln(y))=y,

and injective:

exp⁡(x)=exp⁡(x′)⇒ln⁡(exp⁡(x))=ln⁡(exp⁡(x′))⇒x=x′.\exp(x)=\exp(x^{\prime})\quad\Rightarrow\quad\ln(\exp(x))=\ln(\exp(x^{\prime})% )\quad\Rightarrow\quad x=x^{\prime}.

∎

So, for each positive number yy, ln⁡y\ln y is the real number whose exponential is yy. Recall that Euler’s number is

e=exp⁡(1)=∑n=0∞1n!.e=\exp(1)=\sum_{n=0}^{\infty}\frac{1}{n!}.

It follows that ln⁡e=1\ln e=1. There is an alternative way to define ee, as the limit of the sequence (1+1n)n(1+\frac{1}{n})^{n}. You are now in a position to prove this.

Proposition 7.29 implies that, for all a>0a>0, ln⁡a\ln a is the unique real number yy such that exp⁡(y)=a\exp(y)=a. This motivates the following definition:

Definition 7.31

Let a>0a>0 and x∈ℝx\in{\mathbb{R}}. Then aa to the power xx is

ax=exp⁡(x⁢ln⁡a).a^{x}=\exp(x\ln a).

It follows from Proposition 5.10 that this definition of axa^{x} coincides with the more obvious definition of axa^{x} in the case where xx is an integer.

So now we can make sense of the function f⁢(x)=xrf(x)=x^{r} for all x>0x>0 and any constant r∈ℝr\in{\mathbb{R}} (even if rr is irrational!). It is not hard to prove that f:(0,∞)→ℝf:(0,\infty)\rightarrow{\mathbb{R}} is differentiable, and has the derivative we expect.

Proposition 7.32

Let f:(0,∞)→ℝf:(0,\infty)\rightarrow{\mathbb{R}} be the function f⁢(x)=xrf(x)=x^{r}, where rr is any real constant. Then ff is differentiable, and for all x∈(0,∞)x\in(0,\infty),

f′⁢(x)=r⁢xr−1.f^{\prime}(x)=rx^{r-1}.
