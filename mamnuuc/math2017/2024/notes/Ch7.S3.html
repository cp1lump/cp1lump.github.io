7.3 Uniform convergence of series of functions

In the language of chapter 6, Theorem 7.11 establishes that the sequence of polynomial functions

fk⁢(x)=∑n=0kan⁢xnf_{k}(x)=\sum_{n=0}^{k}a_{n}x^{n}

converges pointwise on (−R,R)(-R,R) to some function f⁢(x)f(x), which we denote

f⁢(x)=∑n=0∞an⁢xn.f(x)=\sum_{n=0}^{\infty}a_{n}x^{n}.

As we learned, pointwise convergence is a rather weak property, from which we can’t usually deduce much about the limit function (e.g. it may be discontinuous, even if each function in the sequence is continuous). We can deduce much more about the limit function if we can establish that the convergence is actually uniform. Recall also that we can show that a sequence of functions converges uniformly even if we have no information about its limit function by showing that the sequence is uniformly Cauchy.

In this section we will develop a test which allows us to establish quickly and easily that a series of functions is uniformly Cauchy, and hence uniformly convergent. We will state the test for general series of functions, not just polynomials, because there are many other useful choices, for example, Fourier series

∑n=1∞an⁢sin⁡n⁢x.\sum_{n=1}^{\infty}a_{n}\sin nx.

Returning to our main focus of study, it would be nice if we could use the M test to prove that power series converge uniformly on (−R,R)(-R,R), where RR is the radius of convergence. In fact, that’s a bit too much to hope for, as we can deduce by thinking about the geometric series

∑n=1∞xn.\sum_{n=1}^{\infty}x^{n}.

This has R=1R=1 (as we showed directly in Example 7.9). If it converged uniformly its limit function f:(−1,1)→ℝf:(-1,1)\rightarrow{\mathbb{R}} would perforce be bounded. But we showed that

f⁢(x)=11−xf(x)=\frac{1}{1-x}

which grows unbounded above as xx approaches 11. So we deduce that this series does not converge uniformly on (−1,1)(-1,1). Nil desperandum! We can prove something almost as good as uniform convergence on (−R,R)(-R,R):

Theorem 7.18

Let the power series ∑n=1∞an⁢xn\displaystyle{\sum_{n=1}^{\infty}a_{n}x^{n}} have radius of convergence R>0R>0 and ρ\rho be any constant in (0,R)(0,R). Then the series converges uniformly on [−ρ,ρ][-\rho,\rho].

Proof.

We apply the Weierstrass M Test with gn⁢(x)=an⁢xng_{n}(x)=a_{n}x^{n}, D=[−ρ,ρ]D=[-\rho,\rho] and Mn=|an|⁢ρnM_{n}=|a_{n}|\rho^{n}. Certainly, for all x∈[−ρ,ρ]x\in[-\rho,\rho]

|gn⁢(x)|=|an|⁢|x|n≤|an|⁢ρn=Mn|g_{n}(x)|=|a_{n}||x|^{n}\leq|a_{n}|\rho^{n}=M_{n}

so condition (i) is satisfied. Since |ρ|<R|\rho|<R, the series converges absolutely at x=ρx=\rho, that is

∑n=0∞Mn=∑n=0∞|an|⁢ρn\sum_{n=0}^{\infty}M_{n}=\sum_{n=0}^{\infty}|a_{n}|\rho^{n}

converges, by Theorem 7.11. Hence condition (ii) is satisfied also. So the series converges uniformly on [−ρ,ρ][-\rho,\rho] by the Weierstrass M Test (Theorem 7.16). ∎

Each of the partial sums of a power series is a polynomial, and hence is continuous. It follows from Theorems 6.14 and 7.18 that the limit function

f⁢(x)=∑n=0∞an⁢xnf(x)=\sum_{n=0}^{\infty}a_{n}x^{n}

is continuous on [−ρ,ρ][-\rho,\rho] for every ρ∈(0,R)\rho\in(0,R). But that implies that ff is continuous on (−R,R)(-R,R) since, at every point a∈(−R,R)a\in(-R,R) we can choose ρ=(a+R)/2\rho=(a+R)/2, say (or any other ρ∈(a,R)\rho\in(a,R)). So we immediately deduce:

Corollary 7.19

Let the power series ∑n=1∞an⁢xn\displaystyle{\sum_{n=1}^{\infty}a_{n}x^{n}} have radius of convergence R>0R>0. Then this series converges to a continuous function f:(−R,R)→ℝf:(-R,R)\rightarrow{\mathbb{R}}.

Excellent! But we want (much) more: we’re actually going to prove that the limit function is smooth (that is, infinitely differentiable) on (−R,R)(-R,R).
