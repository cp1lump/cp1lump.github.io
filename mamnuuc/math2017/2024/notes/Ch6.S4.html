Completeness of the set of bounded functions

6.4 Completeness of the set of bounded functions

Recall that a real sequence (an)(a_{n}) is Cauchy if, for each ε>0\varepsilon>0, there exists N∈ℤ+N\in{\mathbb{Z}}^{+} such that, for all n,m≥Nn,m\geq N, |an−am|<ε|a_{n}-a_{m}|<\varepsilon (Definition 1.15), and that this condition turns out to be equivalent to being convergent, that is, (an)(a_{n}) converges if and only if it is Cauchy (Theorem 1.19). This Theorem allows us to prove that a real sequence converges to something A∈ℝA\in{\mathbb{R}} even when we have no idea what AA actually is. The aim of this section is to repeat this trick for uniform convergence of sequences of functions.

For a sequence of functions fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}}, where DD is some fixed subset of ℝ{\mathbb{R}}, we can define what it means to be uniformly Cauchy, in much the same way that we defined uniform convergence (Definition 6.11):

Definition 6.20

A sequence of bounded functions fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} is uniformly Cauchy if, for each ε>0\varepsilon>0, there exists N∈ℤ+N\in{\mathbb{Z}}^{+} such that, for all n,m≥Nn,m\geq N, ‖fn−fm‖<ε\|f_{n}-f_{m}\|<\varepsilon.

As usual, ∥⋅∥\|\cdot\| denotes the sup norm here. Note that this is only well defined for bounded functions, which is why Definition 6.20 specifies that each fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} is bounded.

Our goal in this section is to prove that a sequence of bounded functions is uniformly convergent if and only if it is uniformly Cauchy. The “only if” direction of this statement is much easier to prove than the “if” direction, so let’s start with that.

Theorem 6.22

Let fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} be a sequence of bounded functions converging uniformly to a bounded function f:D→ℝf:D\rightarrow{\mathbb{R}}. Then (fn)(f_{n}) is uniformly Cauchy.

The proof of this follows the proof of Lemma 1.16 (the analogous statement for real sequences) almost word for word. To make this work, we need a “triangle inequality” for the sup norm.

Lemma 6.23

For all bounded functions f,g:D→ℝf,g:D\rightarrow{\mathbb{R}},

‖f+g‖≤‖f‖+‖g‖.\|f+g\|\leq\|f\|+\|g\|.
Proof.

For all x∈Dx\in D, |f⁢(x)+g⁢(x)|≤|f⁢(x)|+|g⁢(x)|≤‖f‖+‖g‖|f(x)+g(x)|\leq|f(x)|+|g(x)|\leq\|f\|+\|g\|, since ‖f‖\|f\| is an upper bound on {|f⁢(x)|:x∈D}\{|f(x)|\>:\>x\in D\} and ‖g‖\|g\| is an upper bound on {|g⁢(x)|:x∈D}\{|g(x)|\>:\>x\in D\}. Hence, ‖f‖+‖g‖\|f\|+\|g\| is an upper bound on {|f⁢(x)+g⁢(x)|:x∈D}\{|f(x)+g(x)|\>:\>x\in D\}. But ‖f+g‖\|f+g\| is, by definition, the least upper bound on this set. Hence ‖f+g‖≤‖f‖+‖g‖\|f+g\|\leq\|f\|+\|g\|. ∎

Proof of Theorem 6.22:  Let ε>0\varepsilon>0 be given. Since (fn)(f_{n}) converges to ff uniformly, there exists N∈ℤ+N\in{\mathbb{Z}}^{+} such that, for all n≥Nn\geq N, ‖fn−f‖<ε/2\|f_{n}-f\|<\varepsilon/2. Hence, for all n,m≥Nn,m\geq N,

by Lemma 6.23. □\Box

OK, now for the hard(er) part. We first note that, if (fn)(f_{n}) is uniformly Cauchy, it is certainly pointwise Cauchy, hence pointwise convergent. That is:

Lemma 6.24

Let fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} be a uniformly Cauchy sequence of bounded functions. Then (fn)(f_{n}) converges pointwise to some function f:D→ℝf:D\rightarrow{\mathbb{R}}.

Proof.

Choose and fix x∈Dx\in D. I claim that the real sequence (fn⁢(x))(f_{n}(x)) is Cauchy. To see this, let ε>0\varepsilon>0 by given. Since (fn)(f_{n}) is uniformly Cauchy, there exists N∈ℤ+N\in{\mathbb{Z}}^{+} such that, for all n,m≥Nn,m\geq N, ‖fn−fm‖<ε\|f_{n}-f_{m}\|<\varepsilon. Hence, for all n,m≥Nn,m\geq N,

|fn⁢(x)−fm⁢(x)|≤‖fn−fm‖<ε.|f_{n}(x)-f_{m}(x)|\leq\|f_{n}-f_{m}\|<\varepsilon.

Since the real sequence (fn⁢(x))(f_{n}(x)) is Cauchy, it converges, by Theorem 1.19, to some limit f⁢(x)∈ℝf(x)\in{\mathbb{R}}. Allowing xx to vary in DD, we obtain a function f:D→ℝf:D\rightarrow{\mathbb{R}} to which (fn)(f_{n}) converges pointwise. ∎

Note that we don’t (yet) know that the pointwise limit f:D→ℝf:D\rightarrow{\mathbb{R}} of our uniformly Cauchy sequence of bounded functions is itself a bounded function! We need to prove this.

Lemma 6.25

Let a uniformly Cauchy sequence of bounded functions fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} converge pointwise to f:D→ℝf:D\rightarrow{\mathbb{R}}. Then ff is bounded.

Furthermore, fN:D→ℝf_{N}:D\rightarrow{\mathbb{R}} is a bounded function, so there exists K>0K>0 such that, for all x∈Dx\in D, |fN⁢(x)|≤K|f_{N}(x)|\leq K. I claim that |f||f| is bounded above by K+1K+1.

So we now know that every uniformly Cauchy sequence of bounded functions converges pointwise to a bounded function. The last job is to show that this convergence is actually uniform (recall uniform convergence implies pointwise convergence, but not vice versa, so this is not automatic).

Theorem 6.26

A sequence fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} of bounded functions converges uniformly if and only if it is uniformly Cauchy.

Proof.

We have already proved the “only if” part (Theorem 6.22).

Assume that (fn)(f_{n}) is uniformly Cauchy. Then (fn)(f_{n}) converges pointwise to some bounded function f:D→ℝf:D\rightarrow{\mathbb{R}} by Lemmas 6.24 and 6.25. We must show that ‖fn−f‖→0\|f_{n}-f\|\rightarrow 0.

Let ε>0\varepsilon>0 be given. Since (fn)(f_{n}) is uniformly Cauchy, there exists N∈ℤ+N\in{\mathbb{Z}}^{+} such that, for all n,m≥Nn,m\geq N, ‖fn−fm‖<ε/4\|f_{n}-f_{m}\|<\varepsilon/4. I claim that, for all n≥Nn\geq N,

‖fn−f‖<ε.\|f_{n}-f\|<\varepsilon.

To see this, choose and fix x∈Dx\in D. Then, for all n,m≥Nn,m\geq N,

|fn⁢(x)−f⁢(x)|\displaystyle|f_{n}(x)-f(x)| =\displaystyle= |fn⁢(x)−fm⁢(x)+fm⁢(x)−f⁢(x)|\displaystyle|f_{n}(x)-f_{m}(x)+f_{m}(x)-f(x)| (6.4)
≤\displaystyle\leq |fn⁢(x)−fm⁢(x)|+|fm⁢(x)−f⁢(x)|\displaystyle|f_{n}(x)-f_{m}(x)|+|f_{m}(x)-f(x)|
≤\displaystyle\leq ‖fn−fm‖+|fm⁢(x)−f⁢(x)|\displaystyle\|f_{n}-f_{m}\|+|f_{m}(x)-f(x)|
<\displaystyle< ε4+|fm⁢(x)−f⁢(x)|.\displaystyle\frac{\varepsilon}{4}+|f_{m}(x)-f(x)|.

Note this inequality holds for all m≥Nm\geq N. Now (fn)(f_{n}) converges pointwise to ff, so fn⁢(x)→f⁢(x)f_{n}(x)\rightarrow f(x). Hence, there exists N1∈ℤ+N_{1}\in{\mathbb{Z}}^{+} (depending on xx) such that, for all m≥N1m\geq N_{1}, |fm⁢(x)−f⁢(x)|<ε/4|f_{m}(x)-f(x)|<\varepsilon/4. Applying (6.4) in the case m=max⁡{N,N1}m=\max\{N,N_{1}\}, we see that, for all n≥Nn\geq N,

|fn⁢(x)−f⁢(x)|<ε4+ε4=ε2.|f_{n}(x)-f(x)|<\frac{\varepsilon}{4}+\frac{\varepsilon}{4}=\frac{\varepsilon}% {2}.

∥fn−f∥=sup{|fn(x)−f(x)|:x∈D}≤ε2<ε.\|f_{n}-f\|=\sup\{|f_{n}(x)-f(x)|\>:\>x\in D\}\leq\frac{\varepsilon}{2}<\varepsilon.

Since ε>0\varepsilon>0 was arbitrary, ‖fn−f‖→0\|f_{n}-f\|\rightarrow 0. ∎

Let us denote the set of bounded functions on DD

B⁢(D):={f:D→ℝ:f is bounded}B(D):=\{f:D\rightarrow{\mathbb{R}}\>:\>\mbox{$f$ is bounded}\}

and the set of continuous functions on DD

C⁢(D):={f:D→ℝ:f is continuous}C(D):=\{f:D\rightarrow{\mathbb{R}}\>:\>\mbox{$f$ is continuous}\}

For general sets DD, neither of these is a subset of the other. For example f⁢(x)=x2f(x)=x^{2} is in C⁢(ℝ)C({\mathbb{R}}) but not B⁢(ℝ)B({\mathbb{R}}), and g⁢(x)=1g(x)=1 for x≥0x\geq 0, g⁢(x)=0g(x)=0 for x<0x<0 is in B⁢(ℝ)B({\mathbb{R}}) but not C⁢(ℝ)C({\mathbb{R}}). In the case where D=[a,b]D=[a,b], a closed bounded interval, however, every continuous function is bounded (by the Extreme Value Theorem), so C⁢([a,b])⊂B⁢([a,b])C([a,b])\subset B([a,b]).

We have just shown that the set B⁢(D)B(D) enjoys (with respect to the sup norm) a fundamental property of the set ℝ{\mathbb{R}} (with respect to absolute value): every Cauchy sequence in the set converges to a limit in the set. Mathematicians call this property completeness. So what we have just shown is that B⁢(D)B(D) is complete with respect to the sup norm. In the case where D=[a,b]D=[a,b], it’s not hard to deduce that C⁢(D)C(D) is also complete with respect to the sup norm.

Theorem 6.27

If a sequence of continuous functions fn:[a,b]→ℝf_{n}:[a,b]\rightarrow{\mathbb{R}} is uniformly Cauchy, then it converges uniformly to some continuous function f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}}.

Proof.

Each fnf_{n} is a bounded function (by the Extreme Value Theorem), so (fn)(f_{n}) converges uniformly to some bounded function f:[a,b]→ℝf:[a,b]\rightarrow{\mathbb{R}} by Theorem 6.26. But ff is continuous by Theorem 6.14. ∎

The neat thing about Theorems 6.26 and 6.27 is that they allow us to prove that a sequence of functions converges uniformly without knowing what its limit is. As we will see, this is extremely useful when we come to consider functions defined by power series.

  • •
    ​

    The sup norm of a function f:D→ℝf:D\rightarrow{\mathbb{R}} is ∥f∥=sup{|f(x)|:x∈D}\|f\|=\sup\{|f(x)|\>:\>x\in D\}.

  • •
    ​

    A sequence of functions fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} converges uniformly to f:D→ℝf:D\rightarrow{\mathbb{R}} if the real sequence ‖fn−f‖\|f_{n}-f\| converges to 0.

  • •
    ​

    If fnf_{n} converges uniformly to ff, it converges pointwise to ff. The converse is false.

  • •
    ​

    If a sequence of continuous functions fn:D→ℝf_{n}:D\rightarrow{\mathbb{R}} converges uniformly to ff then ff is continuous.
