The main definition

2.1 The main definition

Now that we have a rigorous definition of the limit of a function, it is straightforward to define derivatives.

Definition 2.1

Let f:D→ℝf:D\rightarrow{\mathbb{R}}, where DD is some subset of ℝ{\mathbb{R}}, and a∈Da\in D be a cluster point of DD. Then ff is differentiable at aa if the limit

limx→af⁢(x)−f⁢(a)x−a\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}

exists. In this case, we denote the limit f′⁢(a)f^{\prime}(a) and call it the derivative of ff at aa. We say that ff is differentiable if it is differentiable at aa for all a∈Da\in D.

Remarks

  • •
    ​

    In general, in order to define limx→ag⁢(x)\lim_{x\rightarrow a}g(x), we only need aa to be a cluster point of the domain of gg: it isn’t necessary in general for aa to be in the domain of gg, so g⁢(a)g(a) may or may not exist. For example

    limx→1x2−1x−1\lim_{x\rightarrow 1}\frac{x^{2}-1}{x-1}

    exists, although the function is undefined at x=1x=1. Note, however, that the limit defining f′⁢(a)f^{\prime}(a) contains the number f⁢(a)f(a), so to be differentiable at aa, the point aa must be both an element and a cluster point of the domain of ff. For example, a function f:(0,∞)→ℝf:(0,\infty)\rightarrow{\mathbb{R}} cannot be differentiable at 0 (since 0∉(0,∞)0\notin(0,\infty)), and a function f:ℤ→ℝf:{\mathbb{Z}}\rightarrow{\mathbb{R}} cannot be differentiable anywhere (since ℤ{\mathbb{Z}} has no cluster points).

  • •
    ​

    The limit in Definition 2.1 is defined as in Definition 1.29. That is, f:D→ℝf:D\rightarrow{\mathbb{R}} is differentiable at aa if there exists a real number, denoted f′⁢(a),f^{\prime}(a), such that, for each ε>0\varepsilon>0, there exists δ>0\delta>0 such that, for all x∈Dx\in D with 0<|x−a|<δ0<|x-a|<\delta,

    |f⁢(x)−f⁢(a)x−a−f′⁢(a)|<ε.\left|\frac{f(x)-f(a)}{x-a}-f^{\prime}(a)\right|<\varepsilon.
  • •
    ​

    Geometrically, the quantity

    s⁢(x)=f⁢(x)−f⁢(a)x−as(x)=\frac{f(x)-f(a)}{x-a}

    is the slope of the straight line passing through the points (a,f⁢(a))(a,f(a)) and (x,f⁢(x))(x,f(x)). Such a straight line is called a chord on the graph y=f⁢(x)y=f(x). As xx approaches aa, the points defining the chord get “arbitrarily close” to one another, and the chords approach a straight line through (a,f⁢(a))(a,f(a)) with slope f′⁢(a)f^{\prime}(a), the so-called tangent line. This is why the derivative f′⁢(a)f^{\prime}(a) is often interpreted as the slope of the graph y=f⁢(x)y=f(x) at the point (a,f⁢(a))(a,f(a)). Note, however, that this is just an interpretation. The definition of f′⁢(a)f^{\prime}(a) is in terms of limits which are, in turn, defined precisely in Definition 1.29.

  • •
    ​

    If f:D→ℝf:D\rightarrow{\mathbb{R}} is differentiable, then its derivative is also a function f′:D→ℝf^{\prime}:D\rightarrow{\mathbb{R}}, mapping DD to ℝ{\mathbb{R}}. There is a very popular alternative notation for the derivative in this case, namely

    f′⁢(x)=d⁢fd⁢x.f^{\prime}(x)=\frac{df}{dx}.

    This notation is convenient in some circumstances, but it tends to blur the distinction between a function (in this case f′f^{\prime}) and the value of the function at a particular point (in this case f′⁢(x)f^{\prime}(x)), so we will tend to avoid it.

Let’s verify that some simple, familiar functions have the derivatives we expect. In each case, we will give a direct ε\varepsilon—δ\delta proof that the limit

limx→af⁢(x)−f⁢(a)x−a\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}

exists and has the expected value.

Example 2.2

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} such that f⁢(x)=cf(x)=c, a constant. Then ff is differentiable (everywhere) and f′⁢(a)=0f^{\prime}(a)=0 for all a∈ℝa\in{\mathbb{R}}.

Proof.

We must show that, for each ε>0\varepsilon>0 there is some δ>0\delta>0 such that

|f⁢(x)−f⁢(a)x−a−0|<ε\left|\frac{f(x)-f(a)}{x-a}-0\right|<\varepsilon

for all xx satisfying 0<|x−a|<δ0<|x-a|<\delta. So, let ε>0\varepsilon>0 be given. Let δ=1>0\delta=1>0. Then for all x∈ℝx\in{\mathbb{R}} with 0<|x−a|<δ0<|x-a|<\delta,

|f⁢(x)−f⁢(a)x−a−0|\displaystyle\left|\frac{f(x)-f(a)}{x-a}-0\right| =\displaystyle= |c−cx−a|=0<ε.\displaystyle\left|\frac{c-c}{x-a}\right|=0<\varepsilon.

∎

So, for a constant function, the same δ\delta will work for every ε>0\varepsilon>0. (We chose δ=1\delta=1 but any other δ>0\delta>0 works equally well.) Here’s another example that’s equally obliging:

Example 2.3

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} such that f⁢(x)=xf(x)=x. Then ff is differentiable (everywhere) and f′⁢(a)=1f^{\prime}(a)=1 for all a∈ℝa\in{\mathbb{R}}.

We must show that, for each ε>0\varepsilon>0 there is some δ>0\delta>0 such that

|f⁢(x)−f⁢(a)x−a−1|<ε\left|\frac{f(x)-f(a)}{x-a}-1\right|<\varepsilon

for all xx satisfying 0<|x−a|<δ0<|x-a|<\delta. So, let ε>0\varepsilon>0 be given. Let δ=1>0\delta=1>0. Then for all x∈ℝx\in{\mathbb{R}} with 0<|x−a|<δ0<|x-a|<\delta,

OK, all is as we expected. What about functions which are not differentiable? How do we prove they aren’t? A function f:D→ℝf:D\rightarrow{\mathbb{R}} is not differentiable at aa precisely if the limit

limx→af⁢(x)−f⁢(a)x−a\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a}

does not exist. We can prove this by thinking carefully about the negation of the statement defining a limit (Definition 1.29), but it’s usually easier to exploit Theorem 1.36. Recall that this says that a function, gg say, has limit LL at aa if and only if, for all sequences (xn)(x_{n}) in D\{a}D{\backslash}\{a\} converging to aa, g⁢(xn)g(x_{n}) converges to LL. So to prove that limx→ag⁢(x)\lim_{x\rightarrow a}g(x) doesn’t exist, it’s enough to find just one sequence (xn)(x_{n}) in D\{a}D{\backslash}\{a\} converging to aa whose image sequence (g⁢(xn))(g(x_{n})) does not converge.

Example 2.7

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} such that f⁢(x)=|x|f(x)=|x|. Then ff is not differentiable at 0.

Proof.

Assume, towards a contradiction, that ff is differentiable at 0 with derivative f′⁢(0)f^{\prime}(0). Then

limx→0f⁢(x)−f⁢(0)x−0=f′⁢(0).\lim_{x\rightarrow 0}\frac{f(x)-f(0)}{x-0}=f^{\prime}(0).

Let xn=(−1)n/nx_{n}=(-1)^{n}/n. Note that this is a sequence in ℝ\{0}{\mathbb{R}}{\backslash}\{0\} converging to 0. Hence, by Theorem 1.36,

f⁢(xn)−f⁢(0)xn−0→f′⁢(0).\frac{f(x_{n})-f(0)}{x_{n}-0}\rightarrow f^{\prime}(0).

But

f⁢(xn)−f⁢(0)xn−0=|(−1)n/n|(−1)n/n=1(−1)n=(−1)n\frac{f(x_{n})-f(0)}{x_{n}-0}=\frac{|(-1)^{n}/n|}{(-1)^{n}/n}=\frac{1}{(-1)^{n% }}=(-1)^{n}

which does not converge, a contradiction. Hence ff is not differentiable at 0. ∎

Remark Alternatively, we could have defined yn=1/ny_{n}=1/n and zn=−1/nz_{n}=-1/n and noted that both are sequences in ℝ\{0}{\mathbb{R}}{\backslash}\{0\} converging to 0, so if ff is differentiable at 0, then both

There is a link between differentiablity and continuity, as we now show.

Proposition 2.8

Let f:D→ℝf:D\rightarrow{\mathbb{R}} be differentiable at a∈Da\in D. Then ff is continuous at aa.

Proof.

By Theorem 1.43, it suffices to show that limx→af⁢(x)=f⁢(a)\displaystyle\lim_{x\rightarrow a}f(x)=f(a). Define

s:D\{a}→ℝ,s⁢(x)=f⁢(x)−f⁢(a)x−a.s:D{\backslash}\{a\}\rightarrow{\mathbb{R}},\qquad s(x)=\frac{f(x)-f(a)}{x-a}.

Then, by assumption, ss has a limit (denoted f′⁢(a)f^{\prime}(a)) at aa. But for all x∈D\{a}x\in D{\backslash}\{a\},

f⁢(x)=f⁢(a)+(x−a)⁢s⁢(x),f(x)=f(a)+(x-a)s(x),

and hence, by the Algebra of Limits (Theorem 1.35),

limx→af⁢(x)=f⁢(a)+0×f′⁢(a)=f⁢(a).\lim_{x\rightarrow a}f(x)=f(a)+0\times f^{\prime}(a)=f(a).

∎

Remark. The converse of Proposition 2.8 is false: if ff is continuous at aa, a cluster point of its domain, it does not follow that ff is differentiable there. We’ve already seen a counteraxample: f⁢(x)=|x|f(x)=|x| is continuous at 0 but is not differentiable at 0 (see Example 2.7). In this case, the function fails to be differentiable only at a single isolated point. It is straightforward to construct functions which are differentiable only at a single isolated point.

Example 2.9

Let f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} be defined such that

f⁢(x)={xif x∈ℚ,x−x2if x∉ℚ.f(x)=\left\{\begin{array}[]{ll}x&\mbox{if $x\in{\mathbb{Q}}$,}\\ x-x^{2}&\mbox{if $x\notin{\mathbb{Q}}$.}\end{array}\right.

I claim that ff is differentiable at 0, with f′⁢(0)=1f^{\prime}(0)=1, but is not differentiable anywhere else.

Proof.

Let a∈ℝa\in{\mathbb{R}} and assume that ff is differentiable at aa. Then, by Proposition 2.8, ff is continuous at aa. For each n∈ℤ+n\in{\mathbb{Z}}^{+} there exist rn∈ℚr_{n}\in{\mathbb{Q}} and in∈ℝ\ℚi_{n}\in{\mathbb{R}}{\backslash}{\mathbb{Q}} such that a<rn<a+1/na<r_{n}<a+1/n and a<in<a+1/na<i_{n}<a+1/n. Clearly rn→ar_{n}\rightarrow a and in→ai_{n}\rightarrow a by the Squeeze Rule. Hence, by the definition of continuity, f⁢(rn)→f⁢(a)f(r_{n})\rightarrow f(a) and f⁢(in)→f⁢(a)f(i_{n})\rightarrow f(a). But rnr_{n} is rational, so f⁢(rn)=rn→af(r_{n})=r_{n}\rightarrow a, and ini_{n} is irrational, so f⁢(in)=in−in2→a−a2f(i_{n})=i_{n}-i_{n}^{2}\rightarrow a-a^{2}. Both these limits equal f⁢(a)f(a), so a=a−a2a=a-a^{2}, whence a=0a=0. So if ff is differentiable at aa, then a=0a=0. Equivalently, ff is not differentiable at any a≠0a\neq 0.

It remains to show that ff is differentiable at 0 with f′⁢(0)=1f^{\prime}(0)=1, that is,

limx→0f⁢(x)−f⁢(0)x−0=1.\lim_{x\rightarrow 0}\frac{f(x)-f(0)}{x-0}=1.

In this case, it’s easier to use Theorem 1.36 than to give a direct ε\varepsilon—δ\delta argument. So, let (xn)(x_{n}) be any sequence in ℝ\{0}{\mathbb{R}}{\backslash}\{0\} converging to 0, and consider

sn=f⁢(xn)−f⁢(0)xn−0=f⁢(xn)xn.s_{n}=\frac{f(x_{n})-f(0)}{x_{n}-0}=\frac{f(x_{n})}{x_{n}}.

We must show that sn→1s_{n}\rightarrow 1. If xnx_{n} is rational, then |sn−1|=|xn/xn−1|=0|s_{n}-1|=|x_{n}/x_{n}-1|=0, whereas if xnx_{n} is irrational then

|sn−1|=|xn−xn2xn−1|=|xn|.|s_{n}-1|=\left|\frac{x_{n}-x_{n}^{2}}{x_{n}}-1\right|=|x_{n}|.

Hence, for all nn, 0≤|sn−1|≤|xn|0\leq|s_{n}-1|\leq|x_{n}|, so |sn−1|→0|s_{n}-1|\rightarrow 0 by the Squeeze Rule. Hence sn→1s_{n}\rightarrow 1. It follows that f′⁢(0)=1f^{\prime}(0)=1, as claimed. ∎

So this (admittedly rather bizarre) function is differentiable at 0, and has a positive derivative there. Naive intuition would suggest, therefore, that the function should be increasing, at least for values of xx sufficiently close to 0. In fact, this is false!

Example 2.9 continued Claim: the function ff is not increasing on any neighbourhood of 0. That is, there does not exist ε>0\varepsilon>0 such that ff is increasing on (−ε,ε)(-\varepsilon,\varepsilon).

Proof.

Let ε>0\varepsilon>0 be given. Then there exists an irrational number xx such that 0<x<ε0<x<\varepsilon. By definition, f⁢(x)=x−x2<xf(x)=x-x^{2}<x. Similarly, there exists a rational number yy such that x−x2<y<xx-x^{2}<y<x. But then y<xy<x and f⁢(y)=y>x−x2=f⁢(x)f(y)=y>x-x^{2}=f(x). Hence, ff is not increasing on the interval (−ε,ε)(-\varepsilon,\varepsilon). This is true no matter which positive number ε\varepsilon we choose, which establishes the claim. ∎

Your reaction to this may well be “so what, that’s a really crazy function – after all, it’s only differentiable at the single point 0.” We will see later an example of a function f:ℝ→ℝf:{\mathbb{R}}\rightarrow{\mathbb{R}} which is differentiable everywhere and has f′⁢(0)=1f^{\prime}(0)=1, but still is not increasing on any neighbourhood of 0.
